27. 5.def top ()top = len(stack) -1if top<0print "tidak terdefinisi"elseprint stack[top]maksud dari perintah/instruksi tersebut adalah​


Jawaban:

Penjelasan:

Sepertinya ini perintah untuk mencek apakah stack memiliki elemen atau tidak.

Stack memiliki elemen atas dan bawah (tapi lebih signifikan elemen atas), jika panjang stack = 1, maka elemen atas = bawah.

Pada kode tersebut method top() pertama akan mengurangi panjang variabel list stack sebesar 1 dan menyimpannya ke variabel top.

Lalu jika top bernilai negatif, program akan mencetak "tidak terdefinisi". Yang artinya stack belum memiliki elemen (dan akibatnya elemen teratas belum terdefinisi).

Jika tidak maka program akan mencetak elemen dari stack[] pada indeks berupa nilai variabel top (mencetak elemen teratas pada stack[]).

40. harga top up suatu aplikasi menerapkan fungsi f( x )= 3/ 4 x + 10000 jika kuat top up sebesar Rp50.000 Berapa harga yang perlu dibayar​

[tex]f(x) = \frac{3}{4} \: x + 10.000[/tex]

top up sebesar Rp50.000

[tex]f(50.000) = \frac{3}{4} \: (50.000) + 10.000 \\ \\ f(50.000) =37.500 + 10.000 \\ \\ f(50.000) =47.500[/tex]
