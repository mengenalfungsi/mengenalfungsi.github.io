1. Sebutkan bagian tubuh udang dan fungsinya


tubuh udang teriri atas tiga bagian, yakni cephalothorax abdomen (tubuh), dan uropod (ekor). Cephalothorax merupakan gabungan dari kepala dan dada udang galah. Bagian ini dibungkus oleh kulit keras yang disebut dengan keramas atau cangkang. Bagian depan kepala udang galah terdapat tonjolan karapas yang bergerigi (rostrum). Rostrum digunakan untuk mengidentifikasi jenis udang galah. Kaki renang pada ujung betina agak melebar dan membentuk ruang untuk mengerami telurnya (brood chambers). Uropada berfungsi sebagai pengayah atau yang biasa disebut dengan ekor kipas. [tex]Fisiologi, Anatomi, Karsinologi[/tex]

Tubuh udang terdiri atas
=> Kelenjar hijau = Alat ekskresi
=> Keliped = Alat untuk menjepi mangsa
=> Kaki jalan = Pada betina dapat digunakan untuk menyimpan telur
=> Antena = Memiliki kemoreseptor
=> Jantung = Untuk memompa darah ke seluruh tubuh
=> Otak = Sebagai saraf pusat
=> Telson = Alat untuk megendalikan berenang
