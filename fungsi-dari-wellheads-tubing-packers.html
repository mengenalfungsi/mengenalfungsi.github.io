technicians = x
packer = y

x+y = 540
x = 540 - y

24x + 20y = 12000
12x + 10y = 6000
subtitute x
12(540-y)+10y = 6000
6480-12y+10y = 6000
6480-6000 = 2y
y = 480/2 = 240

substitute y to
x = 540-y
x = 540-240 = 300

so, the number of technicians are 300 and the number of packers are 240
