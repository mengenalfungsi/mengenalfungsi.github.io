Jawaban:

Masukkan kritik dan saran selama kuliah sangat berharga untuk membantu meningkatkan pengalaman belajar dan kualitas hasil akhir, seperti program dan laporan tugas akhir/skripsi. Berikut adalah beberapa masukkan kritik dan saran yang dapat Anda berikan kepada dosen pembimbing dan institusi:

1. Kritik dan saran untuk dosen pembimbing:

- Harapkan komunikasi yang terbuka dan teratur dengan dosen pembimbing. Dosen pembimbing sebaiknya responsif dan tersedia untuk memberikan bimbingan dan umpan balik yang diperlukan.

- Berikan saran agar dosen pembimbing memberikan arahan yang jelas dan terstruktur mengenai langkah-langkah yang harus diambil dalam membuat program dan laporan tugas akhir/skripsi. Pengarahan yang jelas akan membantu mempercepat proses dan mengurangi kebingungan mahasiswa.

- Usulkan dosen pembimbing memiliki ketersediaan waktu yang memadai untuk memberikan bimbingan secara konsisten.

2. Kritik dan saran untuk institusi:

- Ajukan saran agar institusi menyediakan sumber daya yang memadai seperti perpustakaan yang lengkap, fasilitas komputasi yang memadai, dan akses ke jurnal dan literatur yang diperlukan.

- Minta institusi untuk menyelenggarakan seminar atau workshop yang berhubungan dengan penelitian atau penulisan tugas akhir/skripsi, sehingga mahasiswa dapat memperoleh pemahaman yang lebih baik tentang proses penyelesaian tugas akhir/skripsi.

- Sarankan institusi untuk menyediakan kegiatan mentoring atau tutoring bagi mahasiswa, yang bertujuan untuk membantu mereka dalam memahami dan menyelesaikan tugas akhir/skripsi dengan tepat waktu.

3. Alasan umum pelaporan tugas akhir/skripsi terlambat/tidak tepat waktu:

- Keterbatasan waktu yang dihabiskan untuk penelitian dan analisis, sehingga waktu yang tersedia untuk menulis laporan menjadi terbatas.

- Masalah dengan perencanaan dan manajemen waktu yang tidak efektif.

- Kesulitan dalam memahami format dan persyaratan penulisan yang diharapkan oleh institusi.

- Kesulitan dalam mengatasi masalah penelitian atau mencapai target yang ditetapkan dalam waktu yang telah ditentukan.

- Gangguan pribadi atau kehidupan lain yang mempengaruhi fokus dan dedikasi dalam menyelesaikan tugas akhir/skripsi.

Penting untuk berkomunikasi terbuka dengan dosen pembimbing dan institusi untuk memberikan masukkan kritik dan saran. Dengan memberikan umpan balik yang konstruktif, diharapkan dapat terjadi perbaikan yang bermanfaat untuk meningkatkan kualitas belajar dan pencapaian dalam tugas akhir/skripsi.

22. Buatlah algoritma (flowchart) dan program untuk menampilkan semua bilangan dari Batas Awal sampai Batas Akhir dan tentukan bilangan tersebut termasuk ganjil atau genap. Contoh keluaran program adalah sebagai berikut (contoh keluaran untuk input awal = 6, dan akhir = 10). Awal dan akhir diinputkan dari keyboard oleh user program.


Program dalam bahasa java :

// AUTHOR : LordRimuruTempest

import java.util.Scanner;

public class Test {

   public static void main(String[] args) {

       Scanner baca = new Scanner(System.in);

       System.out.println("PROGRAM DENGAN BAHASA JAVA");

       System.out.print("Batas Awal  : ");

       int batasAwal = baca.nextInt();

       System.out.print("Batas Akhir : ");

       int batasAkhir = baca.nextInt();

       System.out.println("Hasil : ");

       for (int i = batasAwal; i <= batasAkhir; i++) {

           if(i % 2 == 0){

               System.out.println(i + " adalah bilangan genap");

           }else{

               System.out.println(i + " adalah bilangan ganjil");

           }

       }

   }

}

Program dalam bahasa C++ :

/* AUTHOR: LordRimuruTempest */

#include <iostream>

using namespace std;

int main()

{

   int batasAwal, batasAkhir;

   cout<<"PROGRAM DENGAN BAHASA C++\n";

   cout<<"Batas awal  : ";

   cin>>batasAwal;

   cout<<"Batas Akhir : ";

   cin>>batasAkhir;

   cout<<"Hasil : \n";

   for(int i = batasAwal;i <= batasAkhir; i ++){

       if(i % 2 == 0){

           cout<<i<<" adalah bilangan genap\n";

       }else{

           cout<<i<<" adalah bilangan ganjil\n";

       }

   }

   return 0;

}

Pembahasan

Disini program akan dibuat menggunakan bahasa java dan c++. Jika bahasa pemrograman yang dipakai berbeda, silahkan menyesuaikan atau meminta ganti di kolom komentar.

Sebelum membuat flowchart dan program, harus ditentukan dulu algoritma yang tepat dalam menyelesaikan permasalahan, Perhatikan bahwa bilangan genap adalah bilangan yang habis dibagi 2, dan bilangan ganjil adalah bilangan yang tidak habis dibagi 2. Maka algoritmanya dapat dibuat seperti berikut :

input nilai batas awal dan batas akhir.Lakukan perulangan menggunakan for, dengan indeks awal i = batasAwal sampai i = batasAkhir dan jangan lupa untuk melakukan increment pada indeks i.Cek satu-persatu nilai sisa bagi i dibagi 2 selama perulangan. Jika i habis dibagi 2, cetak "i adalah bilangan genap", jika tidak maka cetak "i adalah bilangan ganjil".Program selesai.

Algoritma tersebut kemudian digambarkan dengan flowchart / diagram alir (ada di lampiran foto beserta aturan menggambar flowchart).

Penjelasan program dalam bahasa java :

Pertama tama import dahulu fungsi scanner dengan perintah import java.util.Scanner kemudian beri nama objek scanner tersebut dengan nama "baca". kemudian input batas awal dan batas akhir dengan perintah baca.nextint lalu lakukan perulangan for untuk mencetak hasil yang diinginkan sesuai arahan algoritma dan program selesai. Program beserta output-nya dapat dilihat pada lampiran gambar.

Penjelasan program dalam bahasa C++ :

Masukkan library iostream(input output stream) kedalam program untuk menggunakan fungsi input output. Kemudian deklarasikan namespace std di awal program dengan perintah using namespace std sehingga tidak perlu menuliskan identitas std:: pada anggota dari namespace std (contohnya saat memakai fungsi output cout). Masukkan nilai batas awal dan batas akhir menggunakan fungsi cin (c input) kemudian cetak hasil yang diinginkan menggunakan perulangan for sesuai arahan algoritma dan program selesai. Program beserta output-nya dapat dilihat pada lampiran gambar.

36. Apa yang Anda harapkan dari dosen pembimbing, sehingga Anda mampu membuat program dan laporan tugas akhir/skripsi secara mandiri dan tepat waktu?​


Sebagai saran umum, berikut adalah beberapa hal yang bisa diharapkan dari seorang dosen pembimbing agar mahasiswa dapat membuat program dan laporan tugas akhir/skripsi dengan mandiri dan tepat waktu:

1. Arahan dan Panduan: Dosen pembimbing harus memberikan arahan yang jelas dan panduan mengenai tujuan, metodologi, dan format yang diharapkan dalam penulisan program dan laporan tugas akhir/skripsi.

2. Dukungan dan Motivasi: Dosen pembimbing harus memberikan dukungan dan motivasi kepada mahasiswa untuk mengatasi kendala yang mungkin timbul selama proses pengerjaan tugas akhir/skripsi. Dukungan ini bisa berupa saran, inspirasi, dan dorongan agar mahasiswa terus bersemangat dan termotivasi.

3. Ketersediaan dan Komunikasi: Dosen pembimbing yang responsif dan berkomunikasi dengan baik bisa membantu mahasiswa dalam mengatasi pertanyaan, keluhan, atau kesulitan yang muncul. Dosen pembimbing yang tersedia, baik melalui pertemuan tatap muka atau melalui komunikasi elektronik, akan memfasilitasi kemajuan yang tepat waktu dalam pembuatan program dan laporan tugas akhir/skripsi.

4. Evaluasi dan Umpan Balik: Dosen pembimbing harus memberikan evaluasi dan umpan balik yang konstruktif terhadap progres mahasiswa dalam pembuatan program dan laporan tugas akhir/skripsi. Hal ini akan membantu mahasiswa dalam memperbaiki kekurangan dan menghasilkan pekerjaan yang lebih baik.

5. Pengawasan dan Pemantauan: Tugas dosen pembimbing juga termasuk dalam pengawasan dan pemantauan progress mahasiswa dalam pembuatan program dan laporan tugas akhir/skripsi. Dosen pembimbing harus memastikan agar mahasiswa tetap berada pada jalur yang tepat dan mengingatkan jika ada keterlambatan atau kesalahan yang perlu diperbaiki.

Kerjasama yang baik antara mahasiswa dan dosen pembimbing akan menjadi faktor penting dalam menyelesaikan program dan laporan tugas akhir/skripsi secara mandiri dan tepat waktu.

38. Berikan contoh permasalahan terkait dengan Pancasila yang akhir-akhir ini muncul?(minimal 2)


Jawaban:

Pancasila sebagai dasar dan ideologi negara Indonesia merupakan seperangkat nilai yang menjadi pandangan hidup (way of life) bagi negara Indonesia. Kondisi itu meniscayakan bahwa fondasi bernegara dan praktik kehidupan berbangsa dan bernegara harus berlandaskan nilai-nilai yang terkandung di dalam Pancasila.

Konsensus cerdas para pendiri negara tersebut berangkat dari sebuah paham kebangsaan yang terbentuk dari kesamaan nasib, sepenanggungan, dan sejarah serta adanya cita bersama untuk menjadi bangsa yang bersatu, berdaulat, adil, dan makmur dalam sebuah negara kesatuan.

Adanya prinsip bersatu dalam perbedaan membuat Pancasila menjadi semakin kuat dan layak sebagai sebuah ideologi bagi negara Indonesia yang khas dengan keanekaragamannya. Konsep persatuan yang ideal dengan mengkondisikan setiap warga negara hidup berdampingan dan gotong royong tanpa menghilangkan identitas suku bangsa, adat istiadat, ras, ataupun agama.

Berikut 10 kementerian beserta tugas dan wewenangnya:

1. Kementrian Pertanian= menangani tentang produk pertanian dan masalah pertanian di Indonesia.

2. Kementerian Kelautan dan Perikanan= menangani kegiatan maritim di Indonesia.

3. Kementerian Pemuda dan Olahraga= menangani urusan kepemudaan dan perlombaan olahraga.
