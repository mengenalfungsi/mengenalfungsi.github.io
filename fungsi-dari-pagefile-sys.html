1. jelaskan fungsi jpg tif sys pdf cdr


fungsi? mungkin maksud anda ekstensi.
*.sys =  adalah file ekstensi untuk system file. file penting untuk aplikasi.
*.jpg & *.tif =  termasuk file ekstensi yang digunakan untuk file tipe gambar yang sangat umum dipakai.
selanjutnya untuk
*.pdf = digunakan untuk menyimpan file dokumen. biasa dibuka dengan aplikasi adobe reader.
dan terakhir
*.cdr =  digunakan untuk menyimpan file berbasis aplikasi coreldraw untuk gambar.

2. apa fungsi dari ekstensi file berikut jpg,tif,sys,pdf dan cdr


Jpg untuk gambar , tif untuk file gambar tapi lebih hq lagi , kalo sys untuk data suatu sistem contoh file dari wind*ws , kalo pdf untuk file berupa Gambar , tabel , grafik , dokumen dan cdr itu file hasil dari coreldraw alias gambar Arsitektut gitu- JPG format untuk mengkompresi berkas bitmap

- TIFF (Temporary Instruction File Format). Format berkas gambar terbaik. Biasanya digunakan untuk pencetakan dengan kualitas gambar tinggi.

-SYS untuk data dari suatu sistem (misalnya msdos.sys)

- PDF (Portable Document Format) ekstensi untuk dokumen. Keuntungannya bersifat portabel karena hampir bisa dibuka dimanapun

-CDR ekstensi yang dipakai oleh Corel Draw (aplikasi pengolah vektor). Contoh aplikasi lain yang serupa adalah Inkscape yang menggunakan format terstandar yaitu SVG (Scalable Vector Graphic)
