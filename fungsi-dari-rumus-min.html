9. diketahui turunan pertama fungsi fx adalah F aksen x = 4 min 6 x Jika nilai F 3 = MIN 12 Tentukan rumus fungsi fx​


Jawaban:

f(x) = -3x^2 + 4x + 3

Penjelasan dengan langkah-langkah:

tertera di gambar ya..

Rumus fungsi f(x) adalah f(x) =  4x - 3x² + 3.

Pembahasan

Turunan adalah perubahan fungsi dengan semakin berkurangnya pangkat dari variabe dan berubahnya koefisien variabel.Jika [tex]y = a^{x}[/tex] , maka [tex]y' = (ax)^{x-1}[/tex]Integral adaah kebalikan dari turunan. Integral dari turunan suatu fungsi akan menghasilkan fungsi itu sendiri.

Penyelesaian

Diketahui:

f'(x) = 4-6x

f(3) = -12

Ditanyakan:

f(x)

Jawab:

1. Integralkan f'(x) untuk menemukan f(x).

f(x) =  ∫ f'(x) dx

=  ∫ 4-6x dx

= 4(1/(0+1))x^(0+1) - 6(1/(1+1))x^(1+1)

= 4(1/1)x - 6(1/2)x²

= 4x - 3x² + C

2. Subtitusikan persamaan f(x) yang telah ditemukan dengan f(3).

f(x) = 4x - 3x² + C

f(3) = 4(3) - 3(3)² + C

-12 = 12 - 3(9) + C

-12 = 12 - 27 + C

-12 = -15 + C

-12+15 = C

3 = C

3. Subtitusikan nilai C ke persamaan f(x) tadi.

f(x) =  4x - 3x² + C

f(x) =  4x - 3x² + 3

Jadi, rumus fungsi f(x) adalah f(x) =  4x - 3x² + 3.

24. Sum, average, count, max, dan min merupakan contoh rumus fungsi dasar pada Spreadsheet. Terangkanfungsi tersebut! * *​


Jawaban:

Fungsi Sum Di Microsoft Excel

Fungsi Sum digunakan untuk menjumlahkan isi data yang ada pada range tertentu. Bentuk penulisan untuk Fungsi Sum di Microsoft excel adalah =SUM(cell awal : cell akhir). Contohnya kita memiliki tabel nilai siswa seperti dibawah ini, dan kita ingin mengetahui berapa total nilai yang dimiliki dari semua siswa tersebut, maka penulisan rumusnya adalah =SUM(C2:C6)

Fungsi Average di Microsoft Excel digunakan untuk mencari nilai rata rata dari suatu range tertentu. Bentuk penulisan fungsi Average di Microsoft Excel adalah =Average(cell awal : cell akhir). Masih menggunakan tabel siswa

seperti pada contoh fungsi sum diatas, kita akan mencari nilai rata rata dari semua nilai siswa yang ada, maka rumusnya adalah =Average(C2:C6).

Fungsi Min di Microsoft Excel digunakan untuk mencari nilai terendah dari range tertentu. Bentuk penulisannya adalah =Min(cell awal : cell akhir). Masih menggunakan tabel yang sama, kita akan mencoba mencari nilai terendah dari range nilai, maka rumusnya adalah =Min(C2:C6).

Fungsi Count di Microsoft Excel digunakan untuk mencari jumlah total data yang ada pada range tertentu. Bentuk penulisannya adalah =Count(cell awal : cell akhir). Masih menggunakan tabel yang sama, kita akan mencoba mencari jumlah total data dari range nilai, maka rumusnya adalah =Count(C2:C6).
