8. fungsi operator Increment Dan Decrement, dalam bahasa c++....tolong!!!!


Increment adalah operator yang digunakan untuk menambah nilai sebuah variable dengan 1 angka dan decrement digunakan untuk mengurangi nilai sebuah variable 1 angka. Contoh penggunaan increment dan decrement dalam C++:


int a = 10;

++a;

printf("%d\n",a); //Hasilnya menjadi 11

printf("%d\n",a++); //Hasilnya menjadi 11 karena nilai diprint terlebih dahulu lalu  ditambah 1 (Post-increment)

printf("%d\n",a); //Hasilnya menjadi 12 setelah diincrement

a--;

printf("%d",a); //Hasilnya menjadi 11

11. cara hitung operator bitwise shift right yang soalnya 28>>98 gimana ya? bisa tolong di jelaskan


Pengaturan sedikit

Gunakan operator OR bitwise (|) untuk menyetel sedikit.

number | = 1UL << n;

Itu akan mengatur jumlah nth bit. n harus nol, jika Anda ingin mengatur bit pertama dan seterusnya hingga n-1, jika Anda ingin mengatur bit n.

Gunakan 1ULL jika jumlahnya lebih lebar dari yang tidak bertanda panjang; promosi 1UL << n tidak terjadi sampai setelah mengevaluasi 1UL << n di mana perilaku tidak terdefinisi bergeser lebih dari lebar yang panjang. Hal yang sama berlaku untuk semua sisa contoh.

Membersihkan sedikit

Gunakan operator DAN bitwise (&) untuk menghapus sedikit.

number & = ~ (1UL << n);

Itu akan menghapus jumlah bit ke-n. Anda harus membalikkan bit string dengan operator bitwise NOT (~), lalu AND it.

Beralih sedikit

Operator XOR (^) dapat digunakan untuk beralih sedikit.

angka ^ = 1UL << n;

Itu akan mengubah bit ke-n dari angka.

Memeriksa sedikit

Anda tidak meminta ini, tapi saya mungkin juga menambahkannya.

Untuk memeriksa sedikit, geser nomor n ke kanan, lalu bitwise DAN itu:

bit = (number >> n) & 1U;

Itu akan menempatkan nilai bit ke-n dari angka ke dalam bit variabel.

Mengubah bit n ke x

Mengatur bit n ke 1 atau 0 dapat dicapai dengan yang berikut pada implementasi C ++ komplemen 2:

angka ^ = (-x ^ angka) & (1UL << n);

Bit n akan diatur jika x adalah 1, dan dihapus jika x adalah 0. Jika x memiliki nilai lain, Anda mendapatkan sampah. x = !! x akan melakukan booleanize ke 0 atau 1.

Untuk membuat ini independen dari perilaku negasi komplementasi 2 (di mana -1 memiliki semua bit set, tidak seperti pada pelengkap 1 atau tanda / besarnya C + + implementasi), gunakan negasi unsigned.

number ^ = (- (unsigned long) x ^ number) & (1UL << n);

atau

newbie lama yang tidak ditandatangani = !! x; // Juga booleanize untuk memaksa 0 atau 1 angka ^ = (-newbit ^ number) & (1UL << n);

Ini biasanya ide yang baik untuk menggunakan tipe unsigned untuk manipulasi bit portabel.

Ini juga umumnya ide yang baik untuk tidak menyalin / menyisipkan kode pada umumnya dan begitu banyak orang menggunakan makro preprocessor (seperti komunitas wiki menjawab lebih jauh ke bawah) atau semacam enkapsulasi
