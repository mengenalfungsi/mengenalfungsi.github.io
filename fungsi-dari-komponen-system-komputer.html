Penjelasan rinci:

Sistem komputer terdiri dari dari 3 komponen di atas, bila dijabarkan:

1. Hardware bila digolongkan sesuai fungsinya, terdiri dari 4 alat, yaitu:

- alat input: untuk menginput data ke komputer (contoh: mouse).

- alat proses: untuk memproses data di komputer (contoh: processor).

- alat output: sebagai keluaran data dari komputer (contoh: monitor)

- alat penyimpanan: sebagai media penyimpanan data di komputer (contoh: harddisk)

2. Software bila digolongkan sesuai fungsinya, terdiri dari:

- software sistem operasi: Software yg menjadi dasar dari suatu perangkat komputer. Contohnya Windows, Mac OS, Linux.

- software aplikasi: Software yg menangani hal-hal khusus (tertentu). contohnya Ms. Word, Ms. Excel, Ms. Access.

- software utility: Software yg digunakan sebagai alat bantu sistem operasi. Contohnya: Disk Defragment, Scan Disk, Device Manager.

- software bahasa pemrograman: Software yg dipakai untuk membuat software. Contohnya: C++, Delphi, Visual Basic.

3. Brainware ialah pengguna komputer (yaitu manusia), contohnya:

- Web designer: orang yg memakai komputer untuk mendesain suatu website di internet.

- Programmer: orang yg memakai komputer untuk membuat program.

- Network administrator: orang yg memakai komputer dalam jaringan untuk mengelola jaringan komputer.

- Graphic Designer: orang yg memakai komputer untuk tujuan di bidang desain grafis (contohnya: membuat logo, dan lain-lain)

- System analis: orang yg memakai komputer untuk melakukan analisis suatu program.

- dan lain sebagainya.


12. kesimpulan tentang fungsi system output komputer


       A. Pengertian dan fungsi peralatan input pada komputer

Peralatan input pada komputer adalah perangkat yang memberikan perintah langsung atau data ke peralatan proses pada komputer untuk diproses kemudian dikeluarkan berupa informasi kepada pengguna. Peralatan input merupakan bagian komputer yang mempunyai peranan yang sangat penting karena langsung berhubungan dengan user atau pengguna komputer. Berikut adalah macam-macam peralatan input pada komputer dan fungsi masing-masing komponen tersebut.

Macam-macam dan fungsi peralatan input komputer :

Keyboard
Merupakan peralatan input pada komputer yang berfungsi untuk memasukan data atau perintah berupa angka, huruf maupun symbol, Keboard disebut juga papan ketik pada komputer. keyboard sendiri terdiri dari susunan huruf A s/d Z dan angka 0 s/d 9 dan ada beberapa symbol seperti titik dua, koma, titik dan lain-lain, juga terdapat tobol sortcut yang memudahkan kita untuk mengoperasikan komputer.Mouse
Mouse adalah peralatan input pada komputer yang berfungsi untuk menggerakan pointer pada layar monitor. Mouse sangat berguna untuk sistem operasi yang berbasis grafis.Joy Stick
joy Stick adalah alat input yang digunakan untuk memasukan perintah untuk menggerakan permainan pada komputer.Scanner
Scanner adalah alat input yang mempunyai fungsi hampir mirip dengan mesin foto copy, akan tetapi dengan scanner data dapat diubah dalam bentuk file sehingga dapat disimpan ke dalam harddisk komputer.

B. Pengertian dan Fungsi Peralatan Output Pada Komputer

Peralatan Output pada komputer adalah merupakan komponen komputer yang berfungsi menghasilkan informasi yang diperoleh dari hasil proses dari peralatan proses komputer. Output hasil proses dari komputer digolongkan menjadi 4 bentuk yaitu: huruf, gambar, suara dan video. Peralatan output akan menghasilkan informasi untuk manusia sehingga bisa dibaca, dilihat, didengar dan dipahami.
Komputer akan memproses informasi dari peralatan input komputer kemudian menampilkan hasil proses dalam bentuk informasi ke pengguna komputer (manusia). Peralatan output sendiri sangat penting karena akan memberikan informasi secara langsung ke pengguna. Perangkat output juga menjadi suatu keharusan dalam sistem komputer meskipun beberapa peralatan output komputer tersebut dalam kategori periperal komputer / perangkat tambahan.

3A. Exekutiv  :  Berisi  dasar  layanan  sistem  operasi,  seperti  manajemen   memori,  proses  dan  manajemen  thread, keamanan,  I/O,  dan komunikasi interprocess.

28. Suatu peranti dihubungkan ke komputer dan peranti tersebut tidak dikenali oleh komputer tersebut. maka, hal ini sesuai dengan fungsi system operasi sebagai ....


Sistem operasi berfungsi sebagai penghubung antara perangkat keras (hardware) dan perangkat lunak (software) pada komputer. Saat perangkat dihubungkan ke komputer, sistem operasi akan mencoba untuk mengenali perangkat tersebut dengan memeriksa informasi perangkat keras dan mencocokkannya dengan driver atau program pengontrol yang sesuai.

Jika perangkat tersebut tidak dikenali oleh komputer, maka fungsi sistem operasi sebagai penghubung antara perangkat keras dan perangkat lunak tidak berhasil terlaksana dengan baik. Hal ini menunjukkan adanya kesalahan atau masalah dalam proses pengenalan perangkat keras oleh sistem operasi, dan mungkin memerlukan tindakan seperti instalasi driver atau konfigurasi perangkat agar dapat dikenali dengan baik oleh komputer.

A. Resistor Resistor atau disebut juga dengan Hambatan adalah Komponen Elektronika Pasif yang berfungsi untuk menghambat dan mengatur arus listrik dalam suatu rangkaian Elektronika. Satuan Nilai Resistor atau Hambatan adalah Ohm (Ω).

B. Kapasitor (Capacitor) Kapasitor atau disebut juga dengan Kondensator adalah Komponen Elektronika Pasif yang dapat menyimpan energi atau muatan listrik dalam sementara waktu

C. Induktor (Inductor) Induktor atau disebut juga dengan Coil (Kumparan) adalah Komponen Elektronika Pasif yang berfungsi sebagai Pengatur Frekuensi, Filter dan juga sebagai alat kopel (Penyambung).

D. Dioda (Diode) Diode adalah Komponen Elektronika Aktif yang berfungsi untuk menghantarkan arus listrik ke satu arah dan menghambat arus listrik dari arah sebaliknya. Diode terdiri dari 2 Elektroda yaitu Anoda dan Katoda.
