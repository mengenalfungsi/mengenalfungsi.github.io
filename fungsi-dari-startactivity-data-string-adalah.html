Tipe data ini bisa menampung banyak karakter sekaligus. String adalah tipe data untuk teks yang terdiri dari gabungan huruf, angka, dan berbagai karakter. Fungsi ini digunakan untuk membuat identifier string atau teks.

24. Buat lah sebuah 3 fungsi overloading yang memiliki 1 parameter - Jika parameternya int maka outputnya “Tipe data int, hasilnya [parameter * 100]” - Jika parameternya double maka outputnya “Tipe data double, hasilnya [parameter * 1000]” - Jika parameternya string maka outputnya “Tipe data string, nilainya Hello [parameter]” contoh pada fungsi main : int main() { overloadingFunc(10; overloadingFunc(10.0); overloadingFunc("World); return 0; } outputnya : Tipe data int, hasilnya 10000 Tipe data float, hasilnya 1000 Tipe data string, nilainya Hello World


Jawaban dan Penjelasan:

Overloaded Functions

Saya buat 2 versi. Yang pertama, argumen bertipe string menggunakan tipe "string" dari namespace std. Karena ada baris "use namespace std;", maka nama tipenya "string" saja, tidak perlu "std::string".

Versi kedua, argumen bertipe string menggunakan "char*". Pada saat passing argumen ke fungsi, harus menggunakan casting berupa "(char*)", agar tidak ada warning pada saat kompilasi.

Versi 1: tipe data string menggunakan string dari namespace std (std::string)

#include <iostream>

using namespace std;

void overloadingFunc(int i) {

   cout << "Tipe data int, hasilnya " << (100*i) << endl;

}

void overloadingFunc(double d) {

   cout << "Tipe data double, hasilnya " << (1000*d) << endl;

}

void overloadingFunc(string s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc("World");

   return 0;

}

Versi 2: tipe data string menggunakan char*

Bagian di atas ini persis sama seperti di atas.

void overloadingFunc(char* s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc((char*)"World");

   return 0;

}

33. Yang termasuk fungsi string adalah


Pengertian fungsi string pada excel
fungsi string adalah fungsi yang di gunakan untuk mengambil karakter sebanyak n pada posisi tertentu. fungsi string sendiri terbagi menjadi tiga bagian yaitu fungsi left, fungsi right dan fungsi mid. secara bahasa indonesia artinya left (kiri) mid (tengah) dan right (kanan) yang akan kami bahas satu persatu secara gamblang di bawah ini:
1. Fungsi left, fungsi left ini di gunakan untuk mengambil karakter dari sebelah kiri contoh : = left (x,z)
2. fungsi mid, fungsi mid ini di gunakan untuk mengambil karakter dari tengah contoh : = mid (x,y,z)
3. fungsi right, fungsi right ini di gunakan untuk mengambil karakter dari sebelah kanan contoh : = right (x,z).
Keterangan rumus:
1. x : cell kunci atau cell yang akan di ambil karakternya
2. y : jumlah karakter
3. n : jumlah karakter yang di ambil

Sebagai contoh ilustrasi penggunaan fungsi string pada excel anda perhatikan gambar berikut ini:
Keterangan:
mencari angka 11 berasal dari = left (kode,2)
mencari angka AA berasal dari = (kode,4,2)
mencari angka 31 berasal dari = ri (kode,2)
