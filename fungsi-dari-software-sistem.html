Fungsi dari system software adalah sebagai “jembatan“ atau interface antara perangkat keras komputer dengan aplikasi, yang mengendalikan dan mengelola pengoperasian perangkat keras komputer sesuai instruksi dari aplikasi yang digunakan.
 

Pembahasan

System software adalah software yang “menjembatani” atau sebagai penghubung/interface antara perangkat keras komputer dengan aplikasi. System software mengendalikan dan mengelola pengoperasian perangkat keras komputer, sesuai dengan instruksi dari perangkat lunak aplikasi (user application). Dengan demikian, dapat dikatakan pula bahwa system software berfungsi untuk menjamin program-program aplikasi berjalan dengan benar/semestinya.

Contoh dari system software adalah sistem operasi (operating system), seperti Microsoft Windows, OS X, Linux, Unix, dsb.

14. Sebutkan dan jelaskan 3 fungsi software sistem operasi


Jawaban:

1. Menjalankan operasi dasar

2. Mengatur kerja hardware dan software

3. Mengkoordinasi kerja perangkat komputer

Penjelasan:

1. Menjalankan operasi dasar

Fungsi utama dari sistem operasi adalah menjalankan operasi dasar pada komputer. Sistem ini dinilai sebagai komponen vital yang mendasari kerja perangkat lunak atau software lainnya.

Sebelum aplikasi berjalan dan dapat berfungsi pada suatu komputer, maka sistem operasilah yang memungkinkan program atau aplikasi tersebut dapat berjalan dan ditampilkan kepada pengguna atau user yang menggunakan perangkat komputer tersebut.

2. Mengatur kerja hardware dan software

Bisa dikatakan bahwa sistem operasi adalah Resource Manager dalam perangkat komputer.

Maksudnya, sistem operasi berfungsi mengontrol fungsi perangkat keras seperti memori, CPU, harddisk, dan perangkat keras lainnya dan juga mengatur fungsi program software agar terhubung dengan perangkat keras tersebut.

3. Mengkoordinasi kerja perangkat komputer

Selain mengatur perangkat keras dan perangkat lunak agar terhubung satu sama lain, sistem operasi juga berfungsi mengkoordinasikan segala hal didalam komputer terutama menyusun program yang sifatnya kompleks menjadi lebih sederhana dan berurutan. Sistem operasi memudahkan suatu aplikasi agar dapat bekerja dengan lebih efisien.

29. Software yang berfungsi sebagai jembatan komunikasi antara komputer dengan device tertentudengan kerja sistem yang maksimal adalah


Jawaban:

 

Software yang berfungsi sebagai jembatan komunikasi antara komputer dengan device tertentudengan kerja sistem yang maksimal adalah driver.

 

Pembahasan:

 

Dalam komputasi, driver perangkat adalah program komputer yang mengoperasikan atau mengendalikan jenis perangkat keras (hardware) tertentu yang terpasang pada komputer.

 

Sebuah driver menyediakan antarmuka (interface) antara perangkat lunak untuk perangkat keras, dan memungkinkan sistem operasi dan program komputer lainnya untuk mengakses fungsi perangkat keras tanpa perlu mengetahui fungsi atau perintah yang terperinci dari perangkat keras yang digunakan.

 

Ini memudahkan programmer dalam menyusun program, karena tidak perlu memberi perintah spesifik ke perangkat keras dalam kode programnya.

 

Driver berkomunikasi dengan perangkat melalui bus komputer (computer bus) atau subsistem komunikasi yang terhubung dengan perangkat keras. Saat program memanggil memanggil suatu fungsi di driver, driver mengeluarkan perintah ke perangkat.  

 

Jenis driver tergantung pada perangkat keras dan sistem operasi. Sehingga setiap perangkat keras memiliki driver sendiri, demikian pula system operasi yang berbeda memerlukan driver yang berbeda.

 

Perangkat keras yang biasanya menggunakan driver adalah:

 

1.    Printer

2.    Scanner

3.    Adaptor video

4.    Kartu jaringan (network card)

5.    Kartu suara (sund card)

6.    Perangkat input mouse dan keyboard

7.    Perangkat penyimpanan komputer seperti hard disk dan CD-ROM

8.    Kamera digital

38. Apa 3 komponen dari sistem software? Dan apa fungsi dasar dari tiap tiap komponen tersebut?


Software dalam komputer adalah komponen penting karena berperan menjalankan segala perintah yang masuk ke hardware.

Software dibagi menjadi tiga yaitu software sistem, software aplikasi dan software tambahan.

Software sistem merupakan software utama yang sangat penting karena bertugas mengatur komponen pada komputer secara menyeluruh. Contoh software diantaranya adalah Windows Xp, Windows 2000, Windows 7, Linux, IOS dan lain-lain.

Software aplikasi adalah merupakan software pelengkap pada komputer yang digunakan sebagai sarana pengolahan data. Contohnya adalah web browser, software office, game dan lainnya.

Sedangkan software tambahan adalah software yang menjalankan tugas-tugas khusus atau tambahan sekaligus melindungi hardware. Contohnya seperti Data recovery, Disk Defragmenter, Sceensever, Backup, dan lain-lain.

3 komponen dari sistem software

Software sistem.Software aplikasi.Software tambahan.

Fungsi software sistem adalah Untuk membuat komputer dapat menjalankan perintah pengguna.

Fungsi software aplikasi adalah Digunakan untuk mengkoordinasikan tiap komponen dan fungsi komputer.

Fungsi software tambahan adalah Untuk membantu pengelolaan komputer, seperti mengkompres file, melindungi sistem, memformat disket, mengkopi data, mengatur harddisk, dan lain-lain.
