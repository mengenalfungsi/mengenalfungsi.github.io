2. bagaimana cara menentukan domain dan range dalam fungsi?


domain = daerah asal
range = daerah hasil
contoh soal
suatu fungsi f(x) = 2x + 3 dengan daerah asal { 1,2,3,4,5} dan daerah kawan { 1 < x < 15} maka daerah hasilnya adalah
f(1) = 2x + 3
       = 2(1) + 3
       = 2 + 3
       = 5
f(2)  = 2(2) + 3
       = 4 + 3
       = 7
f(3) = 2(3) + 3
       = 6 + 3
       = 9
f(4) = 2(4) + 3
       = 8 + 3
       = 11
f(5) = 2(5) + 3
       = 10+3
       = 13
jawabannya { 5,7,9, 11,13}

28. cara menentukan domain dan range dari f(x)=2x+5​


Jika diberikan fungsi f(x) = 2x + 5, maka:

- Untuk menentukan domain caranya adalah cari nilai x berapa saja yang membuat fungsi f(x) terdefinisi atau memiliki nilai.

Karena berapapun nilai x yang diberikan membuat fungsi tersebut memiliki nilai (misal x = -1, 0, 1/2, [tex]\sqrt{2}[/tex], dst) maka domain fungsi tersebut adalah semua bilangan riil (R) atau dituliskan:

[tex]D_f=[/tex] {[tex]x[/tex] | [tex]x[/tex] ∈ R}

- Untuk menentukan range kita tinggal melihat Df nya saja, karena Df adalah semua bilangan riil maka rangenya juga semua bilangan rill.

36. tentukan domain dan range fungsi linear berikut! . . tolong bantu yaa☹️☹️


Fungsi adalah pemetaan setiap anggota sebuah himpunan (yang dinamakan sebagai domain atau daerah asal) kepada anggota himpunan yang lain (yang dinamakan sebagai kodomain atau daerah kawan). Konsep fungsi adalah salah satu konsep dasar matematika.

Anggota himpunan yang dipetakan dapat berupa apa saja (kata, orang, atau objek lain), namun biasanya yang dibahas adalah besaran matematika seperti bilangan riil.

Pembahasanf(x) = 2x + 5

Fungsi f(x) = 2x + 5 tidak mempunyai batas domain, sehingga domain fungsi tersebut -∞ < x < ∞.

Range dari fungsi tersebut adalah -∞ < x < ∞.

f(x) = 3 - x

Fungsi f(x) = 3 - x tidak mempunyai batas domain, sehingga domain fungsi tersebut -∞ < x < ∞.

Range dari fungsi tersebut adalah -∞ < x < ∞.

f(x) = [tex]\frac{1}{2}[/tex]x - 4

Fungsi f(x) = [tex]\frac{1}{2}[/tex]x - 4 tidak mempunyai batas domain, sehingga domain fungsi tersebut -∞ < x < ∞.

Range dari fungsi tersebut adalah -∞ < x < ∞.

f(x) = 5 - 3x

Fungsi f(x) = 5 - 3x tidak mempunyai batas domain, sehingga domain fungsi tersebut -∞ < x < ∞.

Range dari fungsi tersebut adalah -∞ < x < ∞.
