2. apa perbedaan gambar dengan format jpg,png,dan psd ?


Jawaban:

JPG cocok untuk foto-foto dengan kompresi yang dapat diterima, PNG cocok untuk gambar dengan lapisan atau transparansi, dan PSD cocok untuk pekerjaan penyuntingan dan pengembangan gambar yang lebih lanjut.

Penjelasan:

Perbedaan utama antara format gambar JPG (JPEG), PNG, dan PSD adalah sebagai berikut:

1. Format JPG (JPEG):

- Format kompresi yang digunakan untuk mengurangi ukuran file gambar.

- Ideal untuk gambar dengan banyak warna dan detail, seperti foto-foto.

- Mendukung kompresi lossy, yang dapat mengurangi kualitas gambar jika tingkat kompresinya tinggi.

- Tidak mendukung lapisan atau transparansi.

2. Format PNG:

- Format gambar yang mendukung transparansi dan lapisan.

- Cocok untuk gambar-gambar dengan latar belakang transparan atau gambar-gambar dengan elemen-elemen grafis yang perlu disimpan dalam lapisan terpisah.

- Format lossless, yang berarti kualitas gambar tetap terjaga tanpa hilangnya data saat disimpan.

- Tidak seefisien JPG dalam mengompresi gambar dengan banyak warna dan detail, sehingga ukuran file biasanya lebih besar.

3. Format PSD (Adobe Photoshop):

- Format yang spesifik untuk program Adobe Photoshop.

- Memungkinkan penyimpanan berbagai elemen gambar, lapisan, efek, dan informasi lainnya yang dapat diedit kembali dalam Photoshop.

- Tidak umum digunakan sebagai format distribusi gambar karena ketergantungannya pada perangkat lunak Photoshop.

- Lebih berfokus pada penyuntingan dan pengembangan gambar daripada distribusi gambar.

Jadi, pemilihan format gambar tergantung pada kebutuhan Anda. JPG cocok untuk foto-foto dengan kompresi yang dapat diterima, PNG cocok untuk gambar dengan lapisan atau transparansi, dan PSD cocok untuk pekerjaan penyuntingan dan pengembangan gambar yang lebih lanjut.

30. pengertian format gambar PSD,EPS,JPEG,GIF,PNG,TIFF,BMP


JPEG (Joint Photographic Expert Group) adalah standar kompresi file yang dikembangkan oleh Group Joint Photographic Experts; menggunakan kombinasi DCT dan pengkodean Huffman untuk mengkompresikan suatu file citra. JPEG adalah suatu algoritma kompresi yang bersifat lossy, (yang berarti kualitas citranya agak kurang bagus).

Graphics Interchange Format(GIF) adalah sebuah format yang sering digunakan dalam dunia web maupun dalam dunia citra digital. Format ini sering digunakan karena ukurannya yang relati kecil dan juga banyaknya software editor gambar yang telah mendukung citra ini. GIF berukuran kecil karena membatasi jumlah warnanya sebanyak 256 warna sehingga dapat menghemat ukuran berkas. 

BMP merupakan file gambar yang tidak terkompresi. Dan karena itu ukuran file BMP biasanya besar. File berukuran besar tidak tepat untuk transfer di internet karena akan memenuhi bandwidth dan menjadi lambat.

The TIFF (Tagged Image File Format) adalah format gambar yang fleksibel biasanya menyimpan 16-bit per warna - merah, hijau dan biru untuk total 48-bit - atau 8-bit per warna - merah, hijau dan biru untuk total 24-bit - dan menggunakan nama file atau perpanjangan TIFF TIF. TIFF yang kedua adalah fleksibilitas fitur, dan kutukan, dengan tidak ada satu pembaca semua mampu menangani berbagai jenis file TIFF. TIFF dapat lossy atau lossless. 

(Portable Network Graphics) adalah salah satu format penyimpanan citra yang menggunakan metode pemadatan yang tidak menghilangkan bagian dari citra tersebut (Inggris lossless compression).
