7. tahap-tahap membuat surat dalam Microsft word​


Jawaban:

Berikut adalah tahap-tahap membuat surat dalam Microsoft Word berdasarkan beberapa sumber:

1. Buka Microsoft Word dan pilih menu "File" untuk membuat dokumen baru.

2. Pilih "Surat" atau "Letter" pada daftar jenis dokumen yang tersedia.

3. Pilih jenis kertas yang akan digunakan, seperti ukuran A4 atau Letter.

4. Buat kop surat dengan mengetikkan nama dan alamat instansi pada bagian atas dokumen. Gunakan fitur "Align Center" untuk membuatnya rapi.

5. Buat tanggal surat pada bagian atas kanan dokumen.

6. Buat salam pembuka, seperti "Kepada Yth." atau "Dear".

7. Buat isi surat dengan mengetikkan pesan atau informasi yang ingin disampaikan.

8. Buat salam penutup, seperti "Hormat saya" atau "Salam hangat".

9. Tandatangani surat jika diperlukan.

10. Simpan dokumen dengan nama yang sesuai dan format yang diinginkan.

8. sebutkan dan jelaskan aikon yang ada pada menu home pada microsft word


Group Clipboard terdiri dari sekumpulan menu yaitu:Paste : untuk menempelkan hasil copy atau cut.Cut : untuk memotong teks/objek terpilih.Copy : untuk menggandakan teks/objek terpilih.Format Painter : digunakan untuk meniru format halaman dokumen ke dalam dokumen lainnya.

Menu-menu di Group Font secara khusus berfungsi untuk pemformatan seputar huruf. Terdiri dari :Font : untuk memilih jenis hurufFont Size : untuk mengatur ukuran hurufGrow Font : untuk memperbesar ukuran huruf secara instanShrink Font : untuk memperkecil ukuran huruf secara instanChange Case : untuk mengubah status huruf kapital/huruf kecilClear Formatting : untuk menghapus pemformatan teks terpilihBold : untuk menebalkan teks terpilihItalic : untuk memiringkan Teks terpilihUnderline : untuk memberikan garis bawah pada teks terpilihStrikethrough : memberikan tanda coret padateks terpilihSubscript : untuk mengetik karakter pemangkatanSuperscript : untuk mengetik karakter pemangkatanText Effect : untuk memberikan efek artistik pada teks terpilihText Highlight Color : untuk memberikan warna stabilo di belakang teks terpilihFont Color : untuk mengatur warna teks terpilih.

Group Paragraph terdiri dari sekumpulan menu yang memiliki kesamaan fungsi untuk mengatur paragraf teks. perintah-perintah di Group Paragraph yaitu:Bullets : untuk memberikan tanda bullet di tiap paragraf terpilihNumbering : untuk memberikan format penomoran di tiap paragraf terpilihDecrease Indent : untuk menggeser baris kedua paragraf ke kiriIncrease Indent : untuk menggeser baris kedua paragraf ke kananLeft-to-Right : untuk menjadikan arah teks berjalan dari kiri ke kanan (format Latin)Right-to-Left : untuk menjadikan arah teks berjalan dari kanan ke kiri (format Arab)Sort : untuk menyortir dataShow Paragraph Marks : untuk menampilkan / menyembunyikan tanda koreksi paragrafAlign Text Left : untuk mengatur teks rata kiriCenter : untuk mengatur teks rata tengahAlign Text Right : untuk mengatur teks rata kananJustify : untuk mengatur teks rata kanan-kiriLine Spacing : untuk mengatur jarak antar baris teksShading : untuk mengatur warna latar teks terpilihBorder : untuk memberikan garis tepi pada teks terpilih.

Group Styles terdiri dari dua menu. Yaitu:Heading Styles : berisi pilihan format judul/subjudul dari paragraf terpilihChange Styles : berisi pilihan pengaturan tema paragraf 

Secara umum group Editing terdiri dari Find, Replace, dan Select. fungsinya adalah:Find : untuk mencari kata tertentu berdasarkan keyword tertentu dalam suatu fileReplace : untuk mencari dan mengganti kata yang ditemukan untuk diganti dengan kata tertentu dalam suatu fileSelect : digunakan untuk memilih objek atau teks tertentu di dalam suatu file


9. sebutkan dan jelaskan aikon yang ada pada menu home pada microsft word


- Paste, berfungsi untuk menempatkan objek yang sudah disalin atau dipotong pada tempat kursor berada.
- Copy, berfungsi menyalin objek, biasanya diikuti perintah paste.
- Cut, berfungsi memotong objek, biasanya diikuti perintah paste.
- Font, berfungsi mengetur jenis huruf.
- Font size, berfungsi mengetur ukuran huruf.
- Bold, berfungsi mengatur teks menjadi tebal.
- Italic, berfungsi mengatur teks menjadi miring.
- Underline, berfungsi menambahkan garis bawah pada teks.
- Font Color, berfungsi mengatur warna teks.
- Bullets, berfungsi menambahkan bullet ke paragraf.
- Numbering, berfungsi menambahkan penomoran ke paragraf.
- Decrease Indent, berfungsi mengurangi inden paragraf.
- Increase Indent, berfungsi menambahkan inden paragraf.
- Align Text Left, berfungsi mengatur paragraf menjadi rata kiri.
- Center, berfungsi mengatur paragraf menjadi rata tengah.
- Align Text Right, berfungsi mengatur paragraf menjadi rata kanan.
- Justify, berfungsi mengatur paragraf menjadi rata kiri dan kanan.
- Find, berfungsi menjalankan perintah pencarian.
- Replace, berfungsi menjalankan perintah pencarian dan penggantian kata.

29. Apa saja sejarah sejarah microsft word 2010? PLS JAWAB​


Jawaban:

Microsoft Word sesungguhnya ditulis oleh Richard Brodie, Aplikasi ini diperuntukkan bagi komputer IBM yang menggunakan sistem operasi DOS pada tahun 1983. Pertama kali dirilis untuk umum pada tanggal 2 Mei 1983 dengan sistem operasi DOS. awal kehadirannya memang tidak menguntungkan, karena saat itu pengolah kata masih didominasi oleh Wordperfect yang telah lebih dahulu dirilis pada tahun 1980.

Wordperfect dibuat oleh Bruce Sebastian dan Dr. Alan Ashton pendiri Satellite Software International. Program ini berkembang sangat pesat, hingga pada tahun 1986 dengan kemampuan penomoran paragraf. kemampuan ini sangat dibutuhkan pada dokumen-dokumen di bidang hukum. Kemampuan aplikasi ini terus meningkat hingga Wordprefect 6.0 yang menggunakan mode grafis dengan tampilan Fon WYSIWYG yang cukup menarik.

Ditengah persaingan antara berbagi pengolah kata. inilah saatnya  Microsoft Word terlahir, dengan rival-rival yang terlebih dahulu Menginjak muka bumi ini pada saat bersamaan pula Sistem Operasi Windows Matang, dan diwaktu yang tepat pula, Microsoft Word dikembangkan berdasarkan versi Asli Mac.
