1. Tuliskan 5 bagian dari jendela ms word 2007 beserta fungsiNya


1) office button,untuk membuka menu office
2) baris judul,untuk mengetahui jendela Microsoft Word dan file dokumen yg sedang dibuka pada jendela
3) tombol dialog,untuk membuka kotak dialog serta memodifikasi perintah2 yg ada di dalamnya
4) halaman kerja,untuk tempat mengetik atau mengedit dokumen
5) zoom,untuk mengatur perbesaran atau pengecilan tampilan halaman kerja1. Microsoft Office Button
Apabila tombol ini diklik maka akan muncul beberapa pilihan yaitu :
a. New : berfungsi untuk membuat dokumen baru
b. Open : berfungsi untuk membuka dokumen yang telah dibuat
c. Save : berfungsi untuk menyimpan dokumen yang telah dibuat
d. Save As : berfungsi untuk menyimpan dokumen yang telah dibuat dengan nama berbeda
e. Print : berfungsi untuk mencetak dokumen
f. Prepare : berfungsi untuk mempersiapkan dokumen
g. Send : berfungsi untuk mengirim dokumen
h. Publish : berfungsi untuk mentransfer dokumen

2. Menu bar
Baris ini berguna untuk memberi perintah dengan cara mengklik menu tersebut. Baris ini terdiri dari :
a. Home
b. Insert
c. Page Layout
d. References
e. Mailings
f. Review
g. View

3. Quick Access Toolbar
Menu yang berfungsi untuk mempercepat akses/shortcut

4. Tittle Bar
Baris ini berisi keterangan mengenai program dan nama dokumen yang sedang aktif.

5. Tombol Ukuran
Tombol-tombol ini digunakan untuk mengatur ukuran jendela yang sedang aktif. Ada 4 macam tombol ukuran, yaitu :
a. Minimize, untuk memperkecil ukuran jendela sehingga membentuk sebuah ikon aktif.
b. Maximize, untuk memperbesar ukuran jendela hingga menjadi satu layar penuh.
c. Restore, untuk mengembalikan jendela ke ukuran semula.
d. Close, untuk menutup jendela.

6. Toolbar
Berguna untuk memberi perintah dengan mengklik tombol tersebut.

7. Ruler
Garis pedoman untuk menentukan margin atau batas indentasi.

8. Jendela Dokumen
Di sinilah area kerja tempat kita mengetik naskah.

9. Scroll Bar
Terdiri dari : Vertical scroll bar
Horizontal scroll bar
Berfunsi untuk menggulung dokumen.

10. Status Bar
Pada lajur ini diberikan berbagai informasi, misalnya mengenai letak titik sisip.

11. View Toolbar
Memperlihatkan format-format tampilan dokumen yang sedang dibuat, serta dapat memperbesar dan memperkecil tampilan dokumen.

12. Zoom In / Out
Digunakan untuk memperbesar atau memperkecil ukuran jendela dokumen.

8. Sebutkan 5 bagian dari jendela microsoft word 2007 beserta fungsinya!

Baris penggulung, digunakan untuk menggulung halaman kerja yang sedang ditampilkan secara horisontal dan vertikal.

23. gambar dan tuliskan bagian bagian jendela aplikasi ms word


1)  OFFICE BUTTON : Tombol untuk menampilkan pilihan perintah menu yang berkaitan dengan pengolahan dokumen, seperti : new (membuat dokumen baru), save (menyimpan), dan lain-lain.

2)  QUICK ACCESS TOOLBAR : Merupakan toolbar default yang ditampilkan di bagian kiri atas jendela yang berisikan 4 tombol perintah, yaitu : Save, Undo, Redo,dan Print.

3)  TAB : Komponen yang menampung dan mengelompokan grup-grup yang berisi tombol-tombol perintah.

4)  TITLE BAR : Batang judul lembar kerja atau nama file yang terletak di jendela Microsoft word bagian atas.

5)  VIEW RULER : Bagian fasilitas untuk mengatur garis mistar secara horizontal dan vertical.

6)  RULER : Garis mistar untuk mengatur identasi paragraf.

7)  STATUS BAR : Sebuah batang yang menunjukkan beberapa status, seperti : posisi halaman aktif, total jumlah kata dalam dokumen.

8)  PENGATUR TAMPILAN : Tombol-tombol perintah yang berfungsi untuk mengatur tampilan lembar kerja dokumen.

9)  ZOOM LEVEL : Untuk mengatur ukuran tampilan lembar kerja dokumen.

10)LEMBAR KERJA : Area untuk mengetik dan mengolah teks, tabel, objek. Grafik, dan sebagainya.

11)TAB STOP : Untuk mengatur jarak teks dari margin kiri.

12)GRUP : Bagian dari tab yang berisi tombol-tombol perintah untuk menjalakan aksi tertentu.

13)TOMBOL KOTAK DIALOG : Merupakan tombol dengan simbol panah bawah pada sudut kanan bawah grup yang berfungsi untuk menampilkan kotak dialog.

14)TOMBOL PENGATUR JENDELA : Tombol pengatur jendela terdiri dari :

Minimize : untuk meminimalkan jendela.
Maximize : untuk memaksimalkan jendela.
Close : untuk menutup jendela.

15)TOMBOL HELP : Untuk menampilkan tutorial tentang bagaimana menggunakan Microsoft Word 2007.

16)BARIS PENGGULUNG : Bagian fasilitas untuk menggulung worksheet secara horizontal dan vertical.

32. jelaskan fungsi dari bagian bagian tampilan ms word 2007


- Title Bar: berfungsi untuk sebagai judul dan sebagai penamaan dokumen yang aktif

- Quick Access Toolbar: berfungsi sebagai tempat icon-icon yang paling sering digunakan

- Office Button: berfungsi seperti halnya menu File (seperti pada versi office sebelumnya)

- Ribbon: berfungsi sebagai tempat untuk tab-tab perintah di dokumen aktif

- Tab Names: berfungsi sebagai penamaan tab pada ribbon

- Microsoft Office Word Help: berfungsi sebagai tombol untuk mencari bantuan office Word

- View Ruler Icon: berfungsi untuk menampilkan dan menyembunyikan penggaris (ruler)

- Ruler: berfungsi sebagai penanda ukuran kertas

- Mini Toolbar: berfungsi sebagai pengganti toolbar, sehingga memudahkan pengguna MS. Office Word dalam mengolah data naskah.

- Workspace: berfungsi sebagai tempat naskah/ dokumen Ms. Word diolah.

- Insertion Point: berfungsi sebagai penanda lokasi kursor yang aktif.

- Scroll Bar: berfungsi sebagai penggulung tampilan dokumen (bisa secara vertikal maupun horizontal)

- Zoom: berfungsi sebagai pengatur besarnya/ kecilnya tampilan dokumen di layar monitor

- View Buttons: berfungsi untuk mengatur tampilan dokumen dengan 5 pilihan utama

- Status Bar: berfungsi untuk menampilkan nomor halaman yang aktif, jumlah kata, pemeriksaan ejaan serta bahasa (language) yang digunakan
