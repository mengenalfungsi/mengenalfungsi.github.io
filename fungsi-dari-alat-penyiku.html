7. Penyiku sudutPQS adalah..


∠PQS + ∠SQR = 180°
2x° + 15° + 4x° + 9° = 180°
6x° + 24° = 180°
6x° = 180° - 24°
6x° = 156°
x = 156° / 6°
x = 26°

∠PQS = 2x + 15°
            = 2(26°) + 15°
            = 67°

Penyiku PQS = 90° - 67°
                       = 23°
