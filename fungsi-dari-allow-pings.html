Untuk membuat pilihan sel dalam bentuk dropdown list atau sering juga disebut combo box, ikuti langkah-langkah berikut :

Pilih (seleksi) Sel/Range dimana dropdown list akan ditampilkan.

Pilih Menu Data Validation, kemudian pada Tab Data--Group Data Tools atau gunakan Shortcut excel Alt + A + V + V

Data Validation Excel Menu

Setelah muncul kotak pilihan (option box) "Data Validation" pada bagian dropdown Allow: pilih List.

Cara membuat dropdown list excel 1

Pada bagian Source: tuliskan list atau daftar pilihan data yang akan diijinkan untuk dimasukkan ke dalam cell.

Penjelasan:

Jika list data yang akan kita gunakan relatif sedikit misalnya hanya Ya dan Tidak, atau Laki-laki dan Perempuan, menuliskan daftar list data ke bagian Source List Data Validation tidak akan menjadi masalah.

Namun tentu akan menyulitkan kita jika list data tersebut relatif banyak. Misal nama-nama pelanggan atau customer, kode akun, daftar barang, dan sebagainya. terlebih jika ada kemungkinan data-data tersebut akan berubah suatu saat.

Untuk kebutuhan semacam ini sebaiknya sumber list data jangan ditulis manual langsung. Namun dengan menggunakan referensi range yang mengarah ke list data tersebut. Bisa pada sheet yang sama atau sheet lain yang berbeda.

Berikut cara membuat pilihan di excel yang mengacu pada range tertentu.

Buat dahulu list atau daftar data yang akan kita gunakan untuk dropdown atau combo box pilihan sel ini. Daftar list ini bisa dibuat pada sheet yang sama atau sheet lain yang berbeda. Sebagai contoh saya membuat list nama buah seperti dibawah ini.

Membuat dropdown list dengan referensi range

Langkah berikutnya mirip dengan cara sebelumnya. Bedanya pada bagian Source gunakan referensi absolut yang mengarah pada range list data yang telah kita buat. Perhatikan gambar berikut:

Membuat dropdown list dengan referensi range 2

Selain mengggunakan referensi sel bagian source ini bisa juga kita isi dengan Named range yang panduannya cara membuatnya bisa Anda baca tutorial berikut: 3 Cara Memberi Nama Range Pada Excel.

Misal range buah saya beri nama List_Buah maka pada source tuliskan seperti digambar ini.

Membuat dropdown list dengan referensi named range

Untuk data list data yang dinamis atau ada kemungkinan bertambah maka gunakan nama range yang dinamis. Cara membuat nama range yang dinamis bisa anda pelajari pada bagian ini : Cara Membuat Nama Range Dinamis pada Excel

CARA EDIT DROPDOWN LIST EXCEL

Setelah bisa membuat daftar pilihan atau dropdown list, bagaimana cara mengubahnya?

Untuk edit dropdown list yang telah dibuat caranya sebagai berikut:

Pilih cell atau range yang akan di edit dropdown listnya.

Aktifkan option box "Data Validation" dengan cara memilih tombol Data validation pada Tab Data--Group Data Tools.

Setelah setting data validation muncul silahkan buat perubahan yang Anda inginkan.

Untuk menerapkan perubahan pada semua cell/range yang terpengaruh data validasi tersebut, centang kotak Apply these changes to all other cells with the same settings.

Klik OK.

Cara Edit Dropdown List Excel

CARA MENGHAPUS DROPDOWN LIST EXCEL

Bagaimana cara menghapus pilihan atau dropdown list di excel?

Untuk menghapus dropdown list excel caranya adalah sebagai berikut:

Pilih cell atau range yang akan di hapus dropdown listnya.

Pada Tab Data--Group Data Tools, klik Data Validation.

Apabila hendak mengapus dropdown list untuk semua cell yang terkait(Mempunyai pilihan yang sama), silahkan centang bagian kotak Apply these changes to all other cells with the same settings. Jika hanya ingin menghapus pada cell-cell tertentu yang terpilih saja, abaikan (jangan di centang) kotak pilihan ini.

Klik tombol Clear All untuk menghapus drop down list.

Klik OK untuk menutup kotak pengaturan data validation.

28. my parents allow or allows


Jawaban:

Allow

Penjelasan:

Allow itu untuk yang jamak atau kata ganti "I". Parents itu kan ada huruf s di belakangnya, menandakan kalo itu jamak. Kalo my parent baru pake allows.

┌ allow 」

└► My parents allow

Contoh:

My parents allow me to go to school by car. (Orangtuaku mengizinkanku untuk berangkat sekolah naik mobil)

Pendahuluan

Simple present tense adalah kalimat yang berguna untuk menyatakan suatu kebenaran umum, hal yang terjadi berulangkali, serta hal yang terjadi di masa sekarang.

rumus:

[kalimat nominal]

(+) S + to be + O/C(-) S + to be + not + O/C(?) to be + S + O/C + ?

[kalimat verbal]

(+) S + verb 1 + O/C(-) S + do/does + not + verb 1 + O/C(?) do/does + S + verb 1 + O/C + ?

to be:

is ⇸ subjek he/she/itam ⇸ subjek Iare ⇸ subjek you/we/they

auxiliary:

do ⇸ subjek I/you/we/theydoes ⇸ subjek he/she/it

PEMBAHASAN

Pada soal, diminta untuk menentukan verba yang tepat untuk melengkapi kalimat berikut.

| My parents (allow/allows) |

Perlu diketahui, bahwa verba yang digunakan pada simple present tense afirmatif dan apabila subjeknya he/she/it perlu berakhiran -s/es. Contohnya: reads, allows, cooks, walks, brings, carries.

Subjek 'my parents' adalah subjek jamak – yang berarti termasuk subjek I/you/we/they. Sehingga, verba yang digunakan tidak perlu berakhiran -s/es.

Maka, kata kerja yang benarnya adalah allow, bukan allows.

→ My parents allow ...
