(3x-y) (x+y)= 3 x (X + Y )-Y ( X + Y )

=3x²+3xy-xy-y²

=3x²+2xy-y²

Penjumlahan
[tex]2x + 2y + 10 + 5 + x + 3y\\ = 2x + x + 2y + 3y + 10 + 5 \\ = 3x + 5y + 15[/tex]
Pengurangan
[tex]2x - 2y - 10 - 5 - x - 3y\\ = 2x - x - 2y - 3y - 10 - 5 \\ = x - y - 25[/tex]
Perkalian
[tex]2x \times 2y \times 10 \times 5 \times x \times 3y\\ = 2x \times x \times 2y \times 3y \times 10 \times 5 \\ = 2{x}^{2} \times 6 {y}^{2} \times 50 \\ = 12( {xy})^{2} \times 50 \\ = 600( {xy})^{2} [/tex]

= 6y - 4y + 2a + 3a

= 2y + 5a

2. 2y(3y - 5y) = (-4y)

= 2y(3y) - 2y(5y)

= 6y - 10y

= (-4y)

3. 3a + 5b + 7a - 2b = 10a + 3b

= 3a + 7a + 5b - 2b

= 10a + 3b

4. (x + y)(x + y) = x² - y²

= x × x . y x y

= x² - y²

5. -3a - 2b + 4a + 9b = a + 7b

= -3a + 4a - 2b + 9b

= 1a + 7b

= a + 7b

6. -7b - 2a + 4b - 3a = -3b - a

= -7b + 4b - 2a - 3a

= -3b - 1a

= -3b - a

7. 3b² x 2b² - 4a + a = 6b⁴ - 5a

= 6b⁴ - 5a

8. 62a - 61a + 72b - b = a + 73b

= 1a + 73b

= a + 73b

9. 4a - 5(2 + 4) = 4a - 30
