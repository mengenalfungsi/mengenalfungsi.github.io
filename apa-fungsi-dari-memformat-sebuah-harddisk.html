13. Uraikan dengan singkat mengenai tahapan dalam melakukan format data Harddisk : NTFS dan NTFS22.


Penjelasan:

NTFS adalah salah satu format pemetaan data yang disarankan dan dibuat oleh Microsoft Windows, untuk media penyimpanan diatas 4GB.

Cara memformat media penyimpanan:

1. Buka Windows Explorer

2. Klik kanan pada partisi yang ingin diformat (misal Drive D:)

3. Pada kotak dialog "Format", pilih NTFS

4. Isikan Volume Label (nama drivenya)

5. Klik "Start", dan proses pemformatan akan dimulai


14. apa bedanya format harddisk gpt dan mbr?


GPT(GUID Partition Table) MBR (Master Boot Record)

perbedaan nya :

1. MBR hanya men-support 4 partisi primary, GPT jauh lebih banyak yaitu bisa sampai 128 partisi primary.

2. Kapasitas penyimpanan MBR sangat terbatas, yaitu 2 TB saja. Sedangkan GPT bisa sampai 9,4 ZB(Zeta byte).

3. Pada partisi MBR, jika kita ingin menggunakan lebih dari 4 partisi, kita harus mengorbankan satu partisi primari untuk dijadikan extended, baru kemudian bisa membuat banyak partisi logical. Untuk di GPT, kita tidak perlu melakukan hal seperti itu.

4. Pada MBR hanya sektor pertama saja yang menyimpan informasi mengenai partisi dan sistem operasi.Tetapi di GPT, informasi tersebut dibackup lebih dari satu kali, sehingga meminimalisir akibat jika sektor pertama rusak.

5. MBR dapat digunakan oleh semua OS. Hanya sistem operasi windows XP 64bit dan windows yang kompatibel saja yang dapat menjalankan GPT.

6. Untuk dukungan booting 32-bit di GPT, hanya dimiliki oleh windows 8. Sedangkan sistem operasi lain seperti Windows 7, Vista, tidak memiliki dukungan booting 32-bit dari GPTDi format itu d hilangkan semua file nya

19. format partisi harddisk yang sesuai dengan sistem operasi linux


Partisi Primary : merupakan jenis partisi utama diharddisk untuk system operasi umumnya. Partisi primary hanya bisa dibuat maksimal 4 partisi, hal ini sangat berbeda dengan sistem DOS yang hanya mengijinkan satu jenis partisi primary untuk system. Jenis partisi ini menempati nomor partisi 1, 2, 3 dan 4. Misalnya harddisk sda bisa dibuat sda1, sda2, sda3 dan sda4.

Partisi Extended : Partisi Extended berfungsi untuk mengatasi keterbatasan pembagian partisi. Partisi Extended tidak menangani pengolahan data secara langsung. Untuk dapat menggunakannya, kita harus menciptakan Partisi Logical terlebih dahulu.

Partisi Logical : Partisi yang akan selalu dibuat didalam partisi extended. Nomor partisi akan selalu dimulai  5 dan seterusnya. Jika ada 3 jenis partisi logical maka masing-masing akan menempati sda5, sda6, dan sda7.

20. Saat kita ingin memprastisi harddisk untuk install ulang sistem operasi, pastikan kita memilih yang benar untuk dibagi. Jika kita melakukan format apa yang akan terjadi.


Jawaban:

Ketika kita sedang mempartisi dardisk maka selalu untuk memastikan dalam memilih langkah dengan benar. Karena jika kita memilih format yang akan terjadi yakni Data akan hilang dan tidak bisa kembali lagi. Jadi, ketika melakukan partisi hardisk sebaiknya dilakukan dengan hati-hati dan tidak salah melakukan partisi supaya meminimalisi resiko kehilangan semua data.

Pembahasan :

Partisi hardisk sendiri ialah salah satu metode yang digunakan untuk membagi hardisk pada komputer agar pengelolaan data lebih efektif. Partisi pada komputer diibaratkan sebuah ruang-ruang, di mana ruang tersebut mempunyai isinya masing-masing. Adapun besarnya jumlah partisi di dalam komputer dapat ditentukan sendiri dengan cara yang efektif.

Berikut ini langkah-langkah membagi partisi hardisk menggunakan windows 10:

Pertama, buka Power User (WinKey + X), kemudian pilih Disk Management.

Kedua, pilih salah satu dari partisi tersebut yang ingin dialokasikan untuk membuat partisi baru lagi dengan cara klik kanan lalu pilih Shrink Volume.

Ketiga, tentukan jumlah size yang akan dijadikan partisi baru. Misalnya, ingin membuat partisi sebesar 30GB, maka angka yang dituliskan adalah 30000 (dalam satuan MB). Lalu setelah selesai klik Shrink.

Keempat, untuk membuatnya menjadi partisi yang siap digunakan maka lanjutkan kembali prosesnya dengan cara klik kanan lalu pilih New Simple Volume.

Terakhir, ikuti wizard untuk menentukan size volume, letter drive, dan nama dari partisi tersebut hingga selesai. Dan partisi siap digunakan.

30. Apakah fungsi harddisk, motherboard, RAM, CPU, dan VGA?


Fungsi :  

Harddisk adalah untuk menyimpan data dan sebagai meida penyimpanan utama yang menampung kerja sistem operasi.  Motherboard adalah untuk  mengendalikan serta mengelola kinerja dan lalu lintas keseluruhan komponen yang terkait. Selain itu distribusi data listrik pada tiap komponen juga diatur motherboard Random Access Memory berfungsi sebagai tempat penyimpanan data sementara saat suatu perinta sedang berjalan. RAM sendiri bukanlah media penyimpanan yang mampu menampung data secara permanen dalam jumlah besar. CPU  berfungsi sebagai tempat menjalankan perintah dan program yang ada di penyimpanan utama dengan fetch and execute. VGA berfungsi sebagai konverter sinyal digital menjadi grafis pada output monitor. Secara khusus juga VGA memproses semua yang berhubungan dengan tampilan. Pembahasan  

Harddisk, motherboard, RAM, CPU dan VGA merupakan bagian dari perangkat keras komputer yang kinerjanya diatur oleh sistem operasi.  Komponen - komponen diatas merupakan bagian dari processing device yang berguna dalam mengolah inputan agar dapat menjadi output yang akan dikeluarkan oleh output device
