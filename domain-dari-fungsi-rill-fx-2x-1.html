Jawaban : Daerah hasil = { -13, -3, 3, 5 }

Perhatikan penjelasan berikut.

Ingat! Daerah hasil atau range adalah himpunan yang berisi pasangan di daerah asal atau domain. Jadi untuk mencari daerah hasil, dapat disubstitusikan nilai domain ke dalam fungsi.

Pembahasan:

Fungsi : f(x) = 5 - 2x²

Domain : { x | -2 ≤ x ≤ 3, x bilangan bulat} = { -2, -1, 0, 1, 2, 3}

Dengan memasukkan masing-masing nilai domain ke dalam persamaan didapatkan:

f(-2) = 5 - 2(-2)² = 5 - 2(4) = 5 - 8 = -3

f(-1) = 5 - 2(-1)² = 5 - 2(1) = 5 - 2 = 3

f(0) = 5 - 2(0)² = 5 - 2(0) = 5 - 0 = 5

f(1) = 5 - 2(1)² = 5 - 2(1) = 5 - 2 = 3

f(2) = 5 - 2(2)² = 5 - 2(4) = 5 - 8 = -3

f(3) = 5 - 2(3)² = 5 - 2(9) = 5 - 18 = -13

Jadi, daerah hasil fungsi tersebut adalah { -13, -3, 3, 5 }

Diberikan fungsi f(x) = 2x + 1. Untuk menentukan domain fungsi, kita harus memperhatikan batasan-batasan pada variabel x yang membuat fungsi memiliki nilai yang terdefinisi.

Penjelasan dengan langkah-langkah:

Diketahui:

a.) f(x) = sin 2x

b.) f(x) = 3 sin 2x

c.) f(x) = 4 + 3 sin 2x​

Ditanya:

Tentukan periode, nilai maksimum dan minimum dari fungsi-fungsi tersebut?

Pembahasan:

Fungsi trigonometri bentuk [tex]f(x)=a \sin k(x\pm b)\pm c[/tex] mempunyai periode [tex]=\frac{2\pi}{k}[/tex], nilai maksimum [tex]=|a|+c[/tex], nilai minimum [tex]=-|a|+c[/tex].

a. [tex]f(x) = \sin 2x[/tex]

Periode [tex]=\frac{2\pi}{k}=\frac{2\pi}{2}=\pi[/tex]

Nilai maksimum [tex]=|a|+c=1+0=1[/tex]

Nilai minimum [tex]=-|a|+c=-1+0=-1[/tex]

b. [tex]f(x) = 3\sin 2x[/tex]

Periode [tex]=\frac{2\pi}{k}=\frac{2\pi}{2}=\pi[/tex]

Nilai maksimum [tex]=|a|+c=3+0=3[/tex]

Nilai minimum [tex]=-|a|+c=-3+0=-3[/tex]

a. [tex]f(x) =4+3 \sin 2x=3\sin 2x +4[/tex]

Periode [tex]=\frac{2\pi}{k}=\frac{2\pi}{2}=\pi[/tex]

Nilai maksimum [tex]=|a|+c=3+4=7[/tex]

Nilai minimum [tex]=-|a|+c=-3+4=1[/tex]
