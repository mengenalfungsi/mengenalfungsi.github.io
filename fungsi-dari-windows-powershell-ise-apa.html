11. Masih sama seperti tugas kemarin, namun sekarang dengan Powershell. Buatlah script Powershell yang dapat menampilkan barisan bilangan Fibonacci hingga suku ke-n. (Nilai n diperoleh dari parameter script.) ​


Script Powershell: fibonacci.ps1

# fibonacci.ps1
# oleh: hy

# Get-Fibonacci($n)
# Mengembalikan array berisi
# barisan bilangan Fibonacci,
# mulai dari suku ke-0 hingga
# suku ke-$n.
function Get-Fibonacci {
   param (
       $n
   )
   $F = 0 .. $n
   for ($i = 0; $i -le $n; $i++) {
       if ($i -lt 2) {
           $F[$i] = $i
       }
       else {
           $F[$i] = $F[$i-1] + $F[$i-2]
       }
   }
   $F
}

$n = $args[0]
Write-Output "Barisan bilangan Fibonacci hingga suku ke-${n}:"
(Get-Fibonacci($n)) -join ', '
_____________

Pembahasan

Seperti jawaban saya yang lalu (shellscript fibonacci : https://brainly.co.id/tugas/51769747), relasi rekurensi pada barisan bilangan Fibonacci didefinisikan oleh:

[tex]\begin{aligned}F_n=\begin{cases}0,&{\rm untuk\ }n=0\\1,&{\rm untuk\ }n=1\\F_{n-1}+F_{n-2},&{\rm untuk\ }n > 1\end{cases}\end{aligned}[/tex]

Pada script Powershell di atas, fungsi Get-Fibonacci dengan parameter $n mengembalikan array yang berisi barisan bilangan Fibonacci dari suku pertama (suku ke-0) hingga suku ke-$n. Jadi, misalnya $n = 10, maka array $F akan berisi suku ke-0 hingga suku ke-10 dari barisan bilangan Fibonacci. Banyak elemen array $F adalah 11.

Script fibonacci.ps1 menerima 1 argumen/parameter, yaitu n, dan mencetak barisan bilangan Fibonacci dari suku ke-0 sampai suku ke-n.

24. Berikut adalah fakta tentang c# yang benar adalah A. c# merupakan turunan dari powershell B. c# dapat juga di buat dalam OS Linux C. c# hanya bisa digunakan dalam windows saja D. untuk mengeksekusi c# pengguna harus menggunakan gcc


Jawaban:

Fakta yang benar adalah B. c# dapat juga di buat dalam OS Linux.

C# adalah bahasa pemrograman yang dikembangkan oleh Microsoft. Meskipun awalnya dikembangkan untuk digunakan dalam lingkungan pengembangan perangkat lunak Windows, C# juga dapat digunakan dalam sistem operasi Linux dengan bantuan perangkat lunak seperti Mono atau .NET Core. Mono adalah implementasi open-source dari platform .NET yang memungkinkan pengembangan dan eksekusi program C# di berbagai platform, termasuk Linux.

Pernyataan A tidak benar karena C# bukanlah turunan dari PowerShell. PowerShell adalah bahasa skrip dan lingkungan shell yang dikembangkan oleh Microsoft untuk administrasi dan otomatisasi tugas di lingkungan Windows.

Pernyataan C juga tidak benar. Meskipun C# pertama kali dikembangkan untuk digunakan di Windows, dengan adanya implementasi seperti Mono atau .NET Core, C# dapat digunakan pada berbagai sistem operasi, termasuk Windows, Linux, dan macOS.

Pernyataan D juga tidak benar. Untuk mengeksekusi program C#, Anda tidak menggunakan gcc (GNU Compiler Collection), tetapi Anda menggunakan kompiler C# seperti Microsoft's .NET Compiler atau Roslyn Compiler.
