2. How to choose repository and non repository in turnitin ?


The Turnitin administrator can select which paper repository options appear within their account. You are able to allow instructors to choose which paper repository assignment options they use on each assignment or have all student papers automatically be added to the standard repository. The Turnitin administrator has three options: enable instructor standard repository options, enable instructor expanded repository options, or submit all papers to the standard repository.

Log into Turnitin as the administrator and click on the edit icon to the right of the account name.



Select one of the three options from the paper repository options: enable instructor standard repository options, enable instructor expanded repository options, or submit all papers to the standard repository.


If submit all papers to the standard repository is selected from the paper repository options pull-down menu, all student papers submitted to the account will be stored in the standard paper repository.

If enable instructor standard repository options is chosen instructors will be able to set the assignment option to either store student papers within the standard paper repository or to not store the papers in any repository.

If enable instructor expanded repository options is chosen, instructors will be able to set an assignment options to store student papers in the standard paper repository, in the institution paper repository, no repository, or to allow students to choose between the standard paper repository and the institution paper repository.

Allowing students to choose between the standard paper repository and the institution paper repository option is not available with all integrations.

Click on Submit in the bottom left corner of the account modification window to save any changes made.

3. funsi menu annotation​


Menu annotation merupakan fitur yang dapat digunakan untuk menambahkan catatan atau anotasi ke dalam dokumen atau file gambar. Fungsi utama menu annotation adalah untuk memberikan komentar, saran, atau instruksi tambahan pada dokumen atau file gambar tersebut.

Beberapa contoh penggunaan menu annotation adalah:

Memberikan komentar atau saran pada dokumen draft: Menu annotation dapat digunakan untuk memberikan komentar atau saran pada dokumen draft yang sedang dikerjakan. Ini dapat membantu untuk memperbaiki atau menyempurnakan dokumen tersebut sebelum diterbitkan atau dikirimkan kepada pihak lain.Menambahkan instruksi tambahan pada file gambar: Menu annotation dapat digunakan untuk menambahkan instruksi tambahan pada file gambar, seperti menunjukkan bagian yang perlu diperbaiki atau disempurnakan. Ini dapat membantu untuk memberikan petunjuk yang lebih jelas kepada orang lain yang bekerja pada file tersebut.Menambahkan informasi tambahan pada dokumen: Menu annotation juga dapat digunakan untuk menambahkan informasi tambahan pada dokumen, seperti memberikan definisi kata-kata yang tidak dikenal atau menambahkan catatan kaki yang berguna.

Menu annotation biasanya tersedia pada aplikasi pengolah dokumen atau aplikasi pengeditan gambar, seperti Microsoft Word, Adobe Acrobat, atau Adobe Photoshop. Cara menggunakan menu annotation bervariasi tergantung pada aplikasi yang digunakan, namun umumnya terdapat opsi untuk menambahkan catatan baru, membalas catatan yang sudah ada, atau mengubah tampilan catatan yang ditambahkan.

36. Jelaskan Istilah-istilah berikut ini! a. Paket b. Repository c. Kernel d. Terminal Linux e. Mode GUI f. Mode CLI g. Virtualisasi h. Multibooting


a. Paket adalah jatah atau bagian tugas.

b. Repository adalah sekumpulan paket-paket aplikasi atau program untuk sebuah sistem operasi linux yang digunakan untuk menunjang kinerja dari sebuah aplikasi, program, dan sebagainya yang didapatkan dari Server Mirror atau CD/DVD atau media penyimpanan lainnya.

c. Kernel adalah ketika komputer dihidupkan pertama kali (powered on), maka komputer tersebut akan memanggil sistem operasi dari hard disk melalui RAM. Bagian dari sistem operasi yang berinteraksi langsung dengan perangkat keras komputer.

f. Mode CLI adalah antarmuka pada sistem operasi atau komputer yang menggunakan menu baris perintah atau text atau ketikkan dari keyboard untuk berinteraksi denga sistem operasi atau komputer tersebut.

g. Virtualisasi adalah merupakan sustu metode instalasi sistem operasi yang banyak digunakan, karena mempunyai beberapa kelebihan dibandingkan dengan sistem lain. Salah satu kelebihannya adala menghemat resources terutama hardware.

h. Multibooting adalah suatu kondisi dimana dalam satu buah komputer terdapat sistem operasi lebih dari satu sistem operasi.






37. Gan,ane mau nanya.Cara install SSH di centos 6.4 gimana?Tolong pencerahannya,dan satu lg apakah ada repository di centos ? Jangan ngasal


1.Install Open SSH Server and SSH client
# yum -y install openssh-server openssh-clients
2.Start OpenSSH Service
# chkconfig sshd on
# service sshd start
3.OpenSSH Server configuration
# vi /etc/ssh/sshd_config
4.open port 22
# netstat -tulpn | grep :22
5.Edit /etc/sysconfig/iptables (IPv4 firewall),
# vi /etc/sysconfig/iptables
6.Add the line
-A RH-Firewall-1-INPUT -m state --state NEW -m tcp -p tcp --dport 22 -j ACCEPT
