1. jelaskan apa keuntungan dan kerugian memformat disk, partisi disk dan backup disk


keuntungannya adalah kamu bisa dengan aman menyimpan data di partisi lain, kalo sistem rusak gak akan pusing karena hanya partisi utama yaitu c yang rusak bisa diinstal ulang, memudahkan manajemen file pada harddisk .
kerugiaannya adalah jumlah total kapasitas harddisk berkurang, kecepatan akses sangat berkurang, bila tabel partisi rusak maka tabel partisi lain ikut rusak.
semoga membantu :)
keuntungan memformat disk = virus hilang,besarnya kuantitas memory kembali
kerugian memformat disk = data memory hilang

keuntungan membuat partisi disk = data tampak lebih rapi dan tersusun
kerugian membuat partisi disk = data komputer semakin berat

keuntungan back up data  = saat menginstal data yg di back up tidak hilang
kerugian back up data = virus di data yg di back up tidak ikut terformat

Format merupakan proses pengaturan ulang file system pada disk atau partisi, sehingga sistem dapat mengenali dan menggunakan disk atau partisi tersebut. Proses format dapat dilakukan melalui perintah atau utility yang disediakan oleh sistem operasi, seperti perintah format pada sistem operasi Windows atau mkfs pada sistem operasi Linux.

17. Ukuran alokasi disk yang diperlukan sebuah harddisk untuk menyimpan informasi partisinya adalah sebesar...


Jawaban:

Ukuran alokasi disk yang diperlukan sebuah harddisk untuk menyimpan informasi partisinya adalah sebesar kurang lebih 10 skala matriks.  Skala ini bergantung pada seberapa besar ukuran harddisk awalnya.  Karena faktor perbedaan penghitungan antara skala matriks dengan penghitungan bahasa mesin, seringkali akan terjadi selisih ukuran pada partisi harddisk. Informasi partisi sendiri dibagi menjadi tiga, yaitu partisi primer, partisi lanjutan dan partisi logic.

Penjelasan:

Harddisk merupakan perangkat keras komputer yang berperan sebagai media penyimpanan sekunder. Harddisk sendiri dapat berada di dalam komputer maupun digunakan sebagai media penyimpanan eksternal untuk kapasitas yang lebih besar. Harddisk sendiri merupakan piringan logam yang prosesnya akan berputar yang terhubung dengan komponen lain seperti spindle, ribbon cable, connector, power connector dan lainnya sehingga dapat berfungsi dengan baik.

20. Partition berfungsi untuk????


Partition atau partisi merupakan suatu fungsi yang menjelaskan sifat-sifat statistika suatu sistem dalam kesetimbangan termodinamika. Fungsi ini bergantung pada suhu dan parameter-parameter lainnya, seperti volum dan tekanan gas. Kebanyakan variabel-variabel termodinamika dari suatu sistem, seperti energi, energi bebas, entropi, dan tekanan dapat diekspresikan dalam bentuk fungsi partisi atau turunannya.
