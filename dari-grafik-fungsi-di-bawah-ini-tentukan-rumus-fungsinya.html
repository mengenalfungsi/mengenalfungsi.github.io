grafik tsb berbentuk parabola ax² + bx + c, melalui (–3, 0), (1, 0), dan (0, –3)

dari grafik tsb diketahui

f(x) = ax² + bx + c

f(0) = a(0)² + b(0) + c = –3, maka c = –3

f(–3) = a(–3)² + b(–3) + (–3) = 0, maka 9a – 3b = 3

f(1) = a(1) + b(1) + (–3) = 0, maka a + b = 3

mencari nilai a dan b menggunakan metode el-sub:

9a – 3b = 3 |× 1| 9a – 3b = 3

a + b = 3 |× 9| 9a + 9b = 27

9a – 3b = 3

9a + 9b = 27

___________–

–12b = –24

b = 2

a + b = 3

a + 2 = 3

a = 1

diperoleh a = 1, b = 2 dan c = –3

jadi, rumus f(x) = x²+ 2x – 3

3. Tentukan rumus fungsi kuadrat pada grafik dibawah ini​


Titik puncak: (-2,0), xp = -2 & yp = 0

Melalui titik (0,4), x = 0 & y = 4

Mencari nilai a:

y = a(x – xp)² + yp

4 = a(0 + 2)² + 0

4 = 4a

a = 1

Mencari persamaan kuadrat:

y = a(x – xp)² + yp

y = 1(x + 2)² + 0

y = x² + 4x + 4

Sebagai contoh, jika diperoleh nilai a = 2 dan b = 3 dari dua titik pada garis, dan salah satu titik yang diketahui koordinatnya adalah (4, 5), maka nilai c dapat dicari dengan mengganti nilai x dan y dengan 4 dan 5 pada persamaan ax + by = c, sehingga menjadi:

2(4) + 3(5) = c

8 + 15 = c

c = 23

Sehingga persamaan garis tersebut adalah 2x + 3y = 23.

Bentuk umum fungsi kuadrat adalah f(x) = ax² + bx + c dengan a ≠ 0.  

Jika a > 0 maka kurva terbuka ke atas dan jika a < 0 maka kurva terbuka ke bawah

Ada beberapa cara untuk menentukan persamaan fungsi kuadrat, yaitu

Jika diketahui titik puncak (xp, yp) dan melalui titik (x, y)

y = a(x – xp)² + yp

Jika diketahui titik potong terhadap sumbu x di titik (x₁, 0) dan (x₂, 0) serta melalui titik (x, y)

y = a(x – x₁)(x – x₂)

Pembahasan

Dalam gambar kurva terbuka ke bawah maka  

a < 0 (a = bilangan negatif)

Kurva memotong sumbu x di titik (–1, 0) dan (3, 0)

a. sumbu simetri

xp = ½ (x₁ + x₂)

xp = ½ (–1 + 3)

xp = ½ (2)

xp = 1

b. rumus fungsi kuadrat

y = a(x – x₁)(x – x₂)

y = a(x – (–1))(x – 3)

y = a(x + 1)(x – 3)

y = a(x² – 3x + x – 3)

y = a(x² – 2x – 3)

c. nilai optimum

diperoleh dengan mensubstitusikan x = xp = 1

y = a(x² – 2x – 3)

yp = a(xp² – 2xp – 3)

yp = a(1² – 2(1) – 3)

yp = a(1 – 2 – 3)

yp = a(–4)

yp = –4a  

d. titik puncak

(xp, yp)

= (1, –4a)

e. titik potong grafik dengan sumbu y  

Diperoleh jika x = 0

y = a(x² – 2x – 3)

y = a(0² – 2(0) – 3)

y = a(0 – 0 – 3)

y = a(–3)

y = –3a

(0, –3a)

Jadi agar diperoleh nilai a, dalam soal harus diketahui titik lainnya.
