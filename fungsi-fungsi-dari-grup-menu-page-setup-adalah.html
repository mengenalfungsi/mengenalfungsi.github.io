Secara ringkas, Page Setup berfungsi mengatur halaman.

Penjelasan:

Secara rinci, hal yg dapat diatu Page Setup, meliputi:

- Margins (batas-batas tepi halaman)

- Orientation (orientasi halaman, seperti orientasi vertikal & orientasi horizontal)

Kalau dalam Ms. Excel, juga bisa mengatur Print Area (yaitu area tertentu dalam dokumen aktif yg ingin dicetak dengan printer)

16. 3. Secara khusus, menu page setup difungsikan untuk mengatur halaman. Kemukakan fungsi menu margins dan size dalam membangun sebuah setting pengaturan halaman! Jawab: ​


Jawaban:

Menu Margin adalah fitur sub menu pada program microsoft word yang terletak di bagian menu page layout sisi kiri atas yang dapat digunakan ketika pengguna hendak membuat dokumen baru atau membuka dokumen, margin mempunyai fungsi utama untuk mengatur suatu jarak terhadap batas tepi teks dari masing masing kertas yang hendak dibuat oleh pengguna. Caranya dengan klik menu Layout > Pilih Margin > disana kamu bisa setting sesuai dengan kebutuhan dengan cara pilih Custom Margin, kemudian isi bagian Top untuk jarak atas, Left sisi kiri, right kanan dan bottom bagian bawah kemudian klik OK

Menu Size adalah ukuran kertas yang digunakan apakah A3, A4, A5, F4, Tabloid, Legal, atau Letter. Fitur ini harus disesuaikan dengan kertas "nyata" yang kalian gunakan dan/atau pasang pada Printer yang akan digunakan. Cara settingnya dengan klik menu Layout kemudian pilih size atau custom sesuai dengan kebutuhan.

38. Ada 7 simbol menu yang terdapat pada page setup , jelaskan fungsi dari masing-masing ikon​


Jawaban:

-Kegunaan Ikon-ikon menu Page Layout di MS Word-2 (Margin) ikon yang digunakan untuk mengatur batas pinggir halaman. Batas pinggir meliputi batas atas, batas kanan batas kiri dan abats bawah.

-Kegunaan Ikon-ikon menu Page Layout di MS Word-3 (Orientation) ikon ini mengatur bagai mana orientasi kertas. Apakah kertas miring ataukah tegak, hal itu dapat dipilih dari ikon ini.

Kegunaan Ikon-ikon menu Page Layout di MS Word-4 (Size) merupakan ikon untuk mengatur ukuran kertas, ada banyak ukuran standar yang sudah duisediakan, ndan selain itu dapat juga mengatur ukuran sendiri secara bebas.

-Kegunaan Ikon-ikon menu Page Layout di MS Word-5 (Column) untuk membuat halaman menjadi berkolom

Kegunaan Ikon-ikon menu Page Layout di MS Word-6 (Watermark) untuk memberikan tanda air (tulisan kabur) di belakang dokumen.

-Kegunaan Ikon-ikon menu Page Layout di MS Word-7 (Page Color) memberikan warna pada halaman.

-Kegunaan Ikon-ikon menu Page Layout di MS Word-8 (Page Border) ikon ini adalah untuk memberikan dan mengatur garis pinggir halaman atau bingkai. Sehingga jika ingin memberikan bingkai pada halaman dapat dilakukan dengan ikon ini.

-Kegunaan Ikon-ikon menu Page Layout di MS Word-9 (Ident Left) ikon ntuk mengatur batas kiri tulisan.

-Kegunaan Ikon-ikon menu Page Layout di MS Word-10 (Ident Right) ikon ini adalah untuk mengatur batas kanan tulisan.

-Kegunaan Ikon-ikon menu Page Layout di MS Word-11 (Spacing Before) untuk mengatur jarak paragraph dengan paragraph sebelumnya

-Kegunaan Ikon-ikon menu Page Layout di MS Word-12 (Spacing After) untuk mengatur jarak paragraph dengan paragraph sesudahnya.
