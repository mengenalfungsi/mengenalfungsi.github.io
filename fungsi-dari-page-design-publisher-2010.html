tab Home: berfungsi untuk menampung grup Clipboard, grup Font, grup Paragraph, grup Styles, grup Editing

tab Insert: berfungsi untuk menampung grup Pages, grup Tables, grup Illustrations, grup Apps, grup Media, grup Comments, grup Header & Footer, grup Text, grup Symbols

tab Design: berfungsi untuk menampung grup Document Formatting, grup Page Background

tab Page Layout: berfungsi untuk menampung grup Page Setup, grup Paragraph, grup Arrange

tab References: berfungsi untuk menampung grup Table of Contents, grup Footnotes, grup Citations & Bibliography, grup Captions, grup Index, grup Table of Authorities

tab Mailings: berfungsi untuk menampung grup Create, grup Start Mail Merge, grup Write & Insert Fields, grup Preview Result, grup Finish

tab Review: berfungsi untuk menampung grup Proofing, grup Language, grup Comments, grup Tracking, grup Changes, grup Compare, grup Protect

tab View: berfungsi untuk menampung grup Views, grup Show, grup Zoom, grup Window, grup Macros

tab Foxit PDF: berfungsi untuk menampung grup Create PDF, grup Create Redacted PDF, grup Comment, grup General Settings, grup Connected Info, grup Sign In

Penjelasan:

Sebenarnya tab di Ms. Word jumlahnya bervariasi.

Secara garis besar, setiap tab di Ms. Word berfungsi untuk menampung aneka grup.

Sedangkan grup di tiap tab berfungsi untuk menampung berbagai ikon-ikon perintah tertentu.

Tab yg paling umum yaitu: home, insert, design, page layout, references, mailings, review, view. Sedangkan tab Foxit PDF ialah contoh tab tambahan di Ms. Word.

Untuk lebih jelasnya, amatilah lampiran (tab Foxit PDF ditunjukkan tanda panah merah).

Microsoft Visual Basic .NET adalah sebuah alat untuk mengembangkan dan membangun aplikasi yang bergerak di atas sistem .NET Framework, dengan menggunakan bahasa BASIC. Dengan menggunakan alat ini, para programmer dapat membangun aplikasi Windows Forms, Aplikasi web berbasis ASP.NET, dan juga aplikasi command-line. Alat ini dapat diperoleh secara terpisah dari beberapa produk lainnya (seperti Microsoft Visual C++, Visual C#, atau Visual J#), atau juga dapat diperoleh secara terpadu dalam Microsoft Visual Studio .NET. Bahasa Visual Basic .NET sendiri menganut paradigma bahasa pemrograman berorientasi objek yang dapat dilihat sebagai evolusi dari Microsoft Visual Basic versi sebelumnya yang diimplementasikan di atas .NET Framework. Peluncurannya mengundang kontroversi, mengingat banyak sekali perubahan yang dilakukan oleh Microsoft, dan versi baru ini tidak kompatibel dengan versi terdahulu.

Apakah Visual Basic .NET dianggap sebagai sebuah versi Visual Basic atau benar-benar bahasa yang berbeda merupakan sebuah topik perdebatan yang hangat. Hal ini dikarenakan sintaksis bahasa Visual Basic .NET tidak mengalami perubahan yang sangat drastis, dan hanya menambahkan beberapa dukungan fitur baru seperti penanganan eksepsi secara terstruktur dan ekspresi yang bisa di-short-circuit-kan. Dua perubahan tipe data pun terjadi saat berpindah ke Visual Basic .NET. Dibandingkan dengan Visual Basic 6.0, tipe data Integer yang dimiliki oleh Visual Basic .NET memiliki panjang dua kali lebih panjang, dari 16 bit menjadi 32 bit. Selain itu, tipe data Long juga sama-sama berubah menjadi dua kali lipat lebih panjang, dari 32 bit menjadi 64 bit. Bilangan bulat 16-bit dalam Visual Basic .NET dinamakan dengan Short. Lagi pula, desainer GUI Windows Forms yang terdapat di dalam Visual Studio .NET atau Visual Basic .NET memiliki gaya yang sangat mirip dengan editor form Visual Basic klasik.

Jika sintaksis tidak banyak yang berubah, lain halnya dengan semantik, yang berubah secara signifikan. Visual Basic .NET merupakan sebuah bahasa pemrograman yang mendukung fitur "Bahasa Pemrograman Berorientasi Objek" secara penuh, karena memang didukung oleh arsitektur Microsoft .NET Framework, yang mengandung kombinasi dari Common Language Runtime dan Base Class Library. Visual Basic klasik, hanya merupakan sebuah bahasa pemrogaman berbasis objek, yang berjalan di atas arsitektur Component Object Model (COM).

Perubahan ini telah mengubah banyak asumsi tentang hal yang benar yang harus dilakukan dengan mempertimbangkan performa dan kemudahan untuk dipelihara. Beberapa fungsi dan pustaka perangkat lunak, yang ada di dalam Visual Basic klasik, kini tidak terdapat di dalam Visual Basic .NET; mungkin masih banyak yang masih terdapat di dalam Visual Basic .NET, tetapi tidak seefisien apa yang ditawarkan oleh .NET Framework. Bahkan jika program Visual Basic klasik bisa dikompilasi dengan benar, sebagian besar program Visual Basic klasik harus melalui beberapa proses refactoring untuk mengadopsi fitur bahasa baru secara keseluruhan. 35. Sebutkan & Jelaskan fitur-fitur Design & Page Layout​


Fitur-fitur yang terdapat pada tab Design di Microsoft Word, antara lain:

•Themes : digunakan untuk mengatur tema halaman agar menjadi lebih menarik.

•Document Formatting : untuk mengatur format pada dokumen.

•Color : digunakan untuk memberi warna pada format dokumen yang sudah dibuat.

•Font : mengatur font atau gaya tulisan pada halaman.

•Effect : untuk memberikan efek pada halaman

•Paragraph Spacing : mengatur spasi(jarak) antar paragraf sesuai yang kita inginkan.

•Watermark : menambahkan tulisan semar-semar di tengah-tengah halaman.

•Page Color : memberi warna pada halaman.

•Page Border : menambahkan bingkai pada pinggir halaman.

Fitur-fitur yang terdapat pada tab Page Layout di Microsoft Word, antara lain:

•Margin : fitur ini digunakan untuk mengatur batas-batas pada pinggir halaman. Batas-batasnya meliputi batas atas, batas bawah, batas kiri, dan batas kanan.

•Orientation : fitur ini mengatur orientasi bagaimana orientasi pada kertas, apakah kertas miring ataupun tegak hal tersebut yang dapat dilakukan dari fitur ini.

•Size : fitur untuk mengatur ukuran kertas. Ada jufa ukuran standar yang sudah disediakan dan juga dapat mengatur ukurannya sendiri dengan bebas.

•Column : membuat halaman menjadi memiliki kolom.

•Watermark : untuk memberikan tulisan kabur(tanda air) pada belakang dokumen.

•Page color : digunakan untuk memberikan warna pada halaman.

•Page border : untuk mengatur dan memberikan garis pada pinggir halaman atau juga dapat disebut bingkai.

•Ident Left : digunakan untuk mengatur batas pada kiri tulisan.

•Ident Right : digunakan untuk mengatur batas pada kanan tulisan.

•Spacing Before : untuk mengatur jarak antara paragraf dengan paragraf sebelumnya.

•Spacing After : untuk mengatur jarak antara paragraf dengan paragraf sesudahnya.
