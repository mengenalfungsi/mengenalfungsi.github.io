12. jelaskan fungsi block berikut!​


Block "repeat until" merupakan salah satu jenis perulangan dalam pemrograman. Block ini akan melakukan pengulangan terus-menerus pada satu atau beberapa pernyataan tertentu sampai suatu kondisi tertentu terpenuhi.

Fungsi dari block "repeat until" adalah untuk memungkinkan program melakukan pengulangan pada satu atau beberapa pernyataan sampai kondisi tertentu terpenuhi. Kondisi tersebut biasanya berupa suatu nilai boolean yang menentukan apakah pengulangan harus terus dilakukan atau tidak.

Dalam block "repeat until", pernyataan yang berada di dalam blok akan dieksekusi setidaknya satu kali, dan kemudian akan diperiksa kondisi boolean di bagian akhir blok. Jika kondisi tersebut masih bernilai false, maka pernyataan di dalam blok akan diulangi kembali, dan begitu seterusnya sampai kondisi bernilai true.

Contoh penggunaan block "repeat until" dalam pseudocode:

lua

Copy code

repeat until x > 10

print("x = ", x)

x = x + 1

end repeat

Pada contoh di atas, pernyataan di dalam blok "repeat until" akan terus dieksekusi sampai nilai x lebih besar dari 10. Pada setiap iterasi, nilai x akan ditampilkan pada layar dan nilai x akan terus bertambah satu sampai mencapai 10. Setelah x mencapai nilai 10, pengulangan akan berhenti.

32. Bagian jendela berikut yang merupakan tempat susunan blok kode pogram adalah…. * Stage pane Tab customes Block palette Script area


Jawaban:

D. Script area

Penjelasan:

Script area adalah area di jendela pemrograman visual di mana pengguna dapat menyusun blok-blok kode secara berurutan untuk membuat program atau skrip. Ini adalah tempat di mana logika program diatur dengan menggabungkan berbagai blok kode.

Di dalam Script area, pengguna dapat mengambil blok-blok kode dari Block palette dan menaruhnya secara berurutan untuk membentuk alur eksekusi program. Blok-blok kode dapat disusun, diatur, dan dihubungkan satu sama lain menggunakan ikon-ikon yang ada di blok tersebut. Dengan cara ini, pengguna dapat membentuk instruksi atau alur program dengan cara yang intuitif dan visual.

Setelah blok-blok kode ditempatkan di Script area, mereka dapat dieksekusi berurutan sesuai dengan alur yang telah dibangun. Ini memungkinkan pengguna untuk melihat dan mengontrol jalannya program dengan mudah.

Jadi, Script area adalah bagian jendela yang merupakan tempat susunan blok kode program untuk membuat program atau skrip dalam lingkungan pemrograman visual.
