2. Sebut dan jelaskan shortcut yang ada pada blender


Berikut ini beberapa shorcut yang ada pada aplikasi Blender, yaitu:

Ctrl + Alt + 0, berfungsi untuk mengaktifkan posisi kamera agar dapat berpindah sesuai dengan tampilan yang sedang aktif.Ctrl + 0, berfungsi untuk mengaktifkan kamera yang ingin digunakan.Shift + F + Scroll mouse, digunakan untuk memajukan atau memundurkan view dari kamera tersebut.G, berfungsi untuk mengeser atau memindahkan objek.S, berfungsi untuk melakukan scallingR, berfungsi untuk memutar objek atau mesh yang diseksi.Z, berfungsi untuk menggunakan mode wireframe.Alt + “Key”, berfungsi untuk mengembalikan ke proses semula.Alt+R, berfungsi untuk mengembalikan objek ke bentuk sebelum di rotasi.Alt+S, berfungsi untuk mengembalikan objek ke bentuk sebelum di scalling.Shift+h, berfungsi untuk menghidden objek yang tidak diseleksi.Shift + A, berfungsi untuk menambahkan objek atau mesh baru.Shift + D, berfungsi untuk menggandakan objek atau mesh yang diseleksi.E, berfungsi untuk memanjangkan mesh yang diseleksi.

Pembahasan

Blender adalah salah satu aplikasi atau sofware yang digunakan untuk membuat karya tiga dimensi (3D). Aplikasi Blender bersifat gratis dan  open source, dan dapat digunakan di berbagai sistem operasi, seperti Windows, macOS, serta Linux.

Aplikasi Blender memiliki berbagai macam fitur, yaitu:

3D modelling.Rigging.Fluid dynamics.Keyframe animation.Photorealistic rendering.Video editing.

Ada beberapa kelebihan dari aplikasi Blender, yaitu:

Berbasis Open Source.Mendukung sistem multi-platform.Memiliki fitur yang lengkap.Aplikasi terus dikembangkan.Dan tentunya gratis.

7. tulisan shortcut keyboard untuk software blender dalam bahasa Indonesia secara lengkap yah Terimakasih ​


Blender adalah software yang berfungsi untuk membuat model tiga dimensi. Shortcut keyboard untuk software Blender, yaitu:

Tombol TAB, untuk mengubah antara “edit mode” dan “object select mode”.Tombol “O”, untuk mengaktifkan “proportional vertex editing” pada “edit mode”.Tombol “A”, untuk memilih semua vertices yang terdapat pada sebuah obyek.Tombol “B”, untuk memilih banyak objek dengan menggunakan windows drag.Tombol Space atau spasi, untuk menampilkan menu tool.Tombol angka, untuk mengontrol view, yaitu tombol “7” untuk “top”, tombol “1” untuk “front”, tombol “3” untuk “side”, tombol “0” untuk “camera”, tombol “5” untuk “perspective”, tombol “.”, untuk memperbesar objek yang telah dipilih, tombol “+” dan “-“ untuk memperbesar atau memperkecil.Tombol Shift, untuk memilih objek lebih dari satu.Tombol Arrow atau panah, untuk pembuatan frame dalam sebuah animasi.Tombol “R”,untuk merotasi sebuah objek.Tombol “S”, untuk mengatur skala sebuah objek.Tombol “G”, untuk memindahkan sebuah objek.Tombol “P”, untuk memisahkan objek yang dipilih menjadi sebuah objek dalam “edit mode”.Tombol Shift + “D”, untuk menduplikat sebuah objek.Tombol “E”, untuk meng-extrude objek yang telah dipilih di dalam “edit mode”.Tombol “U”, untuk menampilkan Single User menu, sedangkan di dalam “edit mode” berfungsi sebagai “undo”.Tombol “M”, untuk memindahkan objek ke layer lain, dan di dalam “edit mode” berfungsi sebagai “Mirror”.Tombol “Z”, untuk merubah view dari wireframe menjadi solid.Tombol Alt + “Z”, untuk merubah view tekstur.Tombol “P”, untuk memulai mode game.Tombol ALT / CTRL + “P”, untuk membuat atau menghapus hubungan Parent/Child.Tombol “N”, untuk menampilkan info sebuah obyek dalam bentuk numeric atau angka.Tombol Ctrl + “J”,  untuk menggabungkan objek.Tombol Alt + “A”, untuk menjalankan animasi.Tombol “F”, untuk membuat sebuah sisi pada “edit mode”.Tombol “W”, untuk menggabungkan atau mengiris pada 2 objek atau lebih.Tombol “X” atau tombol delete untuk menghapus objek, vertices, atau sisi.Tombol Function yaitu F1 untuk Load File, F2 untuk Save, F3 untuk Save Image, F4 untuk Lamp Buttons, F5 untuk Material Buttons, F6 untuk Texture Buttons, F7 untuk Animation Buttons, F8 untuk Real Time Buttons, F9 untuk Edit Buttons, F10 untuk Display Buttons, F11 untuk Last Render, F12 untuk Render.Tombol “I”, untuk memasukkan animasi.Tombol ALT + “U”, untuk menampilkan “Global Undo Command”.Tombol ALT + “C”, untuk mengkonversi menjadi meshes, text dan curves.Tombol Shift + Space, untuk menampilkan seluruh layar atau hanya pada viewport yang aktif saja.Tombol Ctrl + “0”, untuk memilih sebuah camera.

Pembahasan

Blender merupakan aplikasi atau software berfungsi untuk membuat karya tiga dimensi (3D) atau model tiga dimensi. Aplikasi Blender bersifat gratis dan  open source, dan dapat digunakan di berbagai sistem operasi, seperti Windows, macOS, serta Linux.

Aplikasi Blender memiliki berbagai macam fitur, yaitu untuk 3D modelling, rigging, fluid dynamics, keyframe animation, photorealistic rendering dan video editing.

13. Apa fungsi shortcut koneksi pada internet?


Ctrl + A = Fungsinya untuk memblok seluruh dokumen (All)

Ctrl + B = Fungsinya memberika efek tebal pada huruf (Bold)

Ctrl + C = Fungsinya untuk mengcopy document (Copy)

Ctrl + D = Fungsinya untuk menggandakan entah itu data atau gambar (Double)

Ctrl + E = Fungsinya untuk mengetengahkan document

Ctrl + F = Fungsinya untuk mencari satu kata atau kalimat (Find)

Ctrl + G = Fungsinya untuk tabulasi pada Open Office

Ctrl + H = Fungsinya untuk mengaktifkan kotak History pada browser (Mozilla, Opera, dll)

Ctrl + I = Fungsinya untuk memiringkan tulisan (Italic)

Ctrl + J = Fungsinya untuk mengaktifkan kotak download pada browser (Mozilla, Opera, dll)

Ctrl + K = Fungsinya untuk mencari cepat dengan mozilla pada kotak search bar

Ctrl + L = Fungsinya untuk memasukkan alamat web pada browser Mozilla, Opera, dll (Location)

Ctrl + M = Fungsinya untuk mengaktifkan email client pada Opera

Ctrl + N = Fungsinya untuk membuka lembar baru (New)

Ctrl + O = Fungsinya untuk membuka document (Open)

Ctrl + P = Fungsinya untuk mencetak halaman (Print)

Ctrl + Q = Fungsinya untuk keluar secara cepat untuk Open Office (Quit)

Ctrl + R = Fungsinya untuk meratakan document sebelah kanan (Right)

Ctrl + S = Fungsinya untuk menyimpan document (Save)

Ctrl + T = Fungsinya untuk membuka tab baru pada browser Mozilla, Opera, dll (Tab)

Ctrl + U = Fungsinya untuk memberikan garis bawah pada kata (Open Office, Word, dll) (Underline)

Ctrl + V = Fungsinya untuk menempelkan kata yang sudah di copy (Paste)

Ctrl + W = Fungsinya untuk mengclosekan tab yang ada dibrowser

Ctrl + X = Fungsinya untuk memindahkan kata (Cut)

Ctrl + Y = Fungsinya untuk mengembalikan apa yang sudah di Undo (Redo)

Ctrl + Z = Fungsinya untuk mengembalikan ke keadaan seperti sebelumnya. (Undo)

38. apa fungsi dari shortcut ctrl+[ dan ctrl+]


Menurut saya sih tidak ada fungsi nya... Karena shortcut "CTRL+" hanya bisa digunakan di CTRL+( A - Z ) & CTRL+( 1 - 9 ).Pada software Microsoft Office (Word, Excel, Powerpoint, dsb), shortcut Ctrl+[ berfungsi untuk untuk memperbesar ukuran huruf sebanyak 1 point/ukuran (misal sebuah text awalnya memiliki ukuran 11, jika diblok lalu kita tekan Ctrl+[, maka ukuran text tersebut akan bertambah jadi 12), sedangkan Ctrl+] berfungsi sebaliknya/untuk memperkecil ukuran huruf sebanyak 1 point/ukuran (misal awalnya berukuran 11, setelah kita tekan Ctrl+], maka ukurannya menjadi 10)
