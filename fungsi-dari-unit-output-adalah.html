A. Cara Kerja Perangkat Keras Komputer (Hardware)

Hardware secara fundamental mempunyai bagian non-fisik yang disebut firmware. Firmware berfungsi sebagai penyedia instruksi dasar yang dapat dilakukan sebuah hardware. Firmware disimpan dalam chip memori khusus pada suatu hardware yang dapat berupa ROM atau EEPROM. Firmware berupa sebuah perangkat lunak kecil (software) yang biasanya ditulis dalam bahasa pemrograman rendah untuk menghasilkan instruksi dasar yang dapat digunakan oleh hardware bersangkutan secara cepat. Instruksi dasar ini dapat dibedakan menjadi 3 yaitu,

Instruksi Dasar Input

Instruksi dasar input adalah instruksi yang dapat memberikan informasi kepada perangkat lunak lainnya yaitu sistem operasi dengan perantara driver. Hal ini terjadi saat pengguna berinteraksi dengan hardware bersangkutan. Misalnya mengetik dengan keyboard, menggeser mouse, dan lain-lain.

Instruksi Dasar Proses

Instruksi dasar proses adalah instruksi yang berperan sebagai penengah proses input atau output yang dilakukan suatu hardware (tidak keduanya). Instruksi dasar proses yang dimaksud adalah instruksi yang hanya terjadi dalam suatu hardware atau bukan dalam sistem komputer secara utuh.

Instruksi Dasar Output

Instruksi dasar output adalah instruksi yang dipanggil oleh perangkat lunak lainnya yaitu sistem operasi dengan perantara driver. Hal ini terjadi saat ada perintah untuk menggunakan hardware tersebut dari pengguna. Misalnya memutar video dengan monitor, mencetak dokumen dengan printer, dan lain-lain

B. Jenis-Jenis Perangkat Keras Komputer dan Fungsinya

Sistem komputer terdiri dari 5 jenis perangkat keras yaitu perangkat keras input, perangkat keras proses, perangkat keras output, perangkat keras penyimpanan, dan perangkat keras kelistrikan. Kelima perangkat keras ini diatur sedemikian rupa membentuk sistem terintegrasi untuk dapat melakukan input, proses, dan output data.

Berikut masing-masing penjelasan dari ketiga perangkat keras tersebut.

Perangkat Keras Input

Perangkat keras input adalah jenis hardware yang berfungsi untuk memasukkan data (input) ke dalam memory sistem komputer. Perangkat keras input utama dari sebuah sistem komputer adalah keyboard. Mouse merupakan perangkat keras input pendukung (peripheral) dari sebuah sistem komputer. Berikut beberapa contoh hardware input pada komputer.

21. Suatu perusahaan memproduksi suatu barang dengan input x. Output yang dihasilkan pada berbagai tingkat penggunaan ditunjukkan dengan fungsi Q = 75 + 5x 2 –1/3 x3. Jika harga input adalah Rp 2.100,-/unit dan harga output per unit Rp 100, berapa unit yang harus diproduksi oleh perusahaan agar keuntungan yang diperolehmaksimum? Berapakah produksi rata-rata?​


Jawab:

Untuk mencari jumlah unit yang harus diproduksi agar keuntungan maksimum, kita perlu memaksimalkan fungsi keuntungan perusahaan. Fungsi keuntungan perusahaan (π) dapat dihitung dengan mengurangi total biaya produksi dari total pendapatan:

π(x) = R(x) - C(x)

Rumus pendapatan (R):

R(x) = Harga output per unit (p) * Jumlah unit (Q)

R(x) = 100Q

Rumus biaya produksi (C):

C(x) = Harga input per unit (c) * Jumlah unit (x)

C(x) = 2100x

Maka, fungsi keuntungan (π):

π(x) = 100Q - 2100x

Dalam hal ini, fungsi output (Q) telah diberikan: Q = 75 + 5x^2 - (1/3)x^3.

Substitusikan nilai Q ke dalam fungsi keuntungan (π):

π(x) = 100(75 + 5x^2 - (1/3)x^3) - 2100x

Selanjutnya, kita perlu mencari nilai x yang memaksimumkan fungsi keuntungan dengan menghitung turunan pertama dan menyelesaikan persamaan turunan tersebut sama dengan nol:

π'(x) = 500x - 100x^2 - 700

π'(x) = 0

500x - 100x^2 - 700 = 0

-100x^2 + 500x - 700 = 0

Selanjutnya, kita dapat menyelesaikan persamaan kuadrat ini untuk mencari nilai-nilai x yang memenuhi.

Dengan menggunakan metode faktorisasi atau rumus kuadrat, kita dapatkan nilai-nilai x = -1 dan x = 7.

Namun, kita perlu memeriksa apakah nilai-nilai ini benar-benar memberikan maksimum atau minimum dengan menggunakan turunan kedua:

π''(x) = 500 - 200x

Untuk nilai x = -1, π''(-1) = 500 + 200 = 700 > 0, yang menunjukkan bahwa ini adalah minimum.

Untuk nilai x = 7, π''(7) = 500 - 1400 = -900 < 0, yang menunjukkan bahwa ini adalah maksimum.

Jadi, jumlah unit yang harus diproduksi oleh perusahaan agar keuntungan maksimum adalah 7 unit.

Produksi rata-rata = Jumlah unit / Harga input per unit

23. Jika fungsi produksi Q = 100K0,5L0,5, w = 30 dan r = 40, berapa biaya minimum yang harus dikeluarkan untuk memproduksi 1444 unit output tersebut


Untuk mengetahui biaya minimum yang diperlukan untuk memproduksi 1444 unit output, kita dapat menggunakan konsep produksi dengan biaya minimum. Biaya minimum terjadi saat produktivitas faktor input terakhir (MPP) dibagi dengan harganya (VMPL) sama di antara semua faktor input. Dalam hal ini, faktor inputnya adalah modal (K) dan tenaga kerja (L), sedangkan harga modal adalah r dan harga tenaga kerja adalah w.

Langkah-langkah yang dapat dilakukan adalah sebagai berikut:

Tentukan persamaan isoquant: Q = 100K^0,5L^0,5 = 1444

Dari persamaan ini, kita dapat mencari kombinasi input yang diperlukan untuk memproduksi 1444 unit output.

Dapatkan MPP K dan MPP L:

MPP K = 0,5Q/K = 50K^(-0,5)L^0,5

MPP L = 0,5Q/L = 50K^0,5L^(-0,5)

Hitung VMPL:

VMPL = turunan parsial Q terhadap L / w = 0,5Q/Lw = 25K^0,5L^(-0,5)/w

Hitung VMpk:

VMpk = turunan parsial Q terhadap K / r = 0,5Q/Kr = 25K^(-0,5)L^0,5/r

Atur VMPL sama dengan VMpk:

VMPL = VMpk

25K^0,5L^(-0,5)/w = 25K^(-0,5)L^0,5/r

K/L = (r/w)^2

Tentukan K dan L:

K/L = (r/w)^2

K/L = (40/30)^2

K/L = 16/9

K = (16/9)L

Substitusikan K ke dalam persamaan isoquant untuk mendapatkan nilai L:

100K^0,5L^0,5 = 1444

100(16/9)^0,5L^0,5 = 1444

L^0,5 = 1444/(100(16/9)^0,5)

L^0,5 = 18

L = 324

Hitung biaya minimum:

Biaya minimum = rK + wL

Biaya minimum = 40(16/9)L + 30L

Biaya minimum = 960 + 972

Biaya minimum = 1932

Jadi, biaya minimum yang harus dikeluarkan untuk memproduksi 1444 unit output adalah 1932.

34. Fungsi biaya sebuah perusahaan C(Q) = 1000 + 2Q, dan fungsi permintaan untuk output perusahaan: P= f(Q) = 500 - Q. Dimana P= harga per unit, Q= output dan C = total biaya. Tentukanlah Fungsi profit?​


Jawaban:

Fungsi profit dapat dituliskan sebagai selisih antara pendapatan total (total revenue) dan biaya total (total cost). Pendapatan total dapat dihitung dengan mengalikan harga per unit dengan jumlah output, sedangkan biaya total dapat dihitung dengan menggunakan fungsi biaya yang telah diberikan.

Fungsi profit dapat dituliskan sebagai berikut:

Profit = Pendapatan total - Biaya total

= P * Q - C(Q)

= (500 - Q) * Q - (1000 + 2Q)

= 500Q - Q^2 - 1000 - 2Q

= -Q^2 + 500Q - 1000

Jadi, fungsi profit dari perusahaan tersebut adalah -Q^2 + 500Q - 1000.
