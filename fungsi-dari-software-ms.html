8. mengidentifikasi software ms word 2007


Hai kak!

Dalam mengidentifikasi "software" ada beberapa klasifikasi yang perlu diperhatikan yaitu :
1. Tipe Aplikasi : tentu banyak sekali aplikasi yang telah dikembangkan pada berbagai platform namun perlu dipertimbangkan dalam ketepatan identifikasi pada perangkat lunak tertentu.

2. Keterbukan Kode : Ada 2 Jenis "keterbukaan kode" dalam produk perangkat lunak manapun yaitu, closed source dan open source.

Identifikasi "Microsoft Word 2007"
1. Tipe Aplikasi
- Suite application, artinya perangkat lunak tersebut mengandung berbagai bundle perangkat lunak lain.Dalam kasus ini Microsoft Word 2007 termasuk dalam suite Microsoft Office 2007.
- Information worker, artinya perangkat lunak tersebut mengizinkan pengguna untuk melakukan pembuatan dan manajemen informasi bagi diri sendiri maupun organisasi.Proses dokumentasi (word documentation) dan analisis data terkandung dalam perangkat lunak tersebut.

2. Kerterbukaan Aplikasi : Closed source, artinya kode program perangkat lunak ini tidak dapat dilihat maupun diizinkan untuk disunting,diubah,ditambah, maupun dikurangi dalam konsumsi pribadi maupun komersial.

Jawaban:

 

1.    Sebutkan minimal 4 yang termasuk dalam kategori software operating system !

 

Microsoft Windows, Linux Ubuntu dan Linux Debian, serta Apple iOS

 

2.    Apa yang kamu ketahui tentang software Ms. Word? Dan berilah 5 contoh/manfaat dari software tersebut !

 

Microsoft Word adalah perangkat lunak atau software pengolah kata (word processor).

 

Manfaat Microsoft Word adalah:

 

a. Dapat membuat konsep dokumen seperti

b. Dapat mengedit dokumen

c. Dapat menyimpan dokumen menjadi format seperti PDF

d. Dapat mencetak dokumen

e. Dapat mengirimkan dokumen ke pengguna lain untuk berkolaborasi

 

3.    Tulislah langkah-langkah mengaktifkan perangkat lunak pengolah kata(word processor)!

 

Pada sistem operasi Windows, dapat diaktifkan dengan menekan tombol Start, lalu memilih menu program Microsoft Word

 

4.    Tulis dan jelaskan instruksi-instruksi yang ada pada program Ms. Word minimal 5!

 

Print: Untuk mencetak dokumen

Save: Untuk menyimpan dokumen

Save As: Untuk menyimpan dokumen ke file yang berbeda

Find: Mencari kata atau kalimat di dokumen

Page Layout: Mengatur ukuran dan setting halaman dokumen

 

5.    Sebutkan langkah-langkah menyimpan dokumen !

 

Pada Microsoft Word, menyimpan dapat dilakukan dengan menekan shortcut Ctrl+S, atau memilih menu File->Save untuk menyimpan pada file yang sama, atau memilih menu File->Save as, untukmenyimpan pada file yang berbeda.

 

6.    Apa yang dimaksud dengan perangkat lunak utility?

 

Perangkat lunak utility adalah perangkat lunak untuk membantu proses analisis, konfigurasi, optimasi, dan membantu pengelolaan sebuah komputer ataupun sistem

 

7.    Sebutkan dan dan jelaskan minimal 7 tombol apa saja yang ada di keyboard !

 

Tombol angka (0-9), tombol huruf (A-Z), tombol fungsi (F1-F12) tombol Print Screen, tombol Home, tombol nd, tombol Delete, tombol Caps Lock, tombol Enter, serta tombol Control dan Alt.

 

8.    Apa yang dimaksud dengan perangkat lunak aplikasi? Berilah contohnya !

 

Perangkat lunak aplikasi  adalah perangkat lunak komputer yang memanfaatkan kemampuan komputer langsung untuk melakukan suatu tugas yang diinginkan pengguna

 

9.    Apa yang dimaksud dengan software? Berilah contohnya !

 

Sofrtware adalah data yang diformat, dan disimpan secara digital, termasuk program komputer, dokumentasinya, dan berbagai informasi yang bisa dibaca, dan ditulis oleh komputer. Dengan kata lain, bagian sistem komputer yang tidak berwujud.

 

10. Apa yang termasuk software Ms. Office? Jelaskan fungsinya software tersebut !

 

Yang termasuk Microsoft Office adalah:

 

a. Microsoft Word: untuk mengolah kata, membuat, mengedit dan menyimpan serta mencetak dokumen seperti surat dan laporan

b. Microsoft Excel: pengolah data untuk perhitungan dan membuat spreadsheet

c. Microsoft PowerPoint: untuk membuat presentasi dan slide

d. Microsoft Access: untuk menyimpan dan mengelola basis data

e. Microsoft OneNote: untuk mencatat dan mengelola catatan singkat

f. Microsoft Publisher: untuk membuat dokumen seperti brosur, majalah dan buku dengan layout yang lebih kompleks

19. 7 Software dapat dibagi berdasarkan fungsinya, salah satunya adalah system software tuliskan fungsi dari sistem software​


Jawaban:

Fungsi Software

Software memiliki fungsi dalam mengatur hardware/perangkat keras yang ada pada komputer. Dengan begitu, komputer yang digunakan dapat bekerja dengan baik sesuai pekerjaan apa yang dibutuhkan. 2. Software juga dapat digunakan sebagai penghubung antara beberapa software yang lain dengan hardware komputer

Penjelasan:

Terserah deh :)

Fungsi dari system software adalah sebagai “jembatan“ atau interface antara perangkat keras komputer dengan aplikasi, yang mengendalikan dan mengelola pengoperasian perangkat keras komputer sesuai instruksi dari aplikasi yang digunakan.
 

Pembahasan

System software adalah software yang “menjembatani” atau sebagai penghubung/interface antara perangkat keras komputer dengan aplikasi. System software mengendalikan dan mengelola pengoperasian perangkat keras komputer, sesuai dengan instruksi dari perangkat lunak aplikasi (user application). Dengan demikian, dapat dikatakan pula bahwa system software berfungsi untuk menjamin program-program aplikasi berjalan dengan benar/semestinya.

Contoh dari system software adalah sistem operasi (operating system), seperti Microsoft Windows, OS X, Linux, Unix, dsb.
