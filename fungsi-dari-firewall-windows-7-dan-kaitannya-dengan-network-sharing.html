20. Apakah yang harus diloakukan apabila sudah memverifikasi windows firewall tetapi tetap tidak berjalan?


Jawaban:

Panggil fungsi "Setup API InstallHinfSection" untuk menginstal Windows Firewall

Untuk menginstal Windows Firewall, ikuti langkah-langkah:

Klik Start, klik Run, ketik cmd, and klik OK.

Pada command prompt, ketik baris perintah berikut, kemudian tekan ENTER:

Rundll32 setupapi,InstallHinfSection Ndi-Steelhead 132 %windir%\inf\netrass.inf

Restart Windows,

Klik Start, klik Run, ketik cmd, dan klik OK.

Pada command prompt, ketik baris perintah berikut, kemudian tekan ENTER:

Netsh firewall reset

Klik Start, klik Run, ketik firewall.cpl, dan tekan ENTER. Pada kotak dialog Windows Firewall, klik On (recommended), kemudian klik OK.

25. Bagai mana cara sharing data pada windows 7


1. Masuk ke Control Panel – System and Security – System.

2. Klik link Change Settings pada bagian kanan di bawah.

3. Klik tombol change hingga menampilkan computer name / domain changes.

4. Edit nama workgroup yang anda inginkan, di contoh saya memakai nama workgroup sebagai default.

5. Masuk ke Control Panel – Network and Internet – Network and Sharing Center.

6. Klik menu Change Adapter Settings.

7. Klik Local Area Connection – klik kanan pada mouse lalu properties.

8. Double klik pada Internet Protocol Version 4 (TCP/IPv4).

9. Masukkan IP address : 192.168.1.3 dan subnet mask : 255.255.255.0

10. Klik ok pada semua windows yang terbuka

11. Masuk ke Control Panel – Network and Internet – Network and Sharing Center – Advanced Sharing Settings.

12. Masuk ke pilihan public (current profile) – pada Network Discovery pilih Turn ON Network Discovery – pada Public Folder Sharing pilih Turn On – pada Password Protect Sharing pilih Turn Off – silahkan Save changes.

13. Pilih folder yang mau di share – klik kanan pada mouse lalu properties – pilih tab sharing.

14. Klik tombol share – pilih level everyone pada panah – klik tombol add – klik tombol share – klik tombol done.

36. Perbedaan layanan samba dengan layanan sharing data dalam windows


Samba adalah layanan jaringan yang memungkinkan pengguna Linux dan Windows untuk berbagi file dan printer. Layanan ini memungkinkan pengguna Linux dan Windows untuk berbagi file dan printer melalui jaringan lokal. Layanan ini juga memungkinkan pengguna Linux untuk mengakses file dan printer yang berada di jaringan Windows.

Sedangkan layanan sharing data dalam Windows adalah layanan yang memungkinkan pengguna Windows untuk berbagi file dan printer di jaringan lokal. Layanan ini hanya memungkinkan pengguna Windows untuk berbagi file dan printer di jaringan lokal. Pengguna Linux tidak dapat mengakses file dan printer yang berada di jaringan Windows menggunakan layanan ini.


37. 1. Apa saja yang dapat dibagikan melalui jaringan komputer lokal? 2. Bagaimana melindungi file yang di sharing agar hanya dapat di akses oleh pengguna tertentu saja? 3. Jelaskan fungsi "Network Discovey", "File and Printer Sharing" pada pengaturan Network Profile! 4.Apa dampak buruk dan dampak positif dari berbagi file pada jaringan komputer?


1. yang dapat dibagikan via LAN antara lain, koneksi internet atau network sharing, data (folder sharing), printer sharing, dll
2. Caranya adalah mengatur pada setting network sharing dengan cara memberikan password atau menentukan apa saya yg bisa disharingkan dengan computer lain yg terhubung pada network yg sama
3. fungsi network discovery adalah menemukan jaringan computer atau internet dimana computer kita terhubung. fungsi file and printer sharing adalah mengatur dan menghubungkan computer atau printer ke beberapa computer dalam satu network yg sama sehingga printer atau file bisa diakses oleh banyak orang.
4. Dampak positive adalah bahwa setiap computer bisa mendapatkan akses yg sama pada file atau printer tertentu sehingga lebih efisien dan tidak perlu berpindah komputer untuk melakukan akses
Dampak buruknya adalah setiap orang mempunyai akses yg sama sehingga memungkinkan pengguna lain melakukan perubahan data yg tidak kita inginkan selain itu sangat mungkin terjadi infeksi virus komputer dari computer satu ke komputer yg lain.
