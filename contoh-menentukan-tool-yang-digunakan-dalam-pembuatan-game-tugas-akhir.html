9. teks artistik dapat dibuat dengan menggunakan..a.fill toolb.intetactive toolc.eyedropper toold.outline toole.text tool


Text tool adalah menu yang digunakan pada software Photoshop atau Corel Draw untuk membuat teks artistik. Penggunaan text memang termasuk salah satu komponen yang penting dalam pembuatan suatu desain grafis. Dengan adanya tulisn kita bisa memberikan nilai tambah atau memperkuat pesan yang ada.

Pembahasan

Untuk memulai atau menambahkan teks artistik pada lembar kerja Corel Draw kita, kita bisa langsung mengaktifkan text tool dengan cara mengkliknya kemudian klik pada lembar kerja kita.

Pada text tool, kita bisa melakukan berbagai macam perubahan pada text kita seperti

Memberikan warna pada tulisan kitaMemberikan efek bayanganMerubah jenis huruf yang dipakai

Dengan adanya text tool ini, kita semakin mudah dalam membuat tulisan pada desain kita.

Pelajari lebih lanjut

Demikian pembahasan mengani pembuatan teks artistik pada aplikasi Corel Draw, pada aplikasi Corel Draw menu yang digunakan adalah menu Text Tool. Untuk mempelajari lebih lanjut mengenai aplikasi Corel Draw atau materi lainnya mengenai desain grafis dapat dibaca pada link berikut

39. Buatkan Proses Cara Membuat Gambar Balon Corel Menggunakan Rectangle Tool dan Ellipse Tool? ​


1. Membuat Kotak atau Persegi Panjang

Rectange Tool pada Toolbox CorelDraw

Kotak atau adalah persegi yang memiliki empat sisi dengan panjang yan sama dan sudut 90 derajat. Sedangkan persegi panjang adalah persegi yang memiliki panjang dan lebar tidak sama dengan sudut 90 derajat. Untuk membuat persegi ataupun persegi panjang ilahkan klik Icon Rectangel Tool di Toolbar sebelah kiri atau tekan tombol F6 di keyboard kemudian langkahnya sebagai berikut:

Persegi: Setelah klik Rectangle Tool tekan tombol Ctrl (jangan dilepas sampai selesai) kemudian klik di titik awal lembar kerja dan tahan sampai titik akhir kemudian lepas.(Lihat gambar)

Persegi Panjang: Klik Rectangle tool kemudian klik di titik awal lembar kerja dan tahan sampai titik akhir kemudian lepas. (Lihat gambar)

Kotak dan Persegi Panjang

Gambar 2: Membuat persegi atau persegi panjang dengan CorelDraw X7

Yang membedakan proses keduanya adalah penggunaan tombol Ctrl di keyboard.

Untuk membuat persegi sesuiai dengan ukuran halaman kerja/kertas Double Klik pada icon Rectangle Tool.

2. Membuat Lingkaran atau Elip

Ellipse tool pada Toolbox

Untuk membuat lingkaran atau elip tidak jauh berbeda dengan cara membuat kotak/persegi panjang hanya saja tools yang digunakan berbeda yaitu Ellipse tool atau F7 di keyboard. Berikut ini langkahnya:

Lingkaran: Setelah klik Ellipse Tool tekan tombol Ctrl (jangan dilepas sampai selesai) kemudian klik di titik awal lembar kerja dan tahan sampai titik akhir kemudian lepas.(Lihat gambar)

Elip: Klik Ellipse Tool kemudian klik di titik awal lembar kerja dan tahan sampai titik akhir kemudian lepas. (Lihat gambar)

Lingkaran dan Elip

Gambar 3: Membuat Objek Lingkaran atau Elip

Toolbox Polygon

- persegi banyak / polygon, untuk membuat segi tiga ataupun segi banyak dengan mengatur jumlah sudut yang diinginkan
