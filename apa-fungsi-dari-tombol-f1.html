1. fungsi tombol f1 sampai tombol f 12


Fungsi F1

Biasanya tombol ini lebih difungsikan untuk menampilkan layar bantuan (help) dari hampir semua jenis aplikasi komputer.
Untuk mengakses beberapa jenis BIOS.
Pada Microsoft Office kombinasi tombol Windows Key + F1 akan menampilkan Microsoft Windows help and support center.
Kombinasi Ctrl + F1 akan menampilkan Task Pane pada Microsoft Office.
Mengaktifkan dan menonaktifkan fungsi mouse komputer.

Fungsi F2

Pada Windows biasa digunakan untuk mengubah/rename ikon ataupun file.
Kombinasi tombol Alt + Ctrl + F2 digunakan untuk membuka sebuah file dokumen pada Microsoft Word.
Kombinasi Ctrl + F2 akan menampilkan Print Preview pada Microsoft Word.
Biasa digunakan untuk mengakses CMOS Setup.

Fungsi F3

Pada MS–DOS berguna untuk mengulangi perintah terakhir.
Kombinasi tombol Shift + F3 akan mengubah teks dalam Microsoft Word dari huruf kecil menjadi huruf besar.
Mengaktifkan maupun menonaktifkan fungsi Wi-Fi

Fungsi F4

Untuk mengulangi aksi/perintah terakhir pada komputer (Undo).
Kombinasi tombol Alt + F4 akan menutup program yang sedang aktif pada Microsoft Windows secara bersamaan.
Kombinasi Ctrl + F4 akan menutup jendela yang sedang terbuka dalam jendela yang sedang aktif dalam Microsoft Windows.
Mengaktifkan maupun menonaktifkan suara/volume.

Fungsi F5

Biasa digunakan untuk refresh sistem operasi atau me-reload halaman pada web browser.
Untuk menjalankan perintah menu find, replace, dan go to window pada Microsoft Word.
Memulai slideshow pada PowerPoint.
Menurunkan pengaturan pencahayaan pada komputer.

Fungsi F6

Untuk memindahkan cursor mouse ke Address bar pada Internet Explorer dan Mozilla Firefox.
Ctrl + Shift + F6 digunakan untuk membuka dokumen Microsoft Word yang lain.
Menaikan pengaturan pencahayaan pada komputer

Fungsi F7

Biasa digunakan untuk menjalankan fitur pemeriksaan spell check dan grammar pada sebuah dokumen Microsoft Word.
Shift + F7 digunakan untuk menjalankan fitur Thesaurus pada Word.
Mengaktifkan layar monitor kedalam perangkat komputer seperti proyektor.

Fungsi F8

Tombol F8 ini digunakan untuk menampilkan Windows startup menu yaitu untuk mengakses Windows Safe Mode.
Menonaktifkan layar monitor kedalam perangkat komputer seperti proyektor.

Fungsi F9

Untuk membuka toolbar Measurements pada Quark 5.0.
Menampilkan informasi sistem operasi.

Fungsi F10

Digunakan untuk mengakses recovery partisi pada perangkat komputer merk HP dan Sony.
Untuk Masuk ke menu CMOS Setup.
Membuka/mengecek email masuk.

Fungsi F11

Untuk membuat layar menjadi full-screen pada semua web browser Internet.
Ctrl + F11 biasa digunakan untuk mengakses recovery partisi pada Komputer merk Dell.
Berfungsi untuk mengakses recovery partisi pada peraangkat komputer merk eMachines,
Gateway, dan Lenovo.

Fungsi F12

Melakukan Save as dokumen pada Microsoft Word.
Kombinasi tombol Shift + F12 berguna untuk menyimpan file dokumen pada Microsoft Word.
Kombinasi tombol Ctrl + Shift + F12 untuk mencetak dokumen pada Microsoft Word.
Untuk menjalankan Firebug.

27. Fungsi tombol F1 sampai F12 pada keyboard komputer ?​


• F1 berfungsi Help Screen (tombol F1 digunakan untuk menampilkan fitur Help pada hampir setiap program)

• F2 berfungsi Rename (mengganti nama file secara cepat)

• F3 berfungsi Search (Untuk mengaktifkan fitur Search pada aplikasi yang sedang berjalan/aktif )

• F4 berfungsi Close Window (Untuk menutup Window )

• F5 berfungsi Refresh/Reload (Memperbarui Window/Browser )

• F6 berfungsi Address Bar (Cara pintas ke Address bar)

• F7 berfungsi Spell and grammar (check Untuk mengaktifkan fungsi spell dan grammar di Microsoft Word)

• F8 berfungsi Boot (Mengakses boot/bios Mengakses boot/bios menu)

• F9 berfungsi Refresh (Refresh dokumen di Microsoft word)

• F10 berfungsi Menu bar (Mengakses menu bar)

• F11 berfungsi Fullscreen (Masuk atau keluar ke mode full screen)

• F12 berfungsi Save as (Mengaktifkan fungsi Save As di Microsoft Word)


28. tombol pada keyboard yang terdiri dari f1 sampai dgn f12 disebut


Tombol pada keyboard yang terdiri dari f1 sampai f12 disebut tombol Function Keys. Selain tombol function keys, terdapat beberapa kelompok tombol lain pada keyboard. Pada keyboard terdapat tombol Escape Keys. Tombol typing keyboard. Tombol cursor keys, dan numeric keypad sertaTombol kontrol (Control keys).

Pebahasan

Keyboard adalah salah satu hardware komputer yang berfung sebagai alat input. Melalui keyboard kita dapat menginput data berupa angka, huruf, dan simbol ke komputer. Keyboard juag menjadi jalan pintas saat bekerja, jika tidak menggunakan mouse.

Seperti yang disebutkan di atas, keyboar terdiri dari beberapa kelompok tombol. Berikut ini kita bahas fungsi dari Function Keys dari f1 sampai f12.

F1 digunakan untuk tombol helpF2 digunakan untuk rename, change volumeF3 digunakan untuk SearchingF4 dikombinasikan dengan Alt berfungsi menutup programF5 digunakan untuk RefreshF6 diguanakn untuk AddressingF7 digunakan untuk memeriksaan kesalahan ketik dan ejaan teksF8  sebagai Awal perintah penyorotan atau pemilihan teks atau objekF9 untuk Mengupdate Field (Mail Merge)F10 dkombiasi dengan tombol Shift berfungsi sebagai alternatif klik kanan.  F11 digunakan untuk memasukkan field berikutnya (Mail Merge)F12 digunakan untuk save as, sebagai tombol Alternatif dan untuk mengaktifkan atau memulai penggunaan menu bar.

33. Tombol yang berfungsi mengubah fungsi tombol penunjuk menjadi tombol numerik adalah....


Jawaban:

Tombol NumLock

Penjelasan:

Perhatikan contoh gambar keyboard berikut ini (khususnya di bagian yg ditunjukkan tanda panah kuning):

- bila tombol NumLock dalam kondisi aktif, maka tombol penunjuk berubah menjadi tombol numerik.

- bila tombol NumLock dalam kondisi nonaktif, maka tombol numerik berubah menjadi tombol penunjuk.

Contohnya dapat diamati dengan seksama:

- tombol numeric 8 dapat berubah jadi tombol penunjuk panah atas

- tombol numeric 6 dapat berubah jadi tombol penunjuk panah kanan

- tombol numeric 4 dapat berubah jadi tombol penunjuk panah kiri

- tombol numeric 2 dapat berubah jadi tombol penunjuk panah bawah


34. apa fungsi tombol f1 berikan 5 fungsi​


Jawaban:

Tombol F1 merupakan salah satu tombol fungsi pada keyboard komputer yang umumnya digunakan untuk membuka menu bantuan (help) pada aplikasi atau sistem operasi. Berikut adalah beberapa fungsi lain dari tombol F1:

Bantuan: Tombol F1 paling sering digunakan untuk membuka menu bantuan pada aplikasi atau sistem operasi. Saat tombol F1 ditekan, sistem akan membuka layar bantuan yang berisi informasi tentang cara menggunakan aplikasi atau sistem operasi tersebut.

Pengaturan: Pada beberapa aplikasi atau sistem operasi, tombol F1 juga dapat digunakan untuk membuka menu pengaturan atau konfigurasi. Saat tombol F1 ditekan, sistem akan membuka layar pengaturan yang memungkinkan pengguna untuk mengubah beberapa pengaturan aplikasi atau sistem operasi tersebut.

Fungsi tambahan: Pada beberapa keyboard, tombol F1 dapat berfungsi sebagai tombol fungsi tambahan seperti mengatur volume atau mengubah kecerahan layar.

Debugging: Tombol F1 juga dapat digunakan dalam debugging atau penyelesaian masalah. Saat tombol F1 ditekan, sistem akan membuka layar debugging yang memungkinkan pengguna untuk melihat pesan kesalahan atau log sistem.

Shortcut: Tombol F1 dapat digunakan sebagai shortcut untuk beberapa fungsi pada aplikasi atau sistem operasi tertentu. Misalnya, pada beberapa program pengolah kata, tombol F1 dapat digunakan untuk membuka menu "search and replace" atau "find and replace".
