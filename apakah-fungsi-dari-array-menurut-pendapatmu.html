public class Main

{

   static int[] arr = {16,4,10,90,27,3,12,28};

   public static void swapData(int a,int b)

   {

       int temp = arr[a];

       arr[a] = arr[b];

       arr[b] = temp;

   }

   public static void sortData(int arrSize)

   {

       for(int i=0; i<arrSize; i++)

       {

           for(int j=1; j<(arrSize-i); j++)

           {

              if(arr[j-1]>arr[j])

                   swapData(j-1,j);

           }

       }

   }

   public static void showData()

   {

       System.out.println("Hasil Pengurutan: ");

       for(int i=0; i<arr.length; i++)

       {

           System.out.println(arr[i]);

       }

   }

public static void main(String[] args) {

 int n = arr.length;

 sortData(n);

 showData();

}

}

9. Teknik pencarian data dalam array dimensi 1 yang menelusuri semua elemen array dari awal sampai akhir, dimana data tidak perlu diurutkan terlebih dahulu (acak) adalah .......


Teknik pencarian data dalam array dimensi 1 yang menelusuri semua elemen array dari awal sampai akhir, dimana data tidak perlu diurutkan terlebih dahulu (acak) adalah:
linear search, atau sering disebut juga sebagai sequential search (pencarian beruntun).
 

Pembahasan

Pada algoritma linear search atau sequential search (pencarian beruntun), elemen array diperiksa satu persatu dari elemen pertama sampai elemen terakhir. Langkah pencarian berhenti ketika menemukan data yang dicari, atau dengan kata lain, elemen array yang sedang dievaluasi bernilai sama dengan data yang dicari. Pada kondisi ini, algoritma mengembalikan nilai indeks elemen tersebut.
Jika sampai elemen terakhir data yang dicari tidak ditemukan, maka algoritma mengembalikan suatu nilai di luar wilayah indeks array, misalnya 0 jika array diawali dengan indeks 1 (one-based array), atau –1 jika array diawali dengan indeks 0 (zero-based array).

Karena sifat pencariannya yang sekuensial, maka array tidak perlu diurutkan terlebih dahulu. Hal ini berbeda dengan algoritma pencarian lain, misalnya binary search, yang mensyaratkan data/array harus sudah terurut.

15. Terdapat array : a [4][3][2] maka jumlah elemen array tersebut adalah


Jenis array yang terdapat dalam soal adalah jenis array multi dimensi karena memiliki lebih dari dua subskrip. Array multi dimensi adalah array yang digunakan untuk menyebutkan array yang memiliki dimensi lebih dari dua subskrip. Untuk menghitung jumlah elemen (isi) dari array tersebut, maka kita harus mengalikan semua isi array yaitu 4x3x2 = 24. Jadi, jumlah elemen array tersebut ada 24 elemen.

Pembahasan:

Array adalah tipe data terstruktur yang berguna untuk menyimpan sejumlah data yang memiliki tipe sama. Bagian yang menyusun array disebut element (isi) array.

Array dapat dibedakan menjadi tiga bagian yaitu:

Array satu dimensi.Array dua dimensi.Array multi dimensi.

Jenis array yang digunakan pada soal diatas merupakan array multi dimensi. Array multi dimensi biasanya digunakan untuk menyebutkan array yang memiliki dimensi lebih dari dua subskrip, contohnya array tiga dimensi, array empat dimensi dan seterusnya. Bentuk array multi dimensi adalah sebagai berikut:

tipe_data [ukuran 1][ukuran 2][ukuran N]

Contoh:

Array tiga dimensi yang memiliki ukuran 1 sebanyak 2, ukuran 2 sebanyak 3, ukuran 3 sebanyak 4, maka penulisan array adalah sebagai berikut:

a [2][3][4]

Kelebihan array antara lain:

Memiliki struktur penyimpanan data yang paling mudah.Memiliki lebih dari satu data.Semua elemen dapat terisi.Memiliki waktu akses yang sama untuk setiap data.Dapat diakses secara random.

Kekurangan array antara lain:

Struktur penyimpanan data bersifat statis.Penggunaan memori menjadi boros jika banyak elemen yang tidak digunakan.Memiliki banyak bahasa pemrograman sehingga sulit diubah saat dieksekusi.

Pembahasan:

Teknik pencarian data dalam array dimensi 1 yang menelusuri semua elemen array dari awal sampai akhir, dimana data tidak perlu diurutkan terlebih dahulu (acak) adalah Linear Search.  Linear Search merupakan suatu teknik pencarian data dimana dalam pencarian nya dilakukan secara satu persatu dan dilakukan secara acak.

Pada teknik pencarian ini jika ditemukan suatu data yang cocok maka program akan mengembalikan output, namun apabila tidak ditemukan kecocokan data maka pencarian akan terus berlanjut hingga akhir dari array tersebut.

Algoritma ini tidak sesuai digunakan untuk set data dengan jumlah yang cukup besar. Hal tersebut dikarenakan kompleksitas dari algorithma ini adalah Ο(n) di mana n adalah jumlah item. Apabila data yang dicari berada di paling akhir dari array, maka program harus menelusuri semua array terlebih dahulu.

31. Urutkan array (3,5,2,4) dengan algoritma insertion sort


Pengurutan array (3,5,2,4) secara algoritma insertion sort adalah sebagai berikut langkahnya:

3  5  2  43  2  5  43  2  5  42  3  5  42  3  4  5

Insertion sort membandingkan suku ke n dan n+1 jika n+1 lebih kecil dari n maka dipindah paling kiri. Begitupun terjadi secara berulang.

Pembahasan

Algoritma adalah sekumpulan instruksi  terstruktur dan terbatas yang diimplementasikan dalam bentuk program komputer untuk memecahkan masalah komputasi tertentu. Dalam matematika dan ilmu komputer, algoritma adalah prosedur langkah demi langkah untuk perhitungan, instruksi yang diberikan kepada komputer untuk melakukan tugas tertentu. Instruksi secara khusus ditulis untuk menyelesaikan tugas  tanpa kesalahan. Dimulai dengan istilah yang digunakan dan diakhiri dengan prosedur: Algoritma pengurutan penyisipan pada dasarnya mengurutkan data yang akan diurutkan menjadi dua bagian: bagian yang tidak disortir dan bagian yang  diurutkan. 36. Terdapat Array : A [2][2] maka jumlah elemen Array tersebut adalah …


Terdapat Array: A[2][2] maka jumlah elemen array tersebut adalah 4 elemen. Untuk mengetahui jumlah elemen pada array dua dimensi tersebut, kita dapat menganggap array tersebut sebagai sebuah matriks, dimana dengan ukuran 2x2 maka, akan didapat empat elemen didalam matriks.

Pembahasan

Array sendiri adalah suatu struktur data yang menyimpan sekelompok elemen dimana setiap elemen akan ditandai dengan suatu index. Array dapat berupa array satu dimensi, dua dimensi, maupun multi dimensi. Dalam konteks soal ini, array yang digunakan adalah array dua dimensi. Array dua dimensi dapat diidentifikasi dengan memiliki dua index penanda seperti, alamat suatu titik pada diagram kartesius. Bentuk umum dari array dua dimensi:

TipeData Variabel [x][y]

Contoh:

int contohMatriks [4][3]

Tipe data dapat berupa int, float, char, dan sebagainyaVariabel merupakan penamaan array yang disesuaikan kebutuhanX jika dianalogikan dalam matriks adalah jumlah barisY jika dianalogikan dalam matriks adalah jumlah kolom

Sehingga dalam array: A[2][2], maka memiliki dua baris dan dua kolom, dengan index:

A[0][0]   A[0][1]

A[1][0]    A[1][1]
