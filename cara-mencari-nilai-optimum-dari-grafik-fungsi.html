1. Nilai Optimum dari grafik fungsi, adalah.…........​


Jawab:

Nilai optimum dari grafik fungsi merupakan nilai tertinggi atau terendah yang dicapai oleh fungsi tersebut. Jika nilai optimum merupakan nilai tertinggi, maka disebut sebagai maksimum, sedangkan jika nilai optimum merupakan nilai terendah, maka disebut sebagai minimum. Nilai optimum ini berguna untuk mengetahui titik kritis atau titik optimal dalam suatu masalah atau perhitungan, seperti dalam optimisasi atau analisis risiko. Contohnya, jika kita ingin memaksimalkan keuntungan suatu investasi, maka kita perlu menentukan nilai optimum dari fungsi yang mewakili keuntungan tersebut untuk mengetahui jumlah investasi yang ideal.

Nilai optimum dari grafik fungsi kuadrat 2/5 x² - 3x + 15 = 0 adalah 9,375. Untuk menyelesaikan soal ini, kita gunakan formula y = -D/4a.

Penjelasan dengan langkah-langkah:

Diketahui:

Persamaan kuadrat 2/5 x² - 3x + 15 = 0

Ditanya:

Nilai optimum dari grafik fungsi kuadrat 2/5 x² - 3x + 15 = 0.

Jawab:

Untuk menyelesaikan soal ini, kita gunakan formula y = -D/4a.

Dari persamaan grafik fungsi 2/5 x² - 3x + 15 = 0

a = 2/5

b = -3

c = 15

D = b² - 4ac

Kita cari nilai optimumnya.

y = -D/4a

y = -(b² - 4ac)/4a

y = -(9 - 24)/(8/5)

y = 15 x 5/8

y = 9,375.

Nilai optimum dari grafik fungsi kuadrat 2/5 x² - 3x + 15 = 0 adalah 9,375.

7. Nilai optimum dari grafik fungsi kwadrat 3x² - 6x + 8 = 0 adalah


Jawab

8

Pembahasan

Nilai optimum adalah nilai puncak grafik (paling ujung) apabila ada titik puncak (x,y) berarti nilai optimum kita ambil yang y. Nilai optimum dapat kita cari dengan rumus sebagai berikut

[tex]y = \frac{-D}{4a} = \frac{-(b^{2}-4ac) }{4a}[/tex]

Keterangan

D = diskriminan

a = terletak di depan, contoh: 3x²

b = terletak di tengah, contoh: -6x

c = terletak di belakang, contoh: 8

Pembahasan Soal

→ Mencari diskriminan (D)

D = b²-4ac

D = (-6)²-4(3)(8)

D = 36 - 96

D = -60

→ Mencari nilai optimum (y)

[tex]y = \frac{-D}{4a}[/tex]

[tex]y = \frac{-(-96)}{4 (3)\\}[/tex]

[tex]y = \frac{96}{12}[/tex]

[tex]y = 8\\[/tex]

Jadi, nilai optimum dari grafik fungsi kuadrat tersebut adalah 8.

20. nilai optimum dari grafik fungsi y=5x pangkat 2 - 7x adalah

y = 5x² - 7x

a = 5 , b = -7 , c = 0

Cara pertama :

kita cari nilai diskriminannya

D = b² - 4ac

D = (-7)² - 4(5)(0)

D = 49 - 0

D = 49

kita subtitusikan nilai D ke dalam rumus y-puncak sehingga

yp = -D / 4a

yp = -49 / 4(5)

yp = -49 / 20

yp = -2,45

Jadi nilai optimumnya adalah

-49/20 atau -2,45

Cara kedua :

kita cari nilai dari x-puncaknya (xp)

xp = -b / 2a

xp = -(-7) / 2(5)

xp = 7 / 10

kita subtitusikan nilai x-puncak ke dalam fungsi kuadratnya

y = 5x² - 7x

f(x) = 5x² - 7x

f(xp) = 5(xp)² - 7(xp)

f(xp) = 5(7/10)² - 7(7/10)

f(xp) = 5(49/100) - 49/10

f(xp) = 245/100 - 490/100

f(xp) = -245/100

f(xp) = -49/20

f(xp) = -2,45

jadi nilai optimumnya adalah

-49/20 atau -2,45

27. Jika diketahui persamaan grafik fungsi kuadrat adalah y = -2x² + 8x - 6,maka tentukanlah nilai optimum dari persamaan grafik fungsi tersebut​


Jawab:

(2, 2)

Penjelasan dengan langkah-langkah:

Dalam fungsi kuadrat y = -2x² + 8x - 6, koefisien a = -2, koefisien b = 8, dan koefisien c = -6.

Titik puncak parabola dapat ditemukan menggunakan rumus x = -b/2a. Dalam hal ini, x = -8/(2*(-2))

                         = -8/(-4)

                        = 2.

Untuk mencari nilai y dari titik puncak tersebut, kita perlu memasukkan nilai x = 2 ke dalam fungsi kuadrat.

y = -2(2)² + 8(2) - 6

= -2(4) + 16 - 6

= -8 + 16 - 6

= 2.

Jadi, titik puncak atau nilai optimum dari persamaan grafik fungsi kuadrat adalah (2, 2).

Cari dahulu kedua akar persamaan tersebut

[tex]x^2-5x+4=0\\(x-4)(x-1)=0\\x_1=4,x_2=1[/tex]

Tambahkan kedua akar tersebut lalu bagi dua

[tex]x_p=\frac{4+1}{2}=\frac{5}{2}[/tex]

Substitusikan nilai barusan ke persamaan

Jawabannya adalah [tex](\frac{5}{2},-\frac{9}{4})[/tex]


29. nilai optimum dari grafik fungsi f(x)=6x²-3 adalah​


Nilai optimum dari grafik fungsi f(x) = 6x² - 3 adalah 3.

Pembahasan

FUNGSI KUADRAT

Bentuk Umum: f(x) = ax² + bx + c, a≠0

a : koefisien dari x²b : koefisien dari xx : variabelc : konstanta

Sumbu Simetri: [tex]x = -\frac{b}{2a}[/tex]

Nilai ekstrim/optimum: [tex]y = -\frac{D}{4a}[/tex]

Titik Puncak: ([tex]x = -\frac{b}{2a}[/tex], [tex]y = -\frac{D}{4a}[/tex])

Determinan: D = b² - 4ac

Diketahui:

f(x) = 6x² - 3

Ditanyakan:

Nilai optimum

Jawab:

Nilai optimum:

[tex]y = - \frac{D}{4a} [/tex]

[tex]y = - \frac{( {b}^{2} - 4ac)}{4a} [/tex]

[tex]y = - \frac{( {0}^{2} - 4(6)(3)) }{4 \times 6} [/tex]

[tex]y = - \frac{ ( - 4 \times 6 \times 3)}{4 \times 6} [/tex]

[tex]y = - ( - 3)[/tex]

[tex]y = 3[/tex]

Jadi, nilai optimumnya adalah 3.
