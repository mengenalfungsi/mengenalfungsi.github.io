3. Tuliskan menu dan ikon pada openoffice org writer​


MENU:

1.Menu File, jika menu ini dipilih maka akan memunculkan Drop-Down menu yang disebut sub menu, di antaranya: New, Open, Save, Close, Print, Exit dan sebagainya. Menu ini berhubungan dengan pengaturan dokumen, seperti membuat dokumen baru, membuka, menyimpan dan mencetak.

2.Menu Edit, berfungsi dalam proses peng-editan atau perubahan tulisan. Di dalamnya sub menu Cut, Copy, Paste, Undo, Redo, Select All, dan sebagainya. Seperti menyalin, memindahkan dan memilih teks atau kalimat pada dokumen.

3.Menu View, menu ini berfungsi untuk mengatur tampilan lembar kerja dan di dalamnya terdapat sub menu PrintLayout, Web Layout, Toolbars, Zoom dan sebagainya. Seperti mengubah ukuran lembar kerja, bentuk tampilan dokumen, meng-aktifkan dan me-non-aktifkan menu toolbar.

4.Menu Insert, berfungsi untuk menyisipkan objek tertentu ke dalam lembar kerja. Di dalamnya terdapat sub menu Manual Breaks, Row, Column, Picture, Table, Object dan sebagainya.

5.Menu Format, berfungsi untuk mengatur format dari lembar kerja seperti format teks, lebar baris dan kolom, ukuran kertas, paragraf, perataan teks dan sebagainya.

6.Menu Table, berfungsi untuk membuat, mengatur dan menyunting tabel.

7.Menu Tools, berfungsi untuk mengubah dan mengatur berbagai macam aturan pada lembar kerja, seperti bahasa, pemeriksaan ejaan, jumlah kata, mail merge, dan sebagainya.

8.Menu Window, berisi menu-menu yang digunakan untuk mengatur dan memilih jendela kerja aplikasi org Writer yang aktif.

9.Menu Help adalah menu bantuan yang berisi fasilitas petunjuk bantuan penggunaan org Writer.
