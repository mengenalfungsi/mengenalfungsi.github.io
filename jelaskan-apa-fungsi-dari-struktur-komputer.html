Bilangan Desimal = (d[n-1] * 2^(n-1)) + (d[n-2] * 2^(n-2)) + ... + (d[1] * 2^1) + (d[0] * 2^0)

Di mana d[i] adalah digit biner ke-i, dan n adalah jumlah digit dalam bilangan biner.

Bilangan Desimal = (1 * 2^4) + (1 * 2^3) + (0 * 2^2) + (0 * 2^1) + (1 * 2^0)

= 16 + 8 + 0 + 0 + 1

= 25

Jadi, bilangan desimal dari 11001 adalah 25
