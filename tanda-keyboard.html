6. tombol bertanda logo windows pada keyboard adalah tombol shortcut yang berfungsi untuk...


Tombol bernama Windows Logo Key itu ternyata mempunyai fungsi yang bisa memudahkan pengguna untuk mengakses menu-menu lain yang ada di komputer bila dipadukan dengan tombol lain (Shortcut). Tapi shortcut yang ada hanya akan berjalan apabila komputermu menggunakan sistem operasi Windows.

Windows Logo Key berada di antara tombol Alt dan Fn/ Ctrl yang berada di sebelah kiri bawah bagian keyboard. Nah, berikut beberapa fungsi shortcut yang bisa digunakan dengan bantuan Windows Logo Key. 

1. Windows: untuk menampilkan Start menu.

2. Windows Logo + D: untuk menampilkan desktop, meminimasi aplikasi windows yang terbuka atau mengembalikan (restore) semua aplikasi windows yang terbuka.

3. Windows Logo + E: untuk membuka Windows Explorer.

4. Windows Logo + F: untuk menampilkan menu pencarian file (Search Results).

5. Windows Logo + Ctrl + F: untuk menampilkan menu pencarian "Search Results - Computer".

6. Windows Logo + F1: untuk membuka menu "Help and Support Center".

7. Windows Logo + R: untuk membuka kotak dialog "Run".

8. Windows Logo + break: untuk membuka kotak dialog "System Properties".

9. Windows Logo + M: untuk meminimasi semua aplikasi windows yang sedang dibuka atau dengan kata lain kembali ke desktop.

10. Windows Logo + shift + M: kebalikan dari Windows Logo + M, yakni untuk meng-undo semua windows yang diminimasi.

11. Windows Logo + L: untuk mengunci workstation

12. Windows Logo + U: untuk membuka menu "Utility Manager". 

13. Windows + tab, layar komputer akan terganti menjadi 3D.

14. Windows + R lalu ketik psr.exe, untuk  memulai rekaman.

15. Tekan windows +R dan masukan osk, di layar monitor akan muncul keyboard virtual

16. Windows ditambah "+" atau "-" , maka kaca pembesar akan muncul di layar, dan kita bisa memperbesar atau memperkecil gambar di layar sesuai keperluan.

Itulah beberapa fungsi tombol Windows Logo Key yang bisa dipadukan dengan beberapa tombol lain. Selamat mencoba!

27. tombol pada keyboard yang berfungsi untuk mengaktifkan dan menonaktifkan referensi/tanda absolute($) yaitu​


Pada keyboard Microsoft Excel tombol yang digunakan untuk mengaktifkan dan menonaktifkan referensi absolut adalah tombol "F4". Saat kursor di posisi sel tertentu, tekan tombol F4 untuk mengaktifkan atau menonaktifkan referensi absolut untuk sel tersebut. Setiap kali tombol F4 ditekan, referensi akan berganti antara absolut, relatif, dan kombinasi dari keduanya. Selain itu juga, kamu bisa menambahkan tanda $ (dollar) sebelum nomor kolom dan baris untuk menjadikan referensi sel sebagai absolut

31. bagaimana cara mengetik tanda-tanda baca pada keyboard dgn 10jari


1. Mengetik tanpa editing, saat memulai
mengetik sebaiknya jangan pedulikan formating, kesesuaian huruf dan tanda baca.
Karena saat mengetik otak kiri kita yang aktif bekerja. Waktu editing nanti
barulah di perhatikan secara mendetil.

2. Minimalkan penggunaan mouse, saat
tangan Anda berpindah dari keyboard ke mouse memakan waktu beberapa detik. Pada
saat yang sama sebenarnya Anda bisa mengetik jauh lebih banyak hurup.

3. Hindari terlalu banyak menekan tombol
backspace, saat Anda salah mengetik kalimat sebaiknya biarkan saja. Ada waktunya
nanti editing untuk merapikan ketikan Anda. Saat mengetik, fokuslah pada
kuantitas bukan kualitas. Untuk mendapatkan kualitas ketikan yang baik, lakukan
editing setiap kali habis mengetik.

4. Maksimalkan penggunaan jari jempol,
jari jempol sebenarnya berfungsi untuk menekan tombol spasi. Namun hal ini
jarang sekali untuk di lakukan. Jari yang paling aktif bekerja adalah telunjuk
dan jari tengah. Sementar yang lainnya hanya menunggu.

5. Jangan berhenti, untuk mendapatkan
kuantitas ketikan sebaiknya jangan pernah berhenti bahkan untuk jeda
sedetikpun. Saat jeda, yakinlah bahwa inspirasi Anda akan berakhir di ujung jari.

6. Berkonsentrasilah, saat mengetik
sebaiknya jaga konsentrasi dan fokus Anda. Jauhkan dari gangguan yang dapat
mengakibatkan Anda kehilangan kecepatan mengetik.

7. Hindari menggaruk kepala, ini waktunya
mengetik jadi saat Anda menggaruk kepala sambil mengetik hasilnya akan semakin

32. Sebutkan tuts tanda / tombol lainnya yang sering digunakan pada keyboard!​


Jawaban:

- tombol Esc

tombol escape untuk membatalkan perintah atau dialog yang kita masukkan akan di batalkan

- tombol tab

tombol tab berfungsi jika ditekan akan memindahkan cursor ke arah kanan dengan satu tabulansi

- tombol caps lock

jika menekan tombol ini maka lampu indikator caps lock pada keyboard menyala, hal ini menandakan bahwa anda menggunakan mode semua huruf besar atau semua huruf kapital, jika anda menekan lagi dan lampu indikator mati maka dalam mode seperti biasa atau semua huruf kecil

- tombol shift

fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a=A), dan untuk memasukkan input yang di atas angka 1,2,3 dan seterusnya (contoh : shift+1=!, shift+/=?)

- tombol ctrl

khusus untuk tombol ini fungsinya untuk mentisipkan perintah secara pintas. contohnya untuk mencopy file tinggal tekan ctrl+c.

- tombol Alt

funsinya sama seperti tombol ctrl yaitu untuk menyisipkan perintah secara pintas. contohnya Alt+F4, itu untuk menutup program yang sedang di jalankan.

- tombol enter

sebagai perintah "ok" yang di berikan oleh program pada komputer dan untuk membuat paragraf baru.

- tombol back space

funsinya untuk mendelete satu huruf yang berada di sebelah kiri kursor. jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata di sebelah kiri kursor

- tombol delete

tombol ini sama seperti tombol backspace, bedanya itu jika tombol delete menghapus satu huruf di sebelah kanan kursor.

- tombol macro

macro ini jika di tekan akan muncul seperi ini "/".

- tombol insert

untuk menyisipkan karakter di posisi kursor

- tombol home

fungsinya untuk langsung menuju bagian atas halaman atau lembar kerja

- tombol End

fungsinya untuk langsung menuju bagian bawah halaman atau lembar kerja

- tombol page Up

fungsinya untuk menggerakkan baris satu demi satu layar ke atas

- tombol page down

fungsinya untuk menggerakkan baris satu demi satu layar kie bawah
