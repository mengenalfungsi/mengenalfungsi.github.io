1. macam macam tombol Size button dan fungsinya


Ada 4 tombol size button yang berguna untuk mengatur besar kecilnya suatu lembar kerja dalam perangkat elektronik yakni :
a) close = menutup jendela
b) minimize = mengecilkan jendela hingga ke taskbar
c) maximize = membesarkan jendela hingga satu layar penuh
d) Restore = mengembalikan ke ukuran jendela semula

13. 1. Sebutkan kegunaan dari:-New tab button-Tombol reload-Tombol back-Tombol close-Tombol minimize-Address bar-Tombol forward-Tombol maximize-Close tab button-Tombol restore​


Jawaban:

- new tab button: berguna untuk membuka tab baru.

-Tombol reload : berguna untuk melakukan refresh tampilan.

-Tombol back : berguna untuk menampilkan halaman yg sebelumnya diakses.

-Tombol close : berguna untuk menutup aplikasi.

-Tombol minimize : berguna untuk meminimalkan tampilan aplikasi di taskbar.

-Address bar : berguna untuk mengetikkan alamat situs website yg akan diakses.

-Tombol forward : berguna untuk menampilkan halaman yg berikutnya (yg sudah diakses).

-Tombol maximize:  berguna untuk memaksimalkan tampilan aplikasi hingga memenuhi layar monitor.

-Close tab button : berguna untuk menutup tab yg aktif (tapi tidak menutup aplikasi).

-Tombol restore​: berguna untuk menampilkan jendela aplikasi dg ukuran tidak maksimal.

Penjelasan:

Tombol-tombol di atas menunjukkan bahwa aplikasi yg dimaksud ialah aplikasi web browser.

Terdapat 4 pilihan yang berkaitan dengan ukuran tampilan layar kerja pada Microsoft Word yang lazim disebut Size Button, yakni:

→ RESTORE 
→ MINIMIZE
→ MAXIMIZE
→ CLOSE

RESTORE, berfungsi dalam mengembalikan jendela kerja pada ukuran yang semula.

CLOSE, berfungsi menutup jendela dokumen kerja.

MINIMIZE, berfungsi memperkecil ukuran dari jendela kerja.

MAXIMIZE, berfungsi dalam memperbesar ukuran dari jendela kerja.

Keempat size button ini terdapat pada Tittle Bar atau baris judul. Size Button ini juga dikenal dengan istilah Kontrol Menu.
