14. 3. Ketiklah kalimat berikut! "Saya belajar menggunakan tombol-tombol kombinasi keyboard menggunakan program Microsoft Word untuk mempercepat dalam pengoperasian Microsoft Word". Lakukan hal-hal berikut dengan melihat pada tabel kombinasi tombol keyboard! a. Blok kalimat tersebut → tekan Ctrl + A. b. Tekan tombol Ctrl + C, dan tekan Ctrl + V sebanyak 10 X. Apa yang terjadi? Berikan penjelasan singkat!​


Jawaban :

Yang terjadi adalah kalimat "Saya belajar menggunakan tombol-tombol kombinasi keyboard menggunakan program Microsoft Word untuk mempercepat dalam pengoperasian Microsoft Word" akan tercopy-paste (terduplikat/ tersalin) sebanyak 10 ×.

Penjelasan :

Ctrl + A fungsinya untuk memblok teks.

Ctrl + C fungsinya untuk meng-copy teks (menduplikat teks)

Ctrl + V fungsinya untuk meng-paste teks (menempelkan duplikat teks)

Jadi, kalau menekan tombol Ctrl + C dan tekan Ctrl + V sebanyak 10 ×, teks atau kalimat tadi jadi tercopy-paste / terduplikat sebanyak 10×.
