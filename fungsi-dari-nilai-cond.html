Fact : ………………………………………………… 2.If Mother ( bring ) much money, she (buy )a new phone.Cond. Sent.Type 3 : ……………………………………….. If I ( know ) your address, I ( take ) you. Cond. Sent Type 3 : ………………………………………. Fact : ……………………………………………………… 5. If it (rain ) hard, I ( stay ) at home today. Cond. Sent. Type 3 : ………………………………………. Fact : …………………………………………………………. 6. If We ( not take ) an umbrella, We (get ) wet on the way. Cond. Sent. Type 3 : …………………………………………. Fact : …………………………………………………………… 7. If Students (pass ) UN, The teacher ( be ) happy.. Cond. Sent. Type 3 : ………………………………………………. Fact : ……………………………………………………………


CONDITIONAL SENTENCE

[1]

Type 3 : If I had been fine, I would have gone for a work.

Fact : I wasn't fine, so I didn't go for a work.

[2]

Type 3 : If my mother had brought much money, she would have bought a new phone.

Fact : My mother didn't bring much money, so she didn't buy a new phone.

[3]

Type 3 : If John hadn't made a noise, the baby would have slept well.

Fact : John made a noise, so the baby didn't sleep well.

[4.]

Type 3 : If I had known your address, I would have taken you.

Fact : I didn't know your address, so I didn't take you.

[5.]

Type 3 : If it had rained hard, I would have stayed at home yesterday

Fact : It didn't rain hard, so I didn't stay at home.

[6.]

Type 3 : If we hadn't taken an umbrella, we would have got wet on the way.

Fact : We took an umbrella, so we didn't get wet on the way.

[7.]

Type 3 : If students had passed UN, the teacher would have been happy.

Fact : Students didn't pass UN, so the teacher was sad.

EXPLANATION

Conditional type 1 [First conditional] menjabarkan keinginan yang kemungkinan terjadi saat ini ataupun nanti-nanti (real possibility) berdasarkan prediksi.

Rumus

If+Simple present, Simple future

If+S+verb1(s/es)..., S+will+verb1......

Exp.

If she has enough money, she will buy a new car.

Fact ➡ She probably buys a new car someday

Conditional type 2 [Second conditional] menjabarkan keinginan yang diragukan terjadi saat ini atau mendatang.

Rumus :

If+Simple past, Past future

If+S+verb2 ......, S+would+verb1......

Exp.

If I didn't visit my neighbor's house, I would meet Mr. Jack.

Fact ➡ I visit my neighbor's house, so I don't meet Mr. Jack.

Conditional type 3 [The third conditional] menjabarkan keinginan yang tidak terwujud di waktu lampau. Pengulasannya saat ini hanyalah sebagai dampak dari kejadian di waktu lampau.

Rumus :

If+Past perfect, Past future perfect

If+S+had+verb3...., S+would+have+verb3....

Exp.

If he had told the truth, his boss wouldn't have been angry.

Fact ➡ he didn't tell the truth, so his boss was angry.

32. Perhatikan segitiga siku-siku dibawah ini! 8 cm 6 cm maka besar sudut tan A adalah... A B C D E 54 2√7 6 2√7 1 100 6 8 8 COND A <​


Jawab:

Untuk mencari besar sudut tan A dalam segitiga siku-siku, kita dapat menggunakan fungsi trigonometri.

Dalam segitiga siku-siku, sudut yang merupakan sudut tegak adalah sudut A.

Diberikan panjang sisi-sisi segitiga sebagai berikut:

AB = 8 cm (sisi yang tegak)

BC = 6 cm (sisi yang sejajar dengan sudut A)

Dalam segitiga siku-siku, kita dapat menggunakan fungsi trigonometri tan untuk mencari besar sudut A:

tan A = AB/BC

Substitusikan nilai sisi ke dalam persamaan:

tan A = 8/6

Sederhanakan persamaan:

tan A = 4/3

Kita dapat mencari nilai sudut A dengan mengambil arktan dari kedua sisi persamaan:

A = arctan(4/3)

Menggunakan kalkulator, kita dapat menghitung nilai sudut A:

A ≈ 53.13°

Jadi, besar sudut tan A dalam segitiga siku-siku dengan panjang sisi AB = 8 cm dan BC = 6 cm adalah sekitar 53.13°.
