2. 3. Selesaikan kasus berikut : Pedagang Kue memiliki modal Rp. 1.000.000,00 untuk membeli Kue Pie dan Donat untuk dijual kembali. Harga beli tiap Kue Pie Rp 4000,00 dan Donat Rp 1.600,00. Tempat etalasenya hanya bisa menampung 400 buah. Tentukan jumlah kue pie dan jumlah donat agar kapasitas tempatnya maksimum dan gambarkan grafiknya.​


Banyaknya jumlah kue pie (x) dan donat (y) agar kapasitas tempatnya maksimum adalah 150 kue pie dan 250 donat.

Gambar grafik ada pada lampiran.

Penjelasan dengan langkah-langkah:

Sistem pertidaksamaan linear dua variabel (SPLDV) adalah suatu sistem pertidaksamaan yang terdiri dari 2 variabel (x dan y). SPLDV ditandai dengan ciri lambang <, >, ≤ , ≥.

Diketahui :

Modal pedagang kue Rp. 1.000.000

Harga kue pie Rp. 4.000

Harga donat Rp. 1.600

Tempat etalase menampung 400 buah kue

Ditanya : Jumlah kue agar kapasitas tempatnya maksimum dan gambarkan grafiknya

Jawab :

Kita misalkan x = kue pie dan y = donat maka ;

Model matematika :

4.000x + 1.600y ≤ 1.000.000 : 800

5x + 2y ≤ 1.250 ......(pers 1)

x + y ≤ 400 ......(pers 2)

x ≥ 0

y ≥ 0

setelahnya dicari titik sasaran nya dari persamaan 1 dan 2 ;

5x + 2y ≤ 1.250

jika x = 0 maka ;

2y ≤ 1.250

y = 625

(0, 625)

jika y = 0 maka ;

5x ≤ 1.250                

x = 250      

(250, 0)              

x + y ≤ 400

jika x = 0 maka ;

y = 400

(0, 400)

jika y = 0 maka ;

x = 400                    

(400, 0)

dari persamaan tersebut tidak didapat kapasitas maksimumnya  maka dicari dengan mengeliminasi pers 1 dan 2 yaitu ;

5x + 2y ≤ 1.250   × 1

x + y ≤ 400          × 2    

5x + 2y = 1250

2x + 2y = 800      

3x = 450

x = 150 dimasukan kedalam pers untuk mendapatkan y maka y = 250

Jadi, jumlah kue pie (x) dan donat (y) agar kapasitas tempatnya maksimum adalah (150, 250).

26. Bagaimana cara membuat diagram pie menggunakan nilai derajat??


Jawaban:

Untuk membuat sebuah diagram lingkaran yang telah diketahui datanya, maka kita harus dapat mengubah satuan masing-masing jumlah data yang telah diketahui nilainya. Jika dibuat dalam bentuk diagram lingkaran persen, maka kita harus mengubah datanya ke bentuk persen, sedangkan jika ingin dibuat dalam bentuk diagram lingkaran derajat, maka diubah menjadi satuan derajat.

Namun, untuk membuat diagram lingkaran bentuk persen atau derajat sebenarnya memiliki konsep yang sama. Dimana untuk menentukan besar sudut pada tiap-tiap ruas bagian lingkaran menggunakan rumus sebagai berikut:

Ruas sudut = (nilai data / total data) x 360º

Sementara itu, jika data yang telah diketahui akan diubah ke bentuk persen (%), maka rumus yang digunakan untuk mengubah nilai tiap-tiap data ke persen adalah sebagai berikut:

Nilai persen = (nilai data / total data) x 100%

Dari penjelasan tersebut, maka langkah-langkah untuk membuat diagram lingkaran dalam bentuk persen atau derajat adalah sebagai berikut:

Menghitung jumlah total data dari semua nilai dataMenghitung besar sudut pada masing-masing bagian data dengan rumus: (nilai data / total data) x 360ºJika nilai data akan dirubah ke bentuk persen, maka gunakan rumus: (nilai data / total data) x 100%Membuat lingkaran penuh dan membagi lingkaran menjadi beberapa ruas dengan besar sudut sesuai masing-masing data

30. pada saat kapan peneliti menggunakan penyajian dan poligon?ata berupa histogram, pie charth d


Pada saat kapan peneliti menggunakan penyajian data poligon, berupa histogram, pie charth?

Di bawah ini dijelaskan mengenai penyajian data poligon, histogram, dan pie chart.

Pembahasan  

Ada beberapa cara penyajian grafik yang dapat kita gunakan antara lain sebagai berikut :

1. grafik garis (line chart)

2. grafik lingkaran (pie chart)

3. grafik batang / balok (bar chart)

4. grafik berupa peta (cartogram)

5. histogram

Poligon

Penyajian Data Poligon memiliki bentuk serupa dengan diagram garis. Sama seperti dengan tujuan penyajian data dalam bentuk lainnya, penyajian data kelompok bentuk poligon digunakan untuk menyajikan data ke dalam bentuk yang lebih menarik. Selain itu, penyajian data kelompok bentuk poligon dapat memudahkan pembaca dalam melihat data.

Poligon tersusun dari dua kata, yaitu poly dan gone. Arti kata poly adalah banyak. Sedangkan arti dari gone adalah titik. Sehingga, poligon dapat diartikan sebagai sudut banyak. Ada dua jenis poligon, yaitu poligon tertutup dan poligon terbuka. Dikatakan poligon tertutup jika titik awal dan titik akhirnya bertemu pada satu titik. Poligon terbuka ditandai dengan titik awal dan titik akhir tidak bertemu pada satu titik. Jadi, penyajian data kelompok bentuk poligon termasuk ke dalam poligon terbuka.

Contoh gambar terlampir di bawah ya.

Histogram

Histogram adalah modifikasi dari diagram batang (bar diagram). Tiap tampilan batang menunjukan proporsi frekuensi pada masing-masing data kategori yang berdampingan dengan interval yang tidak tumpang tindih. Histogram adalah grafik balok yang memperlihatkan satu macam pengukuran dari suatu proses atau kejadian. Grafis ini sangat cocok untuk data yang dikelompokan.

Kegunaan dari pengggunaan histogram adalah sebagai berikut :

1. Diagram batang umumnya digunakan untuk mengambarkan  perkembangan nilai suatu objek penelitian dalam kurun waktu tertentu. Diagram batang menunjukan keterangan-keterangan dengan batang-batang tegak atau mendatar dan sama lebar dengan batang-batang terpisah.

2. Mengetahui dengan mudah penyebaran data yang ada.

3. Mempermudah melihat dan menginterpretasikan data.

4. Sebagai alat pengendali proses, sehingga dapat mencegah timbulnya masalah.

Ciri-ciri dari histogram antara lain sebagai berikut :

1. skala yang digunakan harus di mulai dari 0.

2. diagram batang yang di buat secara vertikal maupun horizontal.

3. diagram harus di lengkapi dengan judul.

4. diagram batang di sajikan dalam bentuk batangan.

Histogram ini juga digunakan dalam menyajikan data yang disusun dalam suatu distribusi frekuensi, distribusi persentase atau telah tersusun.

Pie chart

Pie Chart (diagram lingkaran) adalah penyajian data statistik yang dinyatakan dalam persen atau derajat dapat menggunakan diagram lingkaran.

Kelebihan Penggunaan Pie Chart adalah

a. tempat untuk membuat diagram lingkaran tidak terlalu besar.

b. diagram lingkaran sangat berguna untuk menunjukkan dan membandingkan proporsi dari data

Kekurangan Penggunaan Pie Chart adalah diagram lingkaran tersebut tidak dapat menunjukkan frekuensinya.

Contoh gambar terlampir di bawah ya.

titik tertinggi yang dapat dicapai = (2 , 9)

Penjelasan dengan langkah-langkah:

f(x) = -x² + 4x + 5

a = -1 , b = 4 , c = 5

titik tertinggi = titik puncak (Xp,Yp)

Xp = - b/2a

Xp = - 4/2(-1)

Xp = -4/(-2)

Xp = 2

Yp = f(Xp)

Yp = f(2)

Yp = -2² + 4 × 2 + 5

Yp = -4 + 8 + 5

Yp = 4 + 5

Yp = 9

Titik puncak = (2,9)

Jawaban:

(2,9)

Penjelasan dengan langkah-langkah:

ada dua cara menentukan titik maksimum dari sebuah fungsi kuadrat.

Cara pertama dengan menggunakan rumus

[tex]x_{maks}=\frac{-b}{2a} [/tex]

[tex]y_{maks}=\frac{D}{-4a} = \frac{ {b}^{2} - 4ac }{-4a}[/tex]

untuk suatu fungsi kuadrat y=ax²+bx+c

pada soal di atas dapat diketahui bahwa a=-1, b=4, dan c= 5, sehingga diperoleh

[tex]x_{maks}=\frac{-b}{2a} = \frac{-4 }{2(-1)} = \frac{-4}{-2} = 2[/tex]

[tex]y_{maks}=\frac{D}{-4a} = \frac{ {b}^{2} - 4ac }{-4a} = \frac{ {4}^{2} - 4( - 1)(5) }{ - 4( - 1)} = \frac{16 + 20}{4} = \frac{36}{4} = 9[/tex]

Cara kedua menggunakan turunan

nilai maksimum diperoleh saat y'=0, sehingga

y= -x²+4x+5

y'=-2x+4

-2x+4=0

-2x=-4

x=-4/-2

[tex]x_{maks}=2[/tex]

untuk mendapatkan y maks kita substitusi x=2 ke dalam persamaan y= -x²+4x+5

diperoleh

[tex]y_{maks}=-2²+4(2)+5=-4+8+5=9[/tex]
