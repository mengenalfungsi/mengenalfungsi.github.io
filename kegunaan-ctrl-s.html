3) Ctrl+N (New) digunakan untuk membuka (membuat) data baru

29. Bagaimana cara untuk memotong dokumen slide dengan menggunakan keyboard maka menekan … ctrl s ctrl a ctrl v ctrl l ctrl p


Jawaban:

Untuk memotong dokumen slide dalam program presentasi seperti Microsoft PowerPoint atau Google Slides, Anda dapat menggunakan fitur pemotongan (cut) yang tersedia di dalam program. Caranya adalah dengan memilih teks atau elemen yang ingin dipotong, lalu tekan tombol keyboard "Ctrl + X" untuk memotong dan "Ctrl + V" untuk menyisipkan (paste) ke tempat lain.

Namun, jika Anda ingin memotong dokumen slide di luar program presentasi, misalnya dengan menggunakan software pengolah gambar atau editor teks, caranya mungkin berbeda-beda tergantung pada program yang Anda gunakan.

Penjelasan:

Biasanya, Anda dapat menggunakan kombinasi tombol "Ctrl + X" untuk memotong, "Ctrl + V" untuk menyisipkan, dan "Ctrl + A" untuk memilih seluruh dokumen slide. Namun, pastikan untuk memeriksa dokumentasi atau menu bantuan program yang Anda gunakan untuk mengetahui cara yang tepat.

CTRL + C = Digunakan untuk meng-copy huruf atau gambar.

CTRL + S = Digunakan untuk menyimpan file atau dokumen.

36. apa fungsi dari:CTRL + sCTRL + dCTRL + b CTRL + cCTRL + nCTRL +hCTRL+ jCTRL + ZCTRL + P CTRL + e​


CTRL + S : Save
CTRL + D : Delete atau memilih font
CTRL + B : Menebalkan teks
CTRL + C : Copy
CTRL + N : Dokumen baru
CTRL + H : Menu find and replace
CTRL + J : Membuat teks rata kanan
CTRL + Z : Mengembalikan ke perubahan sebelumnya
CTRL + P : Print
CTRL + E : Membuat teks rata tengah

•CTRL + S (berfungsi untuk menyimpan dokumen).

•Ctrl + D - Memilih Jenis Huruf (Font).

•Ctrl + B - Menebalkan Huruf yang terpilih (Bold).

•CTRL + C (berfungsi untuk mengcopy/menyalin teks maupun File).

•CTRL + N (berfungsi untuk membuka dokumen baru).

•CTRL + H (berfungsi untuk membuka menu/kotak dialog Find and Replace (tab Replace).

•CTRL + J (berfungsi untuk rata kanan-kiri pada teks).

•CTRL + Z ((Undo) berfungsi untuk membatalkan perintah sebelumnya).

•CTRL + P (berfungsi untuk membuka menu/kotak dialog Print).

•CTRL + E (berfungsi untuk rata tengah teks).
