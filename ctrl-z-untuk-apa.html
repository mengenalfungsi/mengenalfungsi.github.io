1. CTRL+A ,CTRL+Z, dan CTRL+ F4 Berfungsi Untuk?


Ctrl + A = Memblok Seluruh Dokumen (All)
Ctrl + Z = Undo
Ctrl + F4 = Untuk Menutup Jendela AktifCtrl + A : memblok semua objek
Ctrl + Z : mengembalikan perintah yang sebelumnya dilakukan
Ctrl + F4 : menutup program yang aktif

2. ctrl+y Merupakan shortcut untuk ? ctrl + z merupakan shotcut untuk ?


Ctrl y merupakan shortcut untuk repeat. Ctrl z merupakan shortcut untuk undo.Untuk di Applikasi Microsoft Office:
Ctrl + z : Undo (membatalkan/mengembalikan perubahan terakhir)
Ctrl + y : Redo (kebalikan dari redo)

6. Tuliskan perbedaan antara ctrl+z dan ctrl+y​


Jawaban:

CTRL + Z ((Undo) berfungsi untuk membatalkan perintah sebelumnya)

Ctrl + Y = fungsi shortcut redo/membalikan tindakan, ini kebalikannya dari undo.

Pertanyaan:

Tuliskan perbedaan antara Ctrl + Z dan Ctrl + Y

Jawaban dan penjelasan:

Ctrl + Z = fungsi shortcut undo/pembatalan tindakan kembali ke sebelumnya

Ctrl + Y = fungsi shortcut redo/membalikkan tindakan, ini kebalikannya undo

Jadi, perbedaan antara Ctrl + Z dan Ctrl + Y adalah undo dan redo
