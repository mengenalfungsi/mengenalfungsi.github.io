1. Berikan contoh soal menghitung Asinambung Tak berhingga (Fx) = 0 dan asinambung Titik, Lim (Fx) ?​


Penyelesaian :

Contoh soal menghitung batas tak berhingga (lim) dari suatu fungsi dan batas titik (lim):

Misalkan kita memiliki fungsi f(x) = 1/x. Kita akan mencari nilai lim f(x) ketika x mendekati tak berhingga (x -> ∞) dan lim f(x) ketika x mendekati suatu titik tertentu (x -> a).

1. Batas tak berhingga (lim) dari f(x) ketika x mendekati tak berhingga:

lim(x -> ∞) 1/x = 0

Artinya, ketika x mendekati tak berhingga, nilai f(x) mendekati nol.

2. Batas titik (lim) dari f(x) ketika x mendekati suatu titik a:

Misalnya, kita akan mencari lim f(x) ketika x mendekati a = 2.

lim(x -> 2) 1/x = 1/2

Artinya, ketika x mendekati 2, nilai f(x) mendekati 1/2.

Jadi, contoh soal tersebut memberikan nilai lim f(x) ketika x mendekati tak berhingga (0) dan lim f(x) ketika x mendekati titik tertentu (1/2).


2. 3. Berikan contoh soal menghitung Asinambung Tak berhingga (Fx) = 0 dan asinambung Titik, Lim (Fx) ?​

Penjelasan dengan langkah-langkah:

Contoh soal untuk menghitung Asimptot Tak Berhingga (Asinambung Tak Berhingga):

Misalkan kita memiliki fungsi f(x) = (3x^3 - 2x^2 + 5x + 1) / (2x^2 - x + 3). Untuk mencari Asimptot Tak Berhingga, kita harus melihat keadaan ketika x mendekati tak terhingga (x -> ∞).

Langkah-langkahnya adalah sebagai berikut:

1. Periksa perbandingan koefisien tertinggi antara pembilang dan penyebut fungsi. Dalam contoh ini, koefisien tertinggi dari pembilang adalah 3 dan koefisien tertinggi dari penyebut adalah 2. Karena koefisien tertinggi pembilang lebih tinggi daripada koefisien tertinggi penyebut, Asimptot Tak Berhingga akan terjadi pada garis lurus dengan persamaan y = (koefisien tertinggi pembilang) / (koefisien tertinggi penyebut).

Dalam contoh ini, persamaan garis Asimptot Tak Berhingga adalah y = 3/2.

Contoh soal untuk mencari asimptot Titik dan memahami batas (lim):

Misalkan kita memiliki fungsi f(x) = (x^2 + 2x - 3) / (x + 1). Untuk mencari asimptot Titik dan memahami batas (lim), kita perlu memeriksa bagaimana fungsi tersebut berperilaku saat x mendekati suatu titik tertentu, dalam hal ini saat x -> -1.

Langkah-langkahnya adalah sebagai berikut:

1. Periksa koefisien tertinggi antara pembilang dan penyebut fungsi. Dalam contoh ini, koefisien tertinggi dari pembilang adalah 1 dan koefisien tertinggi dari penyebut adalah 1. Karena koefisien tertinggi pembilang sama dengan koefisien tertinggi penyebut, Asimptot Titik akan terjadi saat x mendekati titik di mana pembilang dan penyebut membagi nol, yaitu saat x = -1.

2. Untuk mencari batas (lim) saat x mendekati -1, kita perlu mengevaluasi fungsi saat x mendekati -1 dari kedua sisi (-1-) dan (-1+).

Jika kita menggantikan nilai x dengan angka yang mendekati -1 dari kedua sisi, misalnya x = -1.1 atau x = -0.9, kita dapat memberikan perkiraan nilai fungsi yang mendekati titik x = -1.

Misalnya, saat x mendekati -1 dari sisi kiri (x = -1-), kita bisa menggunakan x = -1.1, dan saat x mendekati -1 dari sisi kanan (x = -1+), kita bisa menggunakan x = -0.9.

Jadi, untuk fungsi f(x) = (x^2 + 2x - 3) / (x + 1), saat x mendekati -1, batas (lim) fungsi adalah:

- Saat x mendekati -1- (dari sisi kiri): lim x-> -1- (x^2 + 2x - 3) / (x + 1) = (-1.1^2 + 2(-1.1) - 3) / (-1.1 + 1) = (-0.1) / (-0.1) = 1

- Saat x mendekati -1+ (dari sisi kanan): lim x-> -1+ (x^2 + 2x - 3) / (x + 1) = (-0.9^2 + 2(-0.9) - 3) / (-0.9 + 1) = (-2.1) / (0.1) = -21

Jadi, saat x mendekati -1, batas (lim) fungsi adalah 1 dari sisi kiri dan -21 dari sisi kanan.

4. Lim x tak terhingga √x²+2x-√x²+x​


• Limit Fungsi  

-

Nilai dari  [tex]\tt \lim\limits_{x \to \infty} (\sqrt{x^{2} + 2x} - \sqrt{x^{2} + x} )[/tex] adalah

[tex]\boxed{\mathbf{Pembahasan :}}[/tex]

Limit merupakan operasi atau metode matematika dimana sesuatu dikatakan mendekati nilai suatu bilangan tertentu. Limit tak hingga adalah konsep untuk mengetahui kecendrungan suatu fungsi jika nilai variabelnya dibuat semakin besar yang disimbolkan [tex]\mathbf{\lim\limits_{x \to \infty } f(x) }[/tex] , artinya nilai x semakin besar atau bertambah besar tanpa batas.

Sifat Limit Tak Hingga :

Misalkan p(x) dan q(x) merupakan suatu polinom dengan ax2 dan px2 sebagai derajat atau suku pangkat tertinggi dari polinom , maka :

[tex]\Rightarrow \mathbf{Jika \: a=p \to \lim\limits_{x \to \infty} \sqrt{ax^{2} + bx + c} - \sqrt{px^{2} + qx + r} = \frac{b - q}{2\sqrt{a}}} \\ \Rightarrow \mathbf{Jika \: a > p \to \lim\limits_{x \to \infty} \sqrt{ax^{2} + bx + c} - \sqrt{px^{2} + qx + r} = \infty} \\ \Rightarrow \mathbf{Jika \: a < p \to \lim\limits_{x \to \infty} \sqrt{ax^{2} + bx + c} - \sqrt{px^{2} + qx + r} = - \infty}[/tex]

Perhatikan pembahasan soal berikut :

☞ Diketahui :

[tex]\tt \lim\limits_{x \to \infty} (\sqrt{x^{2} + 2x} - \sqrt{x^{2} + x} )[/tex]

☞ Ditanya :

Nilai limit = ___ ?

☞ Penyelesaian :

[tex]\mathbf{\lim\limits_{x \to \infty} \sqrt{x^{2} + 2x} - \sqrt{x^{2} + x} \to a = p } \\ \mathbf{Karena \: a = p , \: maka \: gunakan \: sifat :} \\ \\ \tt L = \frac{b - q}{2\sqrt{a}} \\ \tt L = \frac{2 - 1}{2\sqrt{1}} \\ \tt L = \frac{1}{2} \\ \\ \boxed{\mathbf{Maka, \: nilai \: dari \: limit \: adalah \: \frac{1}{2}}}[/tex]
