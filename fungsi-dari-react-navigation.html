8. apa bedanya componentDidMount dengan componentWillMount pada react js lifecycle?


Perbedaan komponen DidMount dengan Komponen WillMount dalam lifecycle ReactJs terletak pada sisi eksekusinya. Dalam hal ini komponen Didmount hanya dipanggil di sisi klien ketika sudah menerima datayang berasal dari server. Sementara komponen WillMount dipanggil hanya 1 kali sebelum komponen di render di sisi server untuk melakukan fungsi logic tambahan. Singkatnya WillMount bekerja di sisi server sedangkan DidMount bekerja pada browser atau perangkat pengguna.

Pembahasan:

componentWillMount

ComponentWillMount memiliki konstruktor yang kurang lebih mirip dengan konstruktor ES6. Dengan komponen ini kamu dapat mengatur properti-propreti dan status dengan menggunakan setState.

Sebagai contoh sebut saja ketika kamu ingin menyimpan tanggal ketika komponen tersebut dibuat

componentWillMount() {

 this.setState({ todayDate: new Date(Date.now())});

}

componentDidMount

Metode yang satu ini dipanggil setelah proses render() dan pada umumnya sih digunakan untuk memanggil API dimana kita bisa menginisialisasi DOM dari pihak ke-3.

Misalnya sebut saja kamu punya sebuah aplikasi umpan berita dan kamu menginginkan list berita yang muncul update secara otomatis setiap beberapa jam sekali.

componentDidMount() {

 this.interval = setInterval(this.fetchNews, 3600000);

}

10. sebutkan isi undang undang pelayaran act of navigation


Konflik Dengan Inggris

Sebagai koloni Amerika Utara tumbuh ketegangan dengan Inggris tak terelakkan. Inggris merasa bahwa koloni ada untuk kepentingan negara induk dan sikap ini pasti akan menimbulkan kebencian. Pada awal 1651 Parlemen Inggris mengeluarkan kebijakan navigasi. Ini menyatakan bahwa barang tumbuh atau dibuat di luar Eropa harus diangkut ke Inggris dengan kapal Inggris, peraturan Navigasi lain mengikutinya. 1660 Navigation Act menyatakan bahwa barang-barang tertentu (katun, indigo, gula dan tembakau) hanya dapat diekspor dari koloni ke Inggris atau koloni lainnya. Hal ini diikuti oleh tindakan pada 1670 dan 1673. Namun Inggris membuat sedikit usaha untuk menegakkan tindakan ini dan mereka secara luas diabaikan oleh koloni. (Setelah 1763 Inggris berusaha untuk menegakkan mereka lebih ketat, menyebabkan kemarahan besar di antara koloni).

Pada awal abad ke-18 penduduk koloni Amerika Utara tumbuh pesat. Itu mungkin sekitar 300.000 pada akhir abad ke-17 tetapi dengan 1760 itu lebih 1 juta. Dengan 1780 itu telah menjadi dua kali lipat. Pada awal abad ke-18 penduduk didorong oleh imigran dari Irlandia Utara (kebanyakan dari mereka berasal dari Skotlandia Presbiterian). Ada juga banyak imigran dari Skotlandia itu sendiri. Juga pada awal abad ke-18 ada banyak imigran Jerman datang, hal itu karena tanah murah di Amerika Utara dan menarik banyak orang berharap untuk kehidupan yang lebih baik


11. arti dari React kesukaan


React merupakan kata yang berasal dari bahasa asing, tepatnya Bahasa Inggris. Kata ini memiliki makna 'bereaksi' dan kata bendanya berbentuk 'reaction' yang bermakna reaksi. Kata ini dapat dengan mudah kita temui dalam kehidupan sehari-hari, salah satunya kerap muncul di berbagai situs di internet, termasuk situs-situs sosial media.


Pembahasan

Pada kesempatan kali ini, soal meminta kita untuk mencari arti atau makna dari frasa 'react kesukaan'. Frasa ini sendiri merupakan gabungan dari kata dalam bahasa asing dan kata dalam Bahasa Indonesia. JIka kita ubah kata dari bahasa asing tersebut ke dalam Bahasa Indonesia, maka kita akan mendapatkan frasa baru yaitu 'bereaksi kesukaan'.

Untuk mengetahui makna sebenarnya dari frasa ini, sebenarnya kita perlu mengetahui pada konteks apa frasa tersebut ditulis. Setidaknya, kita harus mengetahui tempat frasa ini digunakan atau ditayangkan.

Mari kita ambil satu contoh.

Jika frasa 'react kesukaan' ditampilkan pada situs-situs tayangan video seperti YouTube, maka frasa tersebut dapat berarti ajakan kepada kita untuk memberikan reaksi atau tanggapan terhadap video yang kita sukai. Tanggapan dapat berupa komentar, tanda suka, tidak suka, membagikan video atau lain sebagainya.

Hal yang kurang lebih sama juga berlaku pada situs-situs lainnya. Pada umumnya, ketika istilah ini muncul, kita diminta untuk memberikan reaksi. Namun, sekali lagi, makna ini belum pasti kebenarannya karena pada soal ini kita tidak diberikan informasi tentang lokasi penerapan frasa ini. Makna yang sebenarnya bisa sama sekali berbeda dengan yang kakak sajikan. Jadi, jika kamu memiliki informasi tentang tempat frasa ini ditayangkan, silahkan disampaikan. Kakak akan sebisa mungkin membantu.

Untuk artikel kali ini, kita akan fokus pada bagaimana membuat proyek Node.js baru.

Membuat Proyek Node.js

Sebelum membuat proyek, buatlah folder baru terlebih dahulu. Folder ini akan digunakan sebagai tempat penyimpanan berkas proyek dan JavaScript yang kita tulis nanti. Kami sarankan, Anda buat folder tersebut di alamat C -> javascript-projects -> nodejs-basic bagi pengguna Windows; home -> javascript-projects -> nodejs-basic bagi pengguna Linux atau macOS.

membuat proyek Node.js baru langkah 1

Selanjutnya, buka folder nodejs-basic menggunakan VSCode. Caranya, pada Visual Studio Code pilih menu File -> Open Folder -> [pilih foldernya]. Folder pun berhasil terbuka melalui VSCode.

membuat proyek Node.js baru langkah 2

Untuk membuat proyek JavaScript, silakan buka Terminal pada VSCode. Pilih menu Terminal -> New Terminal, kemudian tuliskan perintah: npm init

NPM alias Node Package Manager merupakan JavaScript Package Manager bawaan dari Node.js. Melalui NPM ini kita dapat membuat Node.js package (proyek) dan mengelola penggunaan package eksternal yang digunakan. Kita akan membahas NPM lebih detail nanti.

Jika Anda yang tidak menggunakan Visual Studio Code, gunakan Terminal/Command Prompt usungan OS Anda. Namun, sesuaikan lokasinya pada folder proyek ya.

Setelah menuliskan perintah di atas, Anda akan diberikan beberapa pertanyaan untuk mengisi nilai package name, version, description. Semua itu merupakan informasi dasar dari aplikasi yang Anda buat.

membuat proyek Node.js baru langkah 3

Nilai yang berada di dalam tanda kurung merupakan nilai default. Anda dapat menggunakan nilainya dengan langsung menekan tombol Enter. Untuk saat ini, cukup berikan semua pertanyaan dengan nilai default.

Setelah mengisi seluruh pertanyaan yang diberikan, Anda akan diberitahu untuk melihat hasil akhir yang dibuat pada berkas package.json.

membuat proyek Node.js baru langkah 4

Jika nilai yang ditampilkan sudah sesuai, langsung saja tekan tombol Enter. Berkas package.json pun telah berhasil dibuat pada proyek kita.

membuat proyek Node.js baru langkah 5

Voila! Anda berhasil membuat proyek Node.js.

REACT merupakan singkatan dari kata 

(1) Relating (mengaitkan),

(2) Experiencing (mengalami),

(3) Applying (menerapkan),

(4) Cooperating (bekerjasama), dan

(5) Transferring (memindahkan).

Model pembelajaran ini merupakan salah satu dari beberapa jenis stategi pengajaran yang berbasis kontekstual, dimana pembelajaran jenis ini lebih mengutamakan keaktifan peserta didiknya dalam kegiatan pembelajaran di dalam kelas.
