1. Apa fungsi amonium sulfat?


Kegunaan utama ammonium sulfat ialah sebagai pupuk untuk tanah basa (alkalis). Dalam tanah ion ammonium dilepaskan dan membentuk sejumlah kecil asam, yang menurunkan pH keseimbangan tanah, sambil berkontribusi menyumbang nitrogen esensial untuk pertumbuhan tanaman. Kerugian utama atas penggunaan ammonium sulfat ialah kandungan nitrogennya yang relatif rendah dibandingkan ammonium nitrat, yang meningkatkan biaya transportasi. Ammonium sulfat juga digunakan sebagai adjuvant semprot pertanian untuk inseksitisida, herbisida, dan fungisida yang larut dalam air. Di sana ammonium sulfat berfungsi untuk mengikat kation-kation besi dan kalsium yang ada baik dalam air maupun dalam sel. Ammonium sulfat terutama efektif sebagai adjuvant untuk herbisida 2,4-D (amina), glyphosate, dan glufosinate.

5. berapa gram amonium sulfat yang diperlukan untuk membuat larutan amonium sulfat 2 n 0,1 m dan 0,1 n sebanyak 250 ml?​


Jawaban:

Diketahui :

Volume NH3 = 500 ml = 0,5 L

Molaritas NH3 = 0,1 M = 0,1 mol/L

pH = 9

Kb = 10^-5

Ar H = 1 , N = 14 , O = 16 , S = 32

Ditanya : Berapa gram (NH4)2SO4 yang harus ditambahkan ?

Jawab :

pOH :

= 14 - pH

= 14 - 9

= 5

[OH-] :

- log [OH-] = pOH

- log [OH-] = 5

- log [OH-] = 10^-5

[OH-] = 10^-5

mol NH3 :

= M . V

= 0,1 mol/L . 0,5 L

= 0,05 mol

Indeks ion garam :

(NH4)2SO4 <----> 2NH4+ + SO4²-

Indeks ion = 2

mol (NH4)2SO4 :

= Kb × mol basa / [OH-] . indeks ion

= 10^-5 × 0,05 mol / 10^-5 . 3

= 0,025 mol

Mr (NH4)2SO4 :

= 2.Ar N + 8. Ar H + Ar S + 4.Ar O

= 2.14 + 8.1 + 32 + 4.16

= 132 gram/mol

Massa (NH4)2SO4 :

= n . Mr

= 0,025 mol . 132 gram/mol

= 3,3 gram

27. berapa titik didih larutan 6,6 gram amonium sulfat dalam 250 gram larutan jika kb air 0,5 ? (ar amonium sulfat = 132)


∆Tb = Kb × m × i
∆Tb = Kb × ( m (NH4)2SO4 / Mr ) × ( 1000 / m lar ) × i
∆Tb = 0,5°C/molal × (6,6 gr / 132) × (1000/250 gr) × 3
∆Tb = 0,3°C

Tb = Tb° + ∆Tb
= 100°C + 0,3°C
= 100,3°C

38. Pupuk ZA mengandung nitrogen sebagai (NH4)2SO4 (amonium sulfat), berapa gram amonium sulfat yang dapat dibuat dari 14,2 gram Nitrogen?


Jawaban:

66,94 gram

Penjelasan:

Mr (NH4)2SO4 = ( 2 x Ar N ) + ( 8 x Ar H ) + Ar S + ( 4 x Ar O )

=> ( 2 x 14 ) + ( 8 x 1 ) + 32 + ( 4 x 16 ) = 132

massa N = ( jmlh atom N x Ar N ) / Mr (NH4)2SO4 x massa (NH4)2SO4

14,2 = ( 2 x 14 ) / 132 x massa (NH4)2SO4

massa (NH4)2SO4 = 66,94 gram
