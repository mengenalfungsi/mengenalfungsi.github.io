4. Apa perbedaan icon bontang dan kotak putus2 pada komputer


Ikon bintang merupakan shape/bentuk untuk anda menggambar disana ada lingkaran, segitiga tanda panah kubus, dll. Sedangkan dengan ikon kotak garis putus2 berfungsi untuk memindahkan scr penuh atau sebagaian gambar didalamnya untuk di copy/di letakkan di bagian lain. Jadi jelas sekali beda & fungsinya

- keduanya adalah icon "Select". yg kotak = memilih area persegi, yg bintang = memilih area bebas (bidang tak beraturan)  

- agar jelas bedanya, slhkn dicoba begini :  

- buka Paint, pilih warna mis. hijau, klik icon [Fill_with_color], klik di "kertas kerja putih", warna kertas kerja jadi hijau.  

- klik icon [Select] (icon-kotak), klik&drag di kertas-kerja, muncul kotak titik2. klik Edit - Clear_selection. muncul kotak putih.  

- ulangi langkahnya, tp menggunakan icon [Free_form_select] (icon-bintang), buat bidang/area tak beraturan, klik Edit - Clear_selection.

32. fungsi kotak dialog page setup muncul ketika kita memberi perintah pada komputer untuk


Fungsi kotak dialog page setup muncul ketika kita memberi perintah pada komputer untuk melakukan pengaturan terhadap batas atau margin lembar kerja kita. Selain itu menu page setup juga terdapat pada saat kita akan mencetak lembar kerja kita.

Pembahasan

Menu Page setup sendiri dapat kita akses melalui menu page layout, atau bisa pada saat akan mencetak lembar kerja kita kemudian pilih print review. Adapun batas margin atau batas lembar kerja kita yang bisa diatur melalui menu page setup adalah sebagai berikut

Batas AtasBatas BawahBatas KananBatas Kiri

Selain itu pada page setup kita juga bisa melakukan beberapa pengaturan selain batas lembar kerja, pemilihan jenis kertas yang kita gunakan juga bisa melalui menu Page Setup.

Pelajari lebih lanjut

38. Apa stila tulisan dlalam sebiah kotak pada komputer apa


Jawaban:

Stelan tulisan dalam sebuah kotak pada komputer biasanya dapat diatur menggunakan program pengolah kata seperti Microsoft Word, Google Docs, atau aplikasi sejenisnya. Untuk membuat stelan tulisan dalam sebuah kotak, kamu dapat membuat "kotak teks" atau "kotak gambar" di dalam dokumen tersebut. Kemudian, kamu dapat mengatur teks di dalam kotak tersebut dengan memilih jenis huruf, ukuran huruf, warna teks, dan lainnya sesuai dengan preferensimu. Cara ini dapat bervariasi tergantung pada program pengolah kata yang kamu gunakan, namun umumnya ada opsi untuk mengedit teks di dalam objek kotak.
