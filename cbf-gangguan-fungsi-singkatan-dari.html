8. Pada gambar berikut DE//BC. Panjang AD :12cm dan DB: 8cm. a. tentukan besar perbandingan DE : BC! b. Buktikan segitiga DEF dan CBF sebangun! c. Tentukan besar perbandingan: (i) DF:FC (ii) EF:FB


Perhatikan gambar.
Diketahui:
DE sejajar BC, panjang AD = 12 cm, dan DB = 8 cm.

a. Berdasarkan prinsip kesebangunan pada segitiga ABC dan segitiga ADE, maka
DE : BC = AD : AB
↔ DE : BC = AD : (AD + DB)
↔ DE : BC = 12 : (12 + 8)
↔ DE : BC = 12 : 20 = 3 : 5
Jadi, DE : BC = 3 : 5

b. Perhatikan segitiga DEF dan segitiga CBF.
Sudut DEF = sudut CBF, karena merupakan pasangan sudut dalam berseberangan.
Sudut EDF = sudut BCF, karena merupakan pasangan sudut dalam berseberangan.
Sudut DFE = sudut CFB, karena merupakan pasangan sudut yang saling bertolak belakang.
Karena segitiga DEF dan segitiga CBF memiliki tiga pasang sudut bersesuaian yang sama besar, maka segitiga DEF sebangun dengan segitiga CBF.

c. Berdasarkan prinsip kesebangunan pada segitiga DEF dan segitiga CBF, maka
DF : FC = DE : BC
Dari poin a, kita tahu DE : BC = 3 : 5 sehingga DF : FC = 3 : 5
Demikian pula dengan EF : FB =  DE : BC = 3 : 5
Jadi, DF : FC = EF : FB = 3 : 5.

23. D E 60° B 40° F C G 6. Pada gambar diatas, garis AC dan DG sejajar, dan besar sudut CBF 40° dan sudut BEF= 60°. tentukan besar sudut ABE dan sudut BFG? ​


Jawaban:

Besar sudut ABE = 80°

Besar sudut BFG = 20°

Solusi:

1. Karena AC dan DG sejajar, maka sudut FCD = sudut DGB (sudut dalam pada garis sejajar)

2. Sudut CBF = 40°, maka sudut FCD = 180° - 40° - 60° = 80° (jumlah sudut pada segitiga FCB)

3. Karena sudut FCD = sudut DGB dan sudut FGB = 180° - sudut FCD (sudut luar pada segitiga GCD), maka sudut FGB = 100°

4. Sudut BFG = sudut FGB - sudut FBE = 100° - 80° = 20° (sudut dalam pada garis sejajar)

5. Sudut ABE = 180° - sudut FBE - sudut BFG = 180° - 60° - 20° = 100° (jumlah sudut pada segitiga EBF)
