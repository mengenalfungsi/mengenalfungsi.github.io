y = [tex]\sqrt{f(x)}[/tex] ⇒ Df : f(x) ≥ 0 y = [tex]\frac{f(x)}{g(x)}[/tex] ⇒ Df : g(x) ≠ 0 y = [tex]\frac{f(x)}{\sqrt{g(x)}}[/tex] ⇒ Df : g(x) > 0

Df = domain atau daerah asal fungsi

PembahasanMenentukan Daerah Hasil Fungsi (Pemetaan)

Yang dimaksud dengan fungsi atau pemetaan adalah suatu bentuk relasi khusus yang ditandai dengan :

Semua anggota himpunan asal mempunyai pasangan pada daerah himpunan hasil.Semua anggota himpunan asal masing-masing mempunyai tepat satu pasangan pada daerah himpunan hasil.

Suatu bentuk fungsi memiliki tiga buah unsur, yaitu :

Daerah asalNotasi fungsiDaerah hasil

Dan dalam kesempatan ini kita akan membahas soal yang berhubungan dengan menentukan daerah hasil fungsi.

Soal Nomor 1

Diketahui :

Notasi fungsi f(x) = 2x + 5

Daerah asal = (1, 3, 5, 7, 8)

Ditanya :

Daerah hasil dalam bentuk tabel, dimana daerah hasil tersebut merupakan himpunan bilangan cacah.

Penyelesaian :

f(1) = 2(1) + 5 ⇒ f(1) = 7 ⇒ x, y = 1, 7

f(3) = 2(3) + 5 ⇒ f(3) = 11 ⇒ x, y = 3, 11

f(5) = 2(5) + 5 ⇒ f(5) = 15 ⇒ x, y = 5, 15

f(7) = 2(7) + 5 ⇒ f(7) = 19 ⇒ x, y = 7, 19

f(8) = 2(8) + 5 ⇒ f(8) = 21 ⇒ x, y = 8, 21

Jawaban dalam bentuk tabelnya dapat dilihat di lampiran.

Soal Nomor 2

Diketahui :

Notasi fungsi f(x) = ax + b

f(2) = 13  

f(5) = 22

Ditanya :

Nilai a dan b

Penyelesaian :

f(2) = a(2) + b = 13

f(2) = 2a + b = 13

f(5) = a(5) + b = 22

f(5) = 5a + b = 22

Eliminasi b untuk menentukan nilai a.

(2a + b = 13) - (5a + b = 22)

2a - 5a + b  - b = 13 - 22

-3a = -9

a = -9/-3

a = 3

Menentukan nilai b dengan mensubstitusikan nilai a = 3 ke 2a + b = 13

2(3) + b = 13

6 + b = 13

Kedua ruas dikurangi dengan 6

6 + b - 6 = 13 - 6

b = 7

Jadi nilai a dan b berturut-turut adalah 3 dan 7.

Soal Nomor 3

Diketahui :

Notasi fungsi f(x) = 2x - 3

Daerah asal = (1, 2, 3, 4, 5)

Ditanya :

Himpunan daerah hasil.

Penyelesaian :

f(1) = 2(1) - 3 ⇒ f(1) = 2 - 3 ⇒ f(1) = -1

f(2) = 2(2) - 3 ⇒ f(2) = 4 - 3 ⇒ f(2) = 1

f(3) = 2(3) - 3 ⇒ f(3) = 6 - 3 ⇒ f(3) = 3

f(4) = 2(4) - 3 ⇒ f(4) = 8 - 3 ⇒ f(4) = 5

f(5) = 2(5) - 3 ⇒ f(5) = 10 - 3 ⇒ f(5) = 7

Jadi himpunan daerah hasil untuk f(x) = 2x - 3 dengan daerah asal {1, 2, 3, 4, 5) adalah {-1, 1, 3, 5, 7}.

Himpunan daerah hasil.

Penyelesaian :

Jadi himpunan daerah hasil untuk f(x) = 7x + 1 dengan daerah asal (bilangan ganjil kurang dari 10) adalah {8, 22, 36, 50, 64}.
