2. penyetaraan reaksi redoks dengan cara setengah reaksi


Penyetaraan reaksi dengan menggunakan cara setengah reaksi sebagai berikut :

Tentukan zat yang mengalami reduksi dan oksidasi.Setarakan Jumlah koefisien zat yang mengalami reduksi dan oskidasiMenambah H₂O diruas yang kekurangan O sebanyak kekurangannuya apabila dalam suasana asam, dan menambah H₂O diruas yang kelebihan O sebanyak kelebihannya apabila dalam suasana basa.Diruas yang kekurangan H menambah H⁺ jika suasana asam, dan menambah OH⁻ jika susasana basa sebanyak kekurangan dan kelebihnnya.Kemudian setarakan jumlah muatan ion dengan menambahkan elektron.selanjutnya kalikan setiap muatan agar memiliki nilai elektron yang sama.

Penyelesaian Soal :

Contoh soal :

1. H₂SO₄  +  HI  ⇒  H₂S  + I₂ + H₂O   (suasana asam)

Tentukan reduksi dan oksidasinya :

                         OKSIDASI

                      Ι-----------------Ι

 H₂SO₄  +   HI  ⇒   H₂S  + I₂ + H₂O

+1 +6 -2     +1 -1      +1 -2     0    +1 -2

      Ι-------------------------Ι

              REDUKSI

Setarakan reaksi dengan menggunakan cara setengah reaksi sebagai berikut :

Langkah 1 : Untuk mempermudah maka buatlah zat yang mengalami oksidasi dan reduksi sebagai berikut :

Oksidasi     :  I⁻   ⇒   I₂

Reduksi      : SO₄²⁻  ⇒   S²⁻

Langkah 2 : Setarakan jumlah koefisien  zat yang mengalami oksidasi dan reduksi

Oksidasi     :  2I⁻   ⇒   I₂

Reduksi      : SO₄²⁻  ⇒   S²⁻

Langkah 3 : Menambah H₂O diruas yang kekurangan O sebanyak kekurangannya.

Oksidasi     :  2I⁻   ⇒   I₂

Reduksi      : SO₄²⁻  ⇒   S²⁻  +  4H₂O

Langkah 4 : Menambah H⁺ diruas yang kekurangan H.

Oksidasi     :  2I⁻   ⇒   I₂

Reduksi      : SO₄²⁻ + 8H⁺  ⇒   S²⁻ +  4H₂O

Langkah 5 : Setarakan jumlah ion dengan menambah elektron.

Oksidasi     :  2I⁻   ⇒   I₂ + 2e                                      ║ × 4

Reduksi      : SO₄²⁻ + 8H⁺  + 8e ⇒   S²⁻ +  4H₂O       ║ × 1

Langkah 6 : Setarakan jumlah muatan pada kedua reaksi kemudian jumlah kedua reaksi.

Oksidasi     :  8I⁻   ⇒   4I₂ + 8e                                  

Reduksi      : SO₄²⁻ + 8H⁺  + 8e ⇒   S²⁻ +  4H₂O

                  ________________________________ +

                   8I⁻ + SO₄²⁻ + 8H⁺  ⇒  4I₂ +  S²⁻ +  4H₂O

Langkah 7 : Kemudian kembalikan reaksi tersebut menjadi reaksi molekul.

H₂SO₄  +  8HI  ⇒  4I₂  + H₂S  + 4H₂O

                     Reduksi

Langkah 1 : Untuk mempermudah maka buatlah zat yang mengalami oksidasi dan reduksi sebagai berikut :

Oksidasi     : Bi₂O₃   ⇒   BiO₃⁻

Reduksi      : ClO⁻  ⇒   Cl⁻

Langkah 2 : Setarakan jumlah koefisien  selain atom O dan H

Oksidasi     : Bi₂O₃   ⇒   2BiO₃⁻

Reduksi      : ClO⁻  ⇒   Cl⁻

Oksidasi     : Bi₂O₃  ⇒   2BiO₃⁻  + 3H₂O

Reduksi      : ClO⁻ + H₂O   ⇒   Cl⁻

Langkah 6 : Setarakan jumlah muatan pada kedua reaksi kemudian jumlah kedua reaksi.

Oksidasi     : Bi₂O₃ +  6OH⁻ ⇒   2BiO₃⁻  + 3H₂O  + 4e  

Reduksi      : 2ClO⁻ + 2H₂O + 4e  ⇒   2Cl⁻ + 4OH⁻  

                  ________________________________ +

                   Bi₂O₃ +  2OH⁻ + 2ClO⁻ ⇒   2Cl⁻ +  2BiO₃⁻  + H₂O
