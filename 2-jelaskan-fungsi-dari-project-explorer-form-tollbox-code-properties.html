27. 1.jelaskan fungsi search pada windows Explorer 2. jelaskan fungsi view pada windows explorer


1. Search
Digunakan untuk membuka kotak dialog file All files. Jadi pilihan ini sama dengan pilihan find dari Start Microsoft Windows XP. Begitu juga kegunaannya sama yaitu untuk mencari file yang dikehendaki.

2. View
Pilihan ini digunakan untuk mengkostumisasikan jendela1. Fungsi search:
untuk mencari file yang akan kamu gunakan. baik itu foto, dokumen, folder, dll. Kamu juga dapat mencari file yang bersifat executable seperti gimpsetup.exe. jika kamu menekan file tersebut maka akan muncul kotak instalansi.
2. View:
berguna untuk melihat file yang dipilih tanpa harus membukanya. cara ini hanya memerlukan klik 1 kali tanpa membuka program yang mendukung file tersebut seperti halo.docx akan terlihat di mode view jika Microsoft Word terinstal tetapi tidak perlu membuka Ms. Word untuk melihat filenya. Tidak dapat membaca file berekstensi .zip, .exe, dll.

33. jelaskan fungsi workspace dalam internet explorer


Fungsi workspace pada internet explore adalah untuk menampilkan halaman website yang kita akses. Pada saat kita mengakses sebuah halaman website, kita akan menuliskan alamat website kolom adress bar. Internet explore merupakan browser bawaan dari sistem operasi windows. Ketika kita menginstall komputer kita dengan sistem operasi windows, maka secara otomatis internet explore akan ikut terinstall. Sedangkan pada sistem operasi lain seperti linux, browser bawaannya biasanya adalah Mozilla Firefox dan Safari untuk sistem operasi Mac.

Pembahasan

Browser merupakan aplikasi yang digunakan untuk mengakses sebuah website. Selain internet explore beberapa jenis browser lainnya yang sering digunakan oleh pengguna komputer adalah

Mozilla FirefoxGoogle ChromeOpera BrowserSafariTor

Pengguna browser biasanya memilih browser yang mampu menampilan halaman website dengan cepat dan irit penggunaan memori. Sayangngya, internet explore kalah popular dengan browser-browser diatas.

34. jelaskan fungsi masing-masing menu pada internet Explorer​


Jawaban:

Penjelasan:Fungsi Menu Icon Pada Menu Icon Internet Explorer

Tampilan Internet Explorer terdiri atas:

Title Bar

Title bar Internet Explorer

Terletak di sebelah kiri atas pada jendela internet explorer yang merupakan judul dari halaman web yang sedang aktif. Titla bar menampilkan nama atau alamat situs yang sedang dibrowsing. Pada bagian pojok kanan atas terdapat ikon untuk minimize, maximize, restore, dan close.

minize, maximize, restore, close

Menu Bar

Menu Bar Internet Explorer

Menu bar merupakan tempat menu-menu yang terdapat dalam jendela internet explorer atau browser lainnya. Submenu tersebut mempunyai fungsi yang berbeda-beda antara satu dengan yang lain. Setiap menu pada menu bar berisi submenu yang merupakan perintah-perintah dalam internet explorer misalnya perintah untuk menyimpan, mengedit atau mengubah tampilan. Submenu ini akan ditampilkan jika kita memilih atau mengklik menu tersebut. Adapun menu yang terdapat dalam menu bar antara lain : File, Edit, View, Favorites, Tools, dan Help.

Address Bar

Address Bar Internet Explorer

Address bar digunakan untuk mengetikan alamat website yang akan dikunjungi. Address bar dilengkapi sebuah tombol Go yang fungsinya digunakan untuk membuka website yang alamatnya telah diketikan pada address bar.

Toolbars

Toolbars Internet Explorer

Jendela Internet Explorer juga dilengkapi dengan toolbar yang memuat tombol-tombol perintah. Tombol-tombol perintah tersebut merupakan shortcut perintah yang terdapat di menu. Masing-masing tombol menjalankan sebuah perintah. Toolbars terdiri atas ikon-ikon standar yang sering digunakan dalam akses internet. Adapun fungsi dari ikon toolbars adalah sebagai berikut :

Toolbar Internet Explorer dan Fungsinya

Isi Halaman

Isi halaman digunakan untuk tempat menampilkan halaman web dari suatu website yang sedang di akses

Scrolling Bar

Scrolling bar digunakan untuk menggeser atau melihat tampilan halaman web yang ada di atas di bawah yang tidak tampak pada layar monitor.

Scroll Bar vertical dan horizontal

Status Bar

Status Bar Internet Explorer

Statur bar digunakan untuk menampilkan proses pembukaan halaman website, apakah sudah terbuka seluruhnya atau belum. Jika sudah terbuka seluruhnya, maka akan ditampilkan kata Done. Status bar juga berfungsi untuk menampilkan alamat dari hyperlinks. Dan status bar terdapat pada bagian bawah jendela internet explorer yang berfungsi untuk menampilkan proses yang sedang dilakukan pada saat browser menampilkan sebuah halaman web.

Task Pane

Task pane adalah jendela kecil yang muncul di bagian kiri jendela browser apabila kita mengklik tombol search.
