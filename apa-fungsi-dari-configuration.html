1. fungsi dari configuration manager adalah


Configuration management (CM) adalah suatu proses untuk memastikan adanya dokumentasi yang akurat dan efisien untuk berbagai versi dari infrastruktur baik software maupun hardware dimana kedudukan configuration management di dalam service management adalah core dalam organisasi. Dengan kata lain configuration management adalah proses mencatat dan memelihara secara up to date dari semua komponen infrastruktur IT (aset IT, status aset, dan konfigurasinya) serta korelasinya satu dengan yang lain. Penekanan dari configuration management adalah pada hubungan antar component IT. Configuration management tidak mengacu pada aplikasi sistem perangkat lunak atau tingkat versi pengembangan perangkat lunak pada saat pengembangan atau testing. Kompatibilitas hardware dan software secara tepat adalah bagian dari proses manajemen konfigurasi. Menurut sebuah laporan yang dikeluarkan oleh Enterprise Management Associates (EMA), disiplin ilmu seperticonfiguration managementIT menjadi semakin penting untuk mempertahankan tingkat pelayanan dan menjaga semua perangkat keras dan perangkat lunak berfungsi maksimal.

25. Tuliskan urutan perintah untuk memasuki previleged exec mode configuration dalam switch cisco catalyst?


Jawaban:

Untuk memasuki privileged exec mode configuration dalam switch Cisco Catalyst, Anda perlu menggunakan urutan perintah yang tepat. Pertama-tama, Anda harus masuk ke mode user exec dengan perintah "enable." Kemudian, Anda perlu memasukkan kata sandi yang benar untuk mengkonfirmasi otoritas Anda. Setelah terverifikasi, Anda bisa masuk ke privileged exec mode dengan perintah "configure terminal." Dari sini, Anda dapat mengatur konfigurasi switch sesuai dengan kebutuhan Anda menggunakan perintah yang relevan. Penting untuk diingat bahwa hanya pengguna dengan otoritas terbatas yang diizinkan untuk mengakses privileged exec mode.

27. konfigurasi koneksi internet dengan modem ADSL menggunakana. static IP addressb. automatic configuration (DHCP)c. browser Mozillad. galeon​


Konfigurasi koneksi internet dengan model ADSL dapat dilakukan dengan menggunakan Static IP address atau Automatic configuration (DHCP).

Jika menggunakan Static IP address, pengguna harus memasukkan alamat IP yang diberikan oleh penyedia layanan internet (ISP) secara manual ke dalam konfigurasi koneksi internet. Sedangkan jika menggunakan Automatic configuration (DHCP), alamat IP akan didapatkan secara otomatis oleh modem dari server DHCP ISP.

Browser Mozilla dan Galeon adalah aplikasi web browser dan tidak terkait dengan konfigurasi koneksi internet ADSL. Namun, setelah koneksi internet telah dikonfigurasi dengan benar, pengguna dapat menggunakan browser Mozilla atau Galeon untuk mengakses internet.
