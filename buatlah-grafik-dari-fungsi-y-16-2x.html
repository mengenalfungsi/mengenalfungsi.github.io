PEMBAHASAN

Sebelumnya marilah kita pelajari dulu prinsip-prinsip faktorisasi.

Di ketahui persamaan kuadrat x² + 5x + 6 = 0 . Jika ingin di faktorkan bentuk persamaan ini , prinsipnya kita cara angka yang kalau dikalikan dengan angka lain mendapatkan nilai 6 , dan jika ditambahkan angka tersebut mendapatkan 5 sebagai berikut :

3 x 2 = 6

3 + 2 = 5

Jadi persamaan kuadrat x² + 5x + 6 = 0 dapat di faktorkan menjadi

x² + 5x + 6 = 0

( x + 3 ) ( x + 2 ) = 0

Okay dengan prinsip ini kita akan mencoba selesaikan soalnya.

Di berikan fungsi h = - 32t² + 32 , jika balon mencapai tanah itu artinya ketinggiannya akan bernilai 0 alias h = 0 , maka :

h = - 32t² + 32

0 = - 32 (t² -1)

0 = ( t - 1 ) ( t + 1 )

t - 1 = 0 atau t + 1 = 0

t = 1 detik atau t = -1 detik

Berhubung t = - 1 tidak memenuhi berarti balok akan mencapai tanah dalam waktu 1 detik. Tetapi karena balon udara jatuh bukan dari ketinggian maksimal tetapi dari ketinggian 19 kaki maka perlu di cari lagi waktu saat ketinggian tersebut.

Balon udara jatuh dari ketinggian 19 kaki , itu artinya h = 19 sehingga bisa di cari waktu nya sebagai berikut :

19 = - 32t² + 32

19 - 32 = - 32t²

-13 = -32t²

t² = 13/32

t = √(13/32)

  5 = a(1)² + b(1) + c

  5 = a + b + c

  -1 = a - b + c

  4 = c

  5 = a + b + c

  -1 = a - b + 4

  5 = a + b + 4

  a - b = -5

  a + b = 1

  2a = -4

  b = 3

  x² + 6x + 8 = 0

  x = -b / (2a)

ε = (%ΔQd)/(%ΔP)
