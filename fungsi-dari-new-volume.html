31. Jelaskan fungsi Recent new


Jawaban:

Fungsi Recent adalah untuk membuka file terakhir yang kita buka, biasanya sistem akan mendeteksi file file terbaru yang terakhir kita buka/ubah. Fungsi ini biasanya bertujuan untuk memudahkan penggunanya membuka file terakhir tanpa harus mencari file terbarunya.

Fungsi New adalah untuk membuat file baru, biasanya fungsi new menyediakan pilihan file baru yang memiliki template dan ada juga opsi blank file (file yang masih kosong).

note --> Jika pertanyaannya memang Fungsi Recent New, berarti bermaksud pada fungsi Recent
