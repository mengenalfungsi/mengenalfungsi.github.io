17. komponen yang berfungsi untuk memutar cd/dvd adalah


CD-RWDrive
  1.CD – RW Drive adalah perkembang dari CD – ROM Drive bila dalam CD Rom Drive hanya dapat membaca dan memutar CD, sedangkan CD RW Drive tidak hanya dapat memutar dan membaca data pada CD tapi juga dapat melakukan penulisan data (memasukan data) pada media CD, kelebihan lain dari CD RW Drive adalah dapat menghapus data pada media CD tapi tidak semua jenis CD dapat di hapus hanya jenis CD RW saja yang dapat di hapus.
2.DVD–RWDrive
    Tidak berbeda jauh dengan CD RW Drive yang membedakan adalah kemampuan DVD RW selain dapat membaca, menulis dan menghapus pada media CD, DVD RW dapat pula melakuan menghapus datadan penambahan pada media DVD. Dan daya tampung DVD lebih besar dari CD.
3.DVDCombo
    DVD Combo adalah gabungan dari CD RW dan DVD R sehingga kemampuan DVD Combo adalah disamping untuk membaca, menghapus dan menulis pada media CD dia juga dapat melakuakn pembacaaan pada media DVD. Tetapi DVD Combo tidak memiliki kemampuan menulis dan menghapus pada media DVD.

30. Harga 2 DVD dan 6 radio adalah 1.400.000, sama dengan harga 3 DVD dan 2 radio . hitunglah harga 4 DVD dan 3 radio.???!!!


Sistem Persamaan Linear Dua Variabel
SMP Kelas VIII

Misal
harga DVD = x
harga radio = y

2x + 6y = 1.400.000
x + 3y = 700.000
x = 700.000 - 3y ...(1)

3x + 2y = 1.400.000 ...(2)

Substitusi
3x + 2y = 1.400.000
3(700.000 - 3y) + 2y = 1.400.000
2.100.000 - 9y + 2y = 1.400.000
-9y + 2y = 1.400.000 - 2.100.000
-7y = -700.000
y = 100.000

x = 700.000 - 3y
x = 700.000 - 3(100.000)
x = 700.000 - 300.000
x = 400.000

maka
4x + 3y = 4(400.000) + 3(100.000)
             = 1.600.000 + 300.000
             = 1.900.000

36. Perbedaan konfigurasi debian 9 dan debian 8 itu apa ?


1. Kita harus install paket net-tools

Untuk bisa menjalankan perintah ifconfig dalam debian 9, kita harus menginstall net-tools terlebih dahulu. Saat ingin mengintall bind9 untuk membangun dns server, kita juga membutuhkan paket tersebut.


2. Cara Mengetahui Ip address Debian 9 

Setelah melakukan fresh install debian, umunya kita akan mengecek ip address dengan perintah ifconfig, namun dalam debian 9 belum terdapat paket tersebut, jadi kita menggunakan perintah ip a  atau ip addruntuk melihat ip address kita.


3. Perbedaan Nama Interface 

Pada debian 8, debian 7 dan versi sebelumnya, interface jaringan umumnya menggunakan nama eth0, eth1, dll. Namun, dalam debian 9, default interfacenya menggunakan nama enp0s3. Hal tersebut akan cukup membingungkan jika anda sudah terbiasa menggunakan debian 8.


4. Tidak ada paket Squirrelmail

Setelah saya mencoba untuk membangun webmail server, saya sempat dibingungkan karena tidak ada paket squirrelmail. Sebagai gantinya, kita bisa menggunakan Roundcube untuk membangun webmail server.


5. Paket courier-pop

Saat saya mencoba untuk membuat mail server, dan sebelumnya saya terbiasa menggunakan debian 8, saya juga sempat kebingungan karena tidak ada paket courier-pop, dan saya sempat untuk mengintall paket dovecot. Untuk itu anda harus mengintall courier-base dulu sebelum menginstall courier-pop.

38. Harga 2 DVD dan 6 radio adalah 1.400.000, sama dengan harga 3 DVD dan 2 radio . hitunglah harga 4 DVD dan 3 radio.???!!!


DVD = x
Radio = y

Substitusi :
a. 2x + 6y = 1.400.000
b. 3x + 2y = 1.400.000

a. 2x = 1.400.000 - 6y
     x  = (1.400.000 - 6y) ÷ 2
     x  = 700.000 - 3y

b. 3 × (700.000 - 3y) + 2y = 1.400.000
    2.100.000 - 9y + 2y = 1.400.000
    2.100.000 - 7y = 1.400.000
    2.100.000 - 1.400.000 = 7y
    7.000.000 = 7y
    7.000.000 ÷ 7 = y
    1.000.000 = y (Substitusikan ke salah satu persamaan di atas)

a. 2x + 6 × 1.000.000 = 1.400.000
    2x + 6.000.000 = 1.400.000
    2x = 1.400.000 - 6.000.000
    2x = 8.000.000
    x = 8.000.000 ÷ 2
    x = 4.000.000

4x + 3y =
4 × 4.000.000 + 3 × 1.000.000 
16.000.000 + 3.000.000 = 19.000.000
