11. Fungsi dalam microsoft exel untukbmenjumlahkan data yang ditentukan sebagai argumen disebut


Jawaban:

Fungsi SUM

Penjelasan:

Ada 3 rumus SUM dalam microsoft excel

=SUM(cell:cel) tekan enter

=SUM(cell;cell) tekan enter

=SUM(cell:cell;cell:cell)

Perbedaan titik dua ( : ) dengan titik koma ( ; ), kalau titik dua untuk menjumlahkan cell dalam satu kolom, lalu titik koma untuk menjumlahkan beberapa kolom. Atau bisa juga titik dua dan titik koma digabungkan untuk menjumlahkan satu kolom dengan kolom yang lain.

Contoh:

1. =SUM(a1:a10)

2. =SUM(a1;b1;c1;d1)

3. =SUM(a1:a10;b1:b10;c1:c10;d1:d10)
