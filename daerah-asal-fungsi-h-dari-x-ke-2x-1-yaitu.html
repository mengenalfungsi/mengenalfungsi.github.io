|5 + 2x| ≠ 0
5 + 2x ≠ 0

**Soal 1: Daerah Asal (Domain) dari (f o g o h)(x)**

Untuk mencari daerah asal dari fungsi komposisi (f o g o h)(x), kita perlu memperhatikan bahwa domainnya adalah himpunan semua nilai \(x\) yang memenuhi tiga fungsi ini: \(f(x)\), \(g(x)\), dan \(h(x)\).

a. Domain \(f(x)\) adalah seluruh bilangan real.

b. Domain \(g(x)\) adalah seluruh bilangan real.

c. Domain \(h(x)\) adalah seluruh bilangan real.

Karena ketiga fungsi ini memiliki domain yang sama (seluruh bilangan real), maka domain dari (f o g o h)(x) juga seluruh bilangan real.

**Soal 2: Domain dan Range dari (f+g)^-1 dan (f-g)^-1**

Untuk menentukan domain dan range dari (f+g)^-1 dan (f-g)^-1, kita perlu memperhatikan fungsi \(f(x)\) dan \(g(x)\) terlebih dahulu.

a. Fungsi \(f(x) = \frac{2+x}{x}\)

Domain \(f(x)\): Semua bilangan real kecuali \(x = 0\) (karena tidak dapat membagi dengan nol).

Range \(f(x)\): Semua bilangan real kecuali \(y = 1\) (karena tidak dapat mendekati \(y = 1\) pada \(x = 0\)).

b. Fungsi \(g(x) = \frac{2-x}{x}\)

Domain \(g(x)\): Semua bilangan real kecuali \(x = 0\) (karena tidak dapat membagi dengan nol).

Range \(g(x)\): Semua bilangan real kecuali \(y = -1\) (karena tidak dapat mendekati \(y = -1\) pada \(x = 0\)).

Sekarang, mari hitung domain dan range dari (f+g)^-1 dan (f-g)^-1.

a. (f+g)^-1: Ini adalah fungsi invers dari (f+g), jadi domain dan range akan sama dengan domain dan range dari (f+g).

Domain: Semua bilangan real kecuali \(x = 0\).

Range: Semua bilangan real kecuali \(y = 1\).

b. (f-g)^-1: Ini adalah fungsi invers dari (f-g), jadi domain dan range akan sama dengan domain dan range dari (f-g).

Domain: Semua bilangan real kecuali \(x = 0\).

Range: Semua bilangan real kecuali \(y = -1\).

**Soal 3: Domain dan Range dari f(x) dan g(x)**

a. Fungsi \(f(x) = \sqrt{1+x}\)

Domain \(f(x)\): Untuk akar kuadrat, nilai di bawah akar harus non-negatif (\(1+x \geq 0\)). Jadi, \(x \geq -1\).

Range \(f(x)\): Akar kuadrat selalu menghasilkan nilai non-negatif, jadi \(f(x) \geq 0\).

b. Fungsi \(g(x) = x - 3\)

Domain \(g(x)\): Semua bilangan real.

Range \(g(x)\): Semua bilangan real.
