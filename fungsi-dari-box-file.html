21. Fungsi-fungsi submenu file


1) New = Fungsi dari submenu new adalah untuk membuat dokumen baru.
2) Open = Fungsi dari submenu open dalam menu file ini adalah untuk membuka dokumen atau file yang sudah disimpan sebelumnya.
3) Save =  Fungsi dari submenu save dalam menu file ini adalah untuk menyimpan dokumen atau file.
4) Save As= Fungsi dari submenu save as dalam menu file ini adalah untuk menyimpan dokumen atau file dengan format file yang berbeda dari penyimpanan yang pertama.
5) Print = Fungsinya adalah untuk mencetak dokumen dengan printer.

8.Kotak teks atau Text Box adalah area persegi panjang di layar tempat Anda dapat memasukkan teks. Text Box adalah elemen antarmuka pengguna yang umum ditemukan dalam banyak jenis program perangkat lunak, seperti browser web, klien email, dan pengolah kata. Ketika Anda mengklik di text box, kursor yang berkedip ditampilkan, menunjukkan Anda dapat mulai mengetik. Jika Anda menggunakan tablet, mengetuk di dalam text box, dapat secara otomatis menampilkan keyboard di layarmengklik di text box, kursor yang berkedip ditampilkan, menunjukkan Anda dapat mulai mengetik. Jika Anda menggunakan tablet, mengetuk di dalam text box, dapat secara otomatis menampilkan keyboard di layar.

Ada dua jenis text box, – bidang teks dan area teks. Bidang teks adalah kotak kecil yang memungkinkan Anda memasukkan satu baris teks. Ini digunakan untuk memasukkan nilai dasar, seperti nama, angka, atau frasa pendek. Area teks adalah kotak yang lebih besar yang memungkinkan Anda memasukkan beberapa baris teks. Secara umum, menekan Enter di area teks akan memasukkan karakter baris baru, menciptakan jeda baris. Saat Anda menekan Enter saat mengetik dalam bidang teks, kursor akan melompat ke bidang berikutnya atau nilainya akan dikirimkan.
