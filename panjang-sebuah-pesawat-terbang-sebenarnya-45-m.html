Untuk menyelesaikan pertanyaan-pertanyaan tersebut, kita dapat menggunakan rumus dasar:

Kecepatan (v) = Jarak (s) / Waktu (t)

Jarak (s) = Kecepatan (v) × Waktu (t)

Waktu (t) = Jarak (s) / Kecepatan (v)

Berikut adalah penyelesaian dari masing-masing pertanyaan:

Pesawat terbang:

Jarak = 600 km

Waktu = 45 menit = 45/60 jam = 0.75 jam

Kecepatan = Jarak / Waktu = 600 km / 0.75 jam = 800 km/jam

Andi berlari:

Jarak = 7 km

Waktu = 30 menit = 30/60 jam = 0.5 jam

Kecepatan = Jarak / Waktu = 7 km / 0.5 jam = 14 km/jam

Bus dari kota A ke kota B:

Kecepatan = 45 km/jam

Waktu = 10:35 - 08:15 = 2 jam 20 menit = 2.33 jam

Jarak = Kecepatan × Waktu = 45 km/jam × 2.33 jam = 105 km

Panjang keliling lapangan:

Panjang keliling = 500 m

Kecepatan = 20 m/menit

Waktu = Panjang keliling / Kecepatan = 500 m / 20 m/menit = 25 menit

Dino berkendara:

Jarak = 4 km

Waktu = 120 menit = 120/60 jam = 2 jam

Kecepatan = Jarak / Waktu = 4 km / 2 jam = 2 km/jam

Ayda mengisi ember air:

Debit slang = 400 ml/detik

Waktu = 40 detik

Volume = Debit slang × Waktu = 400 ml/detik × 40 detik = 16,000 ml = 16 liter

Jawaban:

- Kecepatan rata-rata pesawat = jarak / waktu = 600 km / 0,75 jam = 800 km/jam

- Kecepatan rata-rata Andi = jarak / waktu = 7 km / 0,5 jam = 14 km/jam

- Waktu perjalanan bus = 10:35 - 08:15 = 2 jam 20 menit = 2,33 jam. Jarak = kecepatan x waktu = 45 km/jam x 2,33 jam = 104,85 km (dibulatkan menjadi 105 km).

- Keliling lapangan = 500 m, maka waktu yang dibutuhkan Danu untuk mengelilingi lapangan adalah = keliling / kecepatan = 500 m / 20 m/menit = 25 menit.

- Kecepatan Dino = jarak / waktu = 4 km / 2 jam = 2 km/jam.

- Debit air dalam satuan liter/detik = 400 ml/detik / 1000 = 0,4 L/detik. Jadi, volume air dalam ember = debit x waktu = 0,4 L/detik x 40 detik = 16 L.

b. menurut rumus diagonal ruang, AG = √(AB² + CG²) = √(12² + 8²) = √(144 + 64) = √208 = 14.4 cm

Tinggi tiang bendera = jarak kepala anak dengan tiang bendera + tinggi badan anak = 13 + 150 = 163 cm

Untuk mencari jarak antara lokasi kecelakaan dengan bandara, kita dapat menggunakan teorema pythagoras. Jika kita sebut jarak x sebagai jarak antara lokasi kecelakaan dengan bandara, maka x² = (25 km)² + (18 km)² = 625 + 324 = 949 km². Sehingga jarak x = √949 km = 30.77 km.
