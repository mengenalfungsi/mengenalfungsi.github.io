3. tuliskan langkah langkah menggambar grafik fungsi Y=3x² ! kemudian gambarlah grafik fungsi tersebut pada kertas berpetak! ​


Tuliskan langkah langkah menggambar grafik fungsi y = 3x²! kemudian gambarlah grafik fungsi tersebut pada kertas berpetak!

Bentuk umum fungsi kuadrat adalah f(x) = ax² + bx + c dengan a ≠ 0. Ada beberapa langkah dalam membuat grafik fungsi kuadrat. Yaitu

1. Menentukan bentuk kurva yaitu  

jika a > 0, maka kurvanya terbuka ke atas  jika a < 0 maka kurvanya terbuka ke bawah

2. Menentukan titik potong terhadap sumbu x dan sumbu y, yaitu

Titik potong terhadap sumbu x jika y = 0 Titik potong terhadap sumbu y jika x = 0

3. Menentukan koordinat titik puncak/titik balik yaitu (xp, yp)

xp = [tex]-\frac{b}{2a}[/tex] ⇒ biasanya disebut sumbu simetri yp = [tex]-\frac{D}{4a}[/tex], dengan D = b² – 4ac (D = diskriminan) atau yp = f(xp)

Pembahasan

y = 3x²

a = 3 b = 0 c = 0

Karena a > 0 yaitu a = 3, maka bentuk parabolanya terbuka ke atas Karena b dan c = 0, maka titik potong terhadap sumbu x, titik potong terhadap sumbu y dan titik puncaknya berada di titik yang sama yaitu (0, 0)

Selanjutnya kita coba substitusikan untuk x bilangan bulat, misalkan kita mulai dari x = –3 sampai x = 3 dengan membuat  tabel

x       –3      –2     –1     0     1     2     3

x²       9        4       1      0      1     4     9

3x²    27     12       3      0     3    12    27

Jadi berdasrkan tabel tersebut terdapat titiktitik yang dilalui grafik tersebut yaitu

(–3, 27), (–2, 12), (–1, 3), (0, 0), (1, 3), (2, 12), (3, 27)

Hubungkan titik-titik tersebut hingga membentuk parabola y = 3x²

8. 8. Diketahui fungsi f(x) = x² - 4x dengan daerah asal (domain) adalah -2 ≤ x ≤ 6, XER. a. Gambar grafik fungsi tersebut pada bidang koordinat (gunakan kertas berpetak) dengan menggunakan bantuan tabel fungsi! b. Tentukan pembuat nol fungsi! c. Tentukan persamaan sumbu simetri grafik fungsi tersebut! d. Tentukan koordinat titik balik minimum fungsi tersebut!​


Penjelasan dengan langkah-langkah:

a. Grafik fungsi f(x) = x² - 4x dapat digambarkan pada bidang koordinat sebagai berikut:

![grafik fungsi f(x) = x^2 - 4x](https://i.imgur.com/6Fh7Xys.png)

Untuk menggambar grafik tersebut, kita dapat menggunakan tabel fungsi berikut:

|x | f(x) |

|---|---|

|-2 | 12 |

|-1 | 5 |

|0 | 0 |

|1 | -3 |

|2 | -4 |

|3 | -3 |

|4 | 0 |

|5 | 5 |

|6 | 12 |

b. Untuk menentukan pembuat nol fungsi, kita harus mencari nilai x yang membuat f(x) = 0. Oleh karena itu, kita perlu menyelesaikan persamaan x² - 4x = 0. Faktorisasi x dari persamaan tersebut, maka didapatkan x(x - 4) = 0. Sehingga, pembuat nol fungsi adalah x = 0 dan x = 4.

c. Sumbu simetri grafik fungsi tersebut merupakan garis vertikal yang membagi grafik menjadi dua bagian yang simetris. Garis ini berada pada titik tengah antara kedua akar persamaan kuadratik yang merupakan pembuat nol fungsi. Oleh karena itu, sumbu simetri grafik fungsi f(x) = x² - 4x adalah x = (0 + 4) / 2 = 2.

d. Titik balik minimum dapat dicari dengan menggunakan persamaan lengkap kuadrat y = a(x - h)² + k, dimana (h,k) adalah koordinat titik balik minimum dan a menentukan arah pembukaan parabola. Kita dapat menyelesaikan persamaan lengkap kuadrat dengan mengisi nilai x, y, dan h yang sudah diketahui, yaitu:

y = x² - 4x

y = a(x - 2)² + k

x² - 4x = a(x² - 4x + 4) + k

x² - 4x = ax² - 4ax + 4a + k

x² - (4 + a)x + (4a + k) = 0

Persamaan tersebut merupakan persamaan kuadratik dalam x. Karena titik balik minimum berada di bawah sumbu x, maka a > 0. Kita dapat mencari nilai a, b, dan c pada persamaan tersebut:

a = 1

b = -(4 + a) = -5

c = 4a + k

Kita juga tahu bahwa koordinat titik balik minimum berada pada sumbu simetri, yaitu x = 2. Oleh karena itu, kita dapat menggunakan nilai x = 2 untuk mencari nilai k:

(2)² - (5)(2) + 4a + k = 0

4 - 10 + 4a + k = 0

4a + k = 6

k = 6 - 4a

Sehingga, persamaan lengkap kuadrat f(x) = x² - 4x dapat ditulis sebagai:

f(x) = (x - 2)² + 2

Dengan demikian, koordinat titik balik minimum adalah (2,2).

Gambar bisa disimak pada lampiran.

Penjelasan dengan langkah-langkah:

Diketahui:

Fungsi f dari bilangan rasional ke bilangan rasional dengan aturan f(x) = 3x + 2.

a. Tentukan pasangan atau peta X = -1, 0, 1.

Jawab:

f(x) = 3x + 2

f(-1) = 3(-1) + 2 = -3 + 2 = -1

f(0) = 3(0) + 2 = 0 + 2 = 2

f(1) = 3(1) + 2 = 3 + 2 = 5

f(a) = 5

3a + 2 = 5

3a = 5 - 2

3a = 3

Pembahasan

Fungsi kuadrat merupakan suatu fungsi yang memiliki bilangan berpangkat. Fungsi kuadrat memiliki hasil berupa lengkungan ke atas dan lengkungan ke bawah. Dalam ilmu matematika, lengkungan ke atas disebut dengan hiberbola dan lengkungan ke bawah disebut dengan parabola. Dalam menyelesaikan fungsi tersebut, kita harus memfaktorkan, mencari sumbu simetri, dan lain lain.

Bentuk Umum Fungsi Kuadrat

[tex]\boxed{\boxed{f(x)=ax^{2} +bx+c}}[/tex]

Rumus Sumbu Simeteri dan Titik Puncak

[tex]x_p=-\frac{b}{2a} \\\\y_p=-\frac{D}{4a} \\\\(x_p,y_p)=(-\frac{b}{2a} ,-\frac{D}{4a} )[/tex]

Pertanyaan

Gambarlah grafik fungsi kuadrat berikut ini pada kertas berpetak f(x) =2x² -8x -10​

Jawab

a. Mencari nilai a

f(x) =2x² -8x -10​, a = 2, b = -8, c = - 10, Karena nilai a positif (2), maka arah grafiknya terbuka ke atas.

b. Menentukan Titik Potong Dengan Sumbu X, y = 0

[tex]f(x)=2x^{2} -8x-10\\\\0=2x^{2} -8x-10\\\\\frac{0}{2} =\frac{2}{2} x^{2} -\frac{8}{2} x-\frac{10}{2} \;\;\;\;\;kedua\;ruas\;dibagi\;2\\\\0=x^{2} -4x-5\\\\0=(x-5)(x+1)\\\\x-5=0\;\;\;atau\;\;\;x+1=0\\\\x=5\;\;\;atau\;\;\;x=-1[/tex]

Jadi titik potong grafik pada sumbu x adalah (5,0) atau (- 1, 0)

c. Menentukan Titik Potong Dengan Sumbu Y, x = 0

[tex]f(x)=2x^{2} -8x-10\\\\f(0)=0^{2} -8(0)-10\\\\f(0)=-10[/tex]

Jadi titik potong grafik pada sumbu y adalah (0,-10)

d. Menentukan Titik Puncak

[tex](x_p,y_p)=(-\frac{b}{2a } ,-\frac{D}{4a} )\\\\(x_p,y_p)=(-\frac{b}{2a } ,-\frac{b^{2} -4ac}{4a} )\\\\(x_p,y_p)=[-\frac{-8}{2(2) } ,-\frac{(-8^{2}-4(2)(-10) }{4(2)} ]\\\\(x_p,y_p)=[\frac{8}{4} ,-\frac{(64+80)}{8} ]\\\\(x_p,y_p)=(2,-\frac{144}{8} )\\\\(x_p,y_p)=(2,-18 )[/tex]

Jadi titik puncaknya adalah (2,-18)

32. gambarkan grafik fungsi kuadrat y = x² + 5x + 6 dengan menggunakan langkah-langkah menggambar yang sudah dijelaskan.jangan lupa tentukan terlebih dahulu titik potong terhadap sumbu x dan y, sumbu simetri, kalau sketsalah grafiknya pada kertas berpetak​

○●○●○●○●○●○●○●○●○●○●○●○●○●○●○●○

PEMBAHASAN

Fungsi kuadrat merupakan suatu gambaran fungsi polinom yang tersusun atas peubah/variabel yang memiliki pangkat tertingginya adalah 2 (dua).

Bentuk umum fungsi kuadrat :

[tex]f(x) = {ax}^{2} + bx + c, a \neq 0[/tex]

Dengan f(x) = y yaitu variabel, x yaitu variabel bebas, sedangkan a dan b yaitu koefisien dan c yaitu suatu konstanta.

Pada fungsi kuadrat, suatu fungsi sangatlah erat hubungannya dengan grafik fungsi.

Grafik fungsi memiliki bentuk seperti parabola sehingga bisa digambarkan dengan menggunakan langkah-langkah tertentu.

Jenis-Jenis Fungsi Kuadrat

1. Apabila pada y = ax 2 + bx + c nilai b dan c adalah 0. Maka fungsi kuadratnya berupa :

[tex]y = sumbu \: 2[/tex]

Hal ini menyebabkan grafik pada fungsi ini menjadi simetris pada x = 0 serta memiliki nilai puncak di titik (0,0).

2. Apabila pada y = ax 2 + bx + c nilai b bernilai 0. Maka fungsi kuadrat akan berupa:

[tex]y = sumbu \: 2 + c[/tex]

Hal ini menyebabkan grafik pada fungsi ini menjadi simetris pada x = 0 serta memiliki titik puncak di (0,c)

3. Apabila titik puncak berada di titik (h,k). Maka fungsi kuadrat akan berbentuk:

[tex]y = a(x - h) \: 2 + k[/tex]

Penggunaan sifat diskriminan pada jenis-jenis akar-akar persamaan kuadrat

[tex]\boxed{ D \geqslant 0 \rightarrow akar \ nyata }[/tex][tex]\boxed{ D > 0 \rightarrow akar \ nyata \ dan \ berlainan/berbeda }[/tex][tex]\boxed{ D = 0 \rightarrow akar \ tunggal/kembar}[/tex][tex]\boxed{ D < 0 \rightarrow akar \ tak \ nyata }[/tex]

Note : Akar tidak nyata dikenal juga akar khayal atau imajiner.

Menggambar sketsa grafik fungsi kuadrat

Mencari titik potong grafik dengan sumbu koordinat.Mencari koordinat titik balik menggunakan [tex]\boxed{x_{p} = - \frac{b}{2a} }[/tex] dan [tex]\boxed {y_{p} = - \frac{b^{2} - 4 ac}{4a} }[/tex].Mencari letak titik bantu yang dilewati oleh grafik apabila diperlukan.

Sifat-sifat Grafik Fungsi Kuadrat

Apabila a > 0 maka kurva terbuka keatas.Apabila a < 0 maka kurva terbuka ke bawah.Apabila D > 0 maka kurva memotong sumbu X dua titik.Apabila D = 0 maka kurva menyinggung sumbu X.Apabila D < 0 maka kurva tidak memotong atau menyinggung sumbu X.
