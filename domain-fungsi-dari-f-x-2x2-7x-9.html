4. domain dari fungsi f(x) akar 9-x^2


Domain dari fungsi f(x) = √(9 - x²)

Pembahasan :

Daerah asal (domain) dari y = √(f(x)) adalah f(x) ≥ 0

Jadi

f(x) = √(9 - x²) terdefinisi jika

(9 - x²) ≥ 0
(3 + x)(3 - x) ≥ 0
x = -3 atau x = 3

Garis bilangan
.......... (-3) ....... (3) .......

uji titik
• untuk daerah sebelah kiri (-3) kita pilih x = -4
(3 + (-4)) (3 - (-4)) = (-1) (7) = -7 => negatif
• untuk daerah antara (-3) dan (3) kita pilih x = 0
(3 + 0) (3 - 0) = (3) (3) = 9 => positif
• untuk daerah sebelah kanan 3 kita pilih x = 4
(3 + 4) (3 - 4) = (7) (-1) = -7 ==> negatif

Jadi garis bilangannya
------ [-3] ++++ [3] -----
ambil daerah yang positif karena ≥ 0
-3 ≤ x ≤ 3

Sehingga domain dari fungsi f(x) = √(9 - x²) adalah
Df = {x | -3 ≤ x ≤ 3, x ∈ R}

17. Diketahui fungsi f: R→ R, dengan f(x)= 5x+2 /7x-10 Tentukan domain dari fungsi tersebut​


Jawaban Dan Langkah-Langkah:

Untuk menentukan domain dari fungsi f(x) = (5x+2)/(7x-10), ada dua hal yang perlu diperhatikan:

1. Pembilang (numerator) adalah 5x+2.

Tidak ada pembatas untuk nilai x yang dapat membuat pembilang tidak terdefinisi. Artinya, kita dapat menganggap pembilang ini terdefinisi untuk setiap nilai x dalam himpunan bilangan real (R).

2. Penyebut (denominator) adalah 7x-10.

Dalam kasus ini, kita perlu memperhatikan bahwa penyebut tidak boleh sama dengan nol, karena akan menghasilkan pembagian dengan nol yang tidak terdefinisi.

Untuk menentukan nilai-nilai x yang membuat penyebut sama dengan nol, kita perlu menyelesaikan persamaan 7x-10 = 0.

7x - 10 = 0

7x = 10

x = 10/7

Jadi, x=10/7 adalah satu-satunya nilai yang tidak termasuk dalam domain fungsi f(x) karena akan menyebabkan penyebutnya sama dengan nol.

Dalam hal ini, domain fungsi f(x) adalah himpunan semua nilai x di bilangan real (R) kecuali x=10/7. Jadi, domain dari fungsi tersebut adalah R \ {10/7}, yang berarti semua nilai x dalam himpunan bilangan real kecuali 10/7.

32. grafik fungsi kuadrat f(x)=2x2-7x-4 adalah


Kategori soal: MATEMATIKA - FUNGSI KUADRAT
Kelas: X
=========

Kalau kamu hanya sekedar membuat sketsa grafik fungsi kuadrat f(x) = 2x² - 7x - 4, maka kamu cukup menentukan bentuk parabola yang terbentuk (apakah membuka ke atas ataukah ke bawah) dan menentukan kedudukan grafik terhadap sumbu X (ditentukan berdasarkan nilai diskriminannya, apakah memotong sumbu X, menyinggung sumbu X, ataukah tidak memotong sumbu X).

Nah, untuk menggambar grafik fungsi kuadrat f(x) = 2x² - 7x - 4 dengan detail, kamu perlu menentukan titik potong grafik dengan sumbu X, titik potong grafik dengan sumbu Y, absis titik puncak grafik, dan nilai optimum (maksimum/minimum) grafik.


Cara pertama: menggambar sketsa grafik fungsi kuadrat
---------------------------------------------------------------------------

Oleh karena grafik fungsi kuadrat f(x) = ax² + bx + c membuka ke atas untuk a > 0, maka grafik fungsi f(x) = 2x² - 7x - 4 dengan a = 2 > 0 juga akan membuka ke atas.

Oleh karena f(x) = 2x² - 7x - 4, maka a = 2, b = -7, dan c = -4, sehingga nilai diskriminan grafik fungsi kuadrat f(x) = 2x² - 7x - 4 adalah

D = b² - 4ac
   = (-7)² - 4(2)(-4)
   = 49 + 32
   = 81

Nah, karena diskriminan grafik fungsi kuadrat f(x) = 2x² - 7x - 4 adalah D = 81 > 0, maka grafik fungsi kuadrat tersebut memotong sumbu X di dua titik yang berbeda.

Berdasarkan kedua informasi di atas, tentunya kamu sudah membuat sketsa grafik f(x) = 2x² - 7x - 4 bukan?


Cara kedua: menggambar grafik fungsi dengan lebih detail

LANGKAH PERTAMA: menentukan titik potong grafik fungsi kuadrat f(x) = 2x² - 7x - 4 dengan sumbu Y.

LANGKAH KEDUA: menentukan titik potong grafik fungsi kuadrat f(x) = 2x² - 7x - 4 dengan sumbu X.

                (2x + 1)(x - 4) = 0
                x = -1/2 atau x = 4

LANGKAH KEEMPAT: menentukan nilai optimum grafik fungsi kuadrat f(x) = 2x² - 7x - 4.
