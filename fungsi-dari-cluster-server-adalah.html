4. Apakah fungsi dari Proxy server


Fungsi Proxy Server

Proxy Server merupakan pihak ketiga yang menjadi perantara antara kedua pihak yang saling berhubungan, dalam hal ini adalah jaringan komputer lokal dan jaringan internet. Prinsipnya, pihak pertama dan pihak kedua tidak akan langsung berhubungan, namun masing-masing berhubungan dengan pihak ketiga yaitu si proxy.

Berikut ini tiga fungsi utama proxy

1. Connection sharing

Proxy bertindak sebagai gateway yang menjadi pembatas antara jaringan lokal dengan jaringan luar. Gateway bertindak juga sebagai sebuah titik dimana sejumlah koneksi dari pengguna lokal dan koneksi jaringan luar juga terhubung kepadanya. Oleh sebab itu, koneksi dari jaringan lokal ke internet akan menggunakan sambungan yang dimiliki oleh gateway secara bersama-sama (connecion sharing).

2. Filtering

Proxy bisa difungsikan untuk bekerja pada layar aplikasi dengan demikian maka dia bisa berfungsi sebagai firewalll paket filtering yang dapat digunakan untuk melindungi jaringan lokal terhadap gangguan maupun ancaman serangan dari jaringan luar. Fungsi filtering ini juga dapat diatur atau dikonfigurasi untuk menolak akses terhadap situs web tertentu dan pada waktu-waktu tertentu juga.

3. Caching

Sebuah proxy server mempunyai mekanisme penyimpanan obyek-obyek yang telah diminta dari server-server yang ada di internet. Dengan mekanisme caching ini maka akan menyimpan objek-objek yang merupakan berbagai permintaan/request dari para pengguna yang di peroleh dari iternet.

8. fungsi dari registrar server​


Registrar server adalah server yang bertanggung jawab untuk mendaftarkan domain names (nama domain) dalam sistem domain name registration.

Pembahasan:

Fungsi dari registrar server adalah sebagai berikut:

1. Pendaftaran Domain: Registrar server berfungsi sebagai entitas yang memungkinkan individu atau organisasi untuk mendaftarkan nama domain yang unik. Ketika seseorang ingin memiliki domain name, mereka dapat mengajukan permohonan kepada registrar server. Registrar server akan memproses permohonan tersebut dan melakukan pendaftaran domain jika nama tersebut tersedia.

2. Manajemen Domain: Setelah pendaftaran domain berhasil, registrar server juga bertanggung jawab untuk mengelola domain tersebut. Ini termasuk pembaruan informasi kontak, pembaruan name server, perpanjangan registrasi, dan pemindahan domain ke registrar server lain jika diperlukan.

3. Penyimpanan Informasi Whois: Registrar server harus menyimpan informasi Whois yang berhubungan dengan setiap domain yang didaftarkan. Informasi Whois meliputi informasi kontak pemilik domain, informasi registrar, tanggal registrasi, tanggal kadaluwarsa, dan informasi teknis lainnya. Informasi Whois ini dapat diakses secara publik oleh siapa saja yang ingin melihat informasi terkait domain.

4. Penentuan Nama Server: Registrar server juga berperan dalam menentukan name server yang bertanggung jawab untuk mengarahkan permintaan DNS (Domain Name System) terkait dengan domain yang didaftarkan. Name server yang ditentukan akan mengubah nama domain menjadi alamat IP yang terkait, memungkinkan website atau layanan yang terhubung dengan domain dapat diakses melalui internet.

5. Penanganan Perpanjangan Registrasi: Registrar server memantau tanggal kadaluwarsa registrasi domain dan bertanggung jawab untuk menginformasikan pemilik domain tentang batas waktu perpanjangan registrasi. Jika pemilik domain tidak memperbarui registrasi sebelum jatuh tempo, registrar server dapat mengakhiri registrasi dan membuat nama domain tersebut tersedia untuk orang lain.

An then...

Secara umum, registrar server berperan penting dalam pemrosesan pendaftaran domain dan pengelolaan domain names. Mereka memainkan peran kunci dalam menjembatani pemilik domain dengan sistem registry domain serta memastikan domain dapat berfungsi secara efektif di internet.

10. fungsi ion H+ dan ion O2- falam ac plasma cluster? ​


Teknologi plasmacluster pada piranti penjernih udara (Air Conditioner) dapat menghasilkan ion H+ dan ion O2-. Kedua ion ini dapat bereaksi deangan udara kotor, membunuh kuman, bakteri dan jamur yang ada dalam ruangan serta menjaga kelembaban kulit. Ion H+ dan ion O2- tersebut mula-mula menempel pada permukaan mikro organisme, dan membentuk OH radikal yang dengan segera menarik atom hidrogen pada organisme sehingga mikroorganisme tersebut menjadi non-aktif.

Pembahasan

Persamaan reaksi kimia merupakan pernyataan yang ditulis dengan rumus kimia yang menginformasikan identitas dan kuantitas zat-zat yang terlibat dalam suatu perubahan kimia ataupun fisika. Semua zat dalam reaksi kimia yang jumlahnya berkurang setelah reaksi, disebut reaktan, ditempatkan di sebelah kiri tanda panah yang mengarah ke kanan. Pada sebelah kanan tanda panah terdapat hasil reaksi (produk), yakni semua zat yang dihasilkan dari reaksi. Kuantitas produk bertambah seiring berjalannya reaksi.

Suatu reaksi kimia harus memiliki kesetaraan unsur-unsur pembentuk antara produk dan reaktan. Oleh karena itu penyetaraan reaksi dapat dilakukan dengan langkah-langkah sebagai berikut :

menambahkan koefisien reaksi yang dinyatakan dengan variabel (misalnya a, b, c, dan d) pada setiap zat baik produk maupun reaktan;menyusun persamaan matematis berdasarkan kesamaan jumlah atom unsur yang sama di ruas kiri maupun kanan, di mana jumlah atom = koefisien × indeks; danmenyelesaikan persamaan-persamaan matematis yang diperoleh dari langkah 2 dengan sebelumnya dengan menetapkan koefisien salah satu zat sama dengan 1, di mana zat yang dipilih biasanya adalah zat dengan rumus kimia paling kompleks.

Contoh :

a Zn + bHCl ⇒ cZnCl2 + dH2

Sehingga diperoleh persaman matematis berikut

atom Zn : a = c

atom H  : b = 2d

atom Cl  : b = 2c

jika c =1, maka diperoleh nilai a=1, b=2 dan d=1. Sehingga diperoleh reaksi setara berikut :

Zn (s) + 2HCl (aq)  ⇒  ZnCl2 (s) + H2 (g)
