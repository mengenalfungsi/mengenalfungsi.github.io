12. fungsi bagian control unit dari processor adalah


Control Unit (CU) adalah salah satu bagian dari CPU yang bertugas untuk memberikan arahan/kendali/ kontrol terhadap operasi yang dilakukan di bagian ALU (Arithmetic Logical Unit) di dalam CPU tersebut. Output dari CU ini akan mengatur aktivitas dari bagian lainnya dari perangkat CPU tersebut

14. bagian - bagian jendela exel dan fungsinya!


1)  OFFICE BUTTON : Tombol yang terdiri dari perintah-perintah standar seperti membuat dokumen baru, membuka dokumen, menyimpan dokumen, mencetak dokumen dan lain-lain.

2)  QUICK ACCESS TOOLBAR : Toolbar yang berisi sejumlah perintah customize quick access toolbar.

3)  CUSTOMIZE QUICK ACCESS TOOLBAR : Tombol untuk memodifikasi perintah-perintah Microsoft Excel 2007 pada quick access toolbar.

4)  TAB : Bagian berbentuk tabulasi yang berisi sejumlah perintah sesuai kelompoknya.

5)  TITLE BAR : Batang judul lembar kerja atau nama file yang terletak di jendela Microsoft Excel bagian atas.

6)  PENGATUR TAMPILAN WORKBOOK : Tombol-tombol perintah yang berfungsi untuk mengatur tampilan workbook,seperti page break preview, page layout, dan normal.

7)  ZOOM LEVEL : Untuk mengatur ukuran tampilan workbook. Tombol Plus (+) untuk memperbesar dan tombol Minus (-) untuk memperkecil tampilanworkbook.

8)  RANGE : Gabungan beberapa sel yang membentuk persegi empat. Keadaan ini biasanya digunakan untuk memblok / menandai data.

9)  SEL – POINTER : Sel yang ditandaipointer pada suatu alamat sel.

10)  TAB SHEET : Tab suatu worksheetyang aktif atau terbuka.

11)  INSERT TAB SHEET : Untuk menyisipkan worksheet baru.

12)  WORKSHEET : Halaman kerja berupa sekumpulan sel, baris dan kolom berwarna putih yang tampak pada saat Microsoft Excel diaktifkan.

13)  ALAMAT SEL : Letak atau posisi sel pada bagian kolom dan baris worksheet.

14)  GRUP : Bagian dari tab yang berisi tombol-tombol perintah untuk menjalakan aksi tertentu.

15)  TOMBOL DIALOG : Merupakan tombol dengan simbol panah bawah pada sudut kanan bawah grup yang berfungsi untuk menampilkan kotak dialog serta memodifikasi perintah-perintah yang disimpan di dalamnya.

16)  TOMBOL PENGATUR JENDELA : Tombol pengatur jendela terdiri dari :

Minimize : untuk meminimalkan jendela.
Maximize : untuk memaksimalkan jendela.
Close : untuk menutup jendela.

17)  TOMBOL HELP : Untuk menampilkan tutorial tentang bagaimana menggunakan Microsoft Excel 2007.

18)  BARIS PENGGULUNG : Bagian fasilitas untuk menggulung worksheet secara horizontal dan vertical.

23. Bagian bagian jendela aplikasi ms.word dan fungsinya


Jawaban:

Misalkan aplikasi Ms. Word 2010:

1. Title Bar: berfungsi untuk memberi keterangan tentang nama file yang aktif.

2. Ribbon: berfungsi untuk menampung tombol-tombol/ ikon-ikon tertentu sesuai kegunaannya.

3. Quick Access Toolbar: berfungsi untuk menampung ikon-ikon yang paling sering dipakai pengguna saat mengetik naskah di Ms. Word

4. Dialogue Box Launcher: berfungsi untuk mempercepat terbukanya kotak dialog tertentu yang ingin disetting.

5. Status Bar: berfungsi untuk menunjukkan status dokumen yang aktif.

6. Scroll Bar: berfungsi untuk menggulung halaman (secara vertikal/ horizontal)

7. Zoom Control: berfungsi untuk mengatur tampilan perbesaran (zoom) dan dapat diset sesuai presentase (semakin tinggi nilai % maka semakin besar tampilan dokumen dan sebaliknya)

8. View Buttons: berfungsi untuk melihat dokumen sesuai keinginan (print layout view, fullscreen view, dan lainnya)

9. Browse Buttons: berfungsi untuk menampilkan menu Browse (Find, Replace, Go To, dan lainnya)

10. Ruller Buttons: berfungsi untuk menampilkan dan menyembunyikan tampilan penggaris (ruler)

11. Tabs: berfungsi sebagai wadah bagi ribbon (berbeda ribbon berbeda nama tab).

32. 1 .bagian bagian office button atau tombol office beserta fungsinya 2.bagian-bagian control beserta beserta fungsinya 3. bagian-bagian tab menu beserta fungsinya 4.bagian-bagian quick access berserta fungsinya


A.New berfungsi untuk membuat sebuah dokumen baru atau menggunakan template.

Open berfungsi untuk membuka lembar dokumen yang telah disimpan (save) sebelumnya atau membuka berbagai jenis dokumen teks yang didukung oleh Word 2007.

Publish berfungsi untuk mempublikasikan dokumen yang sedang terbuka ke dalam berbagai media, seperti Blog di internet dan Document Management Server serta Document Workspace di jaringan komputer lokal.

Close berfungsi untuk menutup dokumen yang sedang dibuka di jendela Word 2007.

34. Jelaskan fungsi bagian bagian jendela microsoft word 2007!


Jawaban:

-Office button, digunakan dan berfungsi untuk membuka menu Office

-Quick access toolbar, digunakan untuk mengakses tombol-tombol toolbar dengan cepat

-Baris judul, digunakan untuk mengetahui judul file atau dokumen yang sedang terbuka

-Tombol ukuran jendela, digunakan dan berfungsi untuk memperbesar dan memperkecil ukuran tampilan jendela.

-Ribbon, berisi tentang tab-tab menu perintah yang ada pada microsoft word 2007

-Baris status, berguna untuk menunjukkan dan memberikan informasi kepada pemakai tentang halaman yang ditampilkan, jumlah kata dan bahasa yang digunakan.

-Halaman kerja, digunakan sebagai halaman kerja atau tempat mengetik dan mengedit dokumen yang sedang ditampilkan atau dibuat.

-Pengatur tampilan halaman kerja, berguna untuk mengatur tampilan halaman kerja pada microsoft word 2007.

-Zoom, digunakan untuk mengatur perbesaran dan pengecilan tampilan halaman kerja.

-View ruler, digunakan untuk menampilkan atau menghilangkan garis penggaris baik secara horisontal maupun vertikal.

-Baris penggulung, digunakan untuk menggulung halaman kerja yang sedang ditampilkan secara horisontal dan vertikal.

-Tombol dialog, digunakan untuk membuka kotak dia- log serta memodifikasi perintah-perintah yang ada di dalamnya.

JENDELA : Tombol pengatur jendela terdiri dari :

Minimize : untuk meminimalkan jendela.

Maximize : untuk memaksimalkan jendela.

Close : untuk menutup jendela.
