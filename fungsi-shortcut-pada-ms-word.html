28. tombol kunci insert (shortcut) ms word


Jawaban:

Tombol kunci shortcut untuk mengaktifkan mode "Overtype" atau "Insert" dalam Microsoft Word adalah tombol "Ins" (Insert) pada keyboard. Jika tombol Ins ditekan, mode saat ini akan berganti antara "Insert" (menulis normal) dan "Overtype" (menulis menggantikan teks yang ada).

Namun, harap dicatat bahwa beberapa laptop atau keyboard tertentu mungkin tidak memiliki tombol Ins secara khusus. Dalam hal ini, Anda dapat menggunakan pengaturan alternatif yang disediakan oleh perangkat lunak pengolah kata seperti Microsoft Word. Untuk mengaktifkan atau menonaktifkan mode "Overtype" melalui tombol kunci keyboard yang berbeda, Anda dapat mengikuti langkah-langkah berikut:

1. Buka Microsoft Word.

2. Klik tab "File" di pojok kiri atas layar.

3. Pilih "Options" (Opsi) di bagian bawah menu File.

4. Dalam jendela Opsi Word, pilih "Advanced" (Lanjutan) dari panel sebelah kiri.

5. Gulir ke bawah hingga Anda menemukan bagian "Editing options" (Opsi Penyuntingan).

6. Centang kotak "Use the Insert key to control overtype mode" (Gunakan tombol Insert untuk mengontrol mode Overtype).

7. Klik "OK" untuk menerapkan perubahan.

Setelah ini, Anda dapat menggunakan fungsi tombol Insert di keyboard apa pun yang tersedia untuk mengaktifkan atau menonaktifkan mode "Overtype" dalam Microsoft Word.

30. carilah pengertian minimal 10 shortcut yang ada pada MS Word office


1.    Ctrl + A = (All) : fungsinya untuk memilih seluruh objek baik gambar atau teks pada lembar kerja Ms Word untuk di Blok semuanya.
2.    Ctrl + B = (Bold) fungsinya membuat kalimat yang diblok atau terseleksi menjadi tercetak tebal.
3.    Ctrl + C = (Copy) fungsinya untuk mengcopy sebuah kata atau kalimat yang juga berfungsi menggandakan teks atau gambar yang diseleksi
4.    Ctrl + D = (Font) fungsinya menampilkan Form ukuran Font untuk mengganti dan mengatur bentuk dan jenis font/ huruf
5.    Ctrl + E = (Center) fungsinya membuat kalimat yang diblok menjadi rata ketenga pada suatu paragraf
6.    Ctrl + F = (Find) fungsinya menampilkan Form untuk kolom pencarian yang berfungsi untuk mencari sebuah kata atau frase
7.    Ctrl + G = (Go To) fungsinya menampilkan Form Find and Replace go to untuk menuju halaman yang diinginkan
8.    Ctrl + H = (Replace) fungsinya menampilkan Form Replace yaitu untuk mengganti kata tertentu
9.    Ctrl + I = (italic) fungsinya membuat hurup menjadi tercetak miring
10. Ctrl + J = (Justify) fungsinya untuk membuat rata kiri dan kanan pada paragraf

36. Sebutkan Shortcut dalam ms. Excel 2016 beserta Fungsinya


Jawaban:

Beberapa shortcut dalam Ms. Excel 2016 antara lain:

1. Ctrl + 1

2. Ctrl + 2

3. Ctrl + 3

4. Ctrl + 4

5. Ctrl + 5

6. F1

7. F2

Penjelasan:

1. Fungsi shortcut Ctrl + 1 yaitu menampilkan kotak dialog Format Cells

2. Fungsi shortcut Ctrl + 2 yaitu menebalkan teks dalam sel yg aktif

3. Fungsi shortcut Ctrl + 3 yaitu memiringkan teks dalam sel yg aktif

4. Fungsi shortcut Ctrl + 4 yaitu menggarisbawahi teks di sel yg aktif (underline)

5. Fungsi shortcut Ctrl + 5 yaitu menggaris teks dalam sel yg aktif (strikethrough)

6. Fungsi shortcut F1 yaitu menampilkan kotak dialog Help

7. Fungsi shortcut F2 yaitu mengedit isi sel yg aktif
