7. jelaskan tentang fungsi penjadwalan yang berbeda-beda tergantung pada jenis operasi perusahaan​


Fungsi penjadwalan berbeda pada tiap kebutuhan prosesnya harus saling berkaitan dengan fungsi lainnya. Secara umum penjadwalan berfungsi untuk merencanakan penggunaan sumber daya secara efisien dan efektif, bagaimana suatu proyek akan dilaksanakan , membantu kelancaran kegiatan agar lebih teroganisir dan memberikan hasil yang maksimal.

       

Pembahasan :        

Dalam bidang operasional terdapat 2 jenis penjadwalan antara lain :      

a) Penjadwalan jangka pendek, berkaitan dengan bagaimana memenuhi permintaan jangka pendek atau permintaan pasar yang disusun untuk pekerjaan berulang atau berbasis harian. Disebut juga sebagai penjadwalan operasi.          

b) Penjadwalan jangka panjang, berkaitan dengan pelaksanaan kegiatan yang waktu pengerjaannya meliputi bulanan sampai tahunan atau biasa disebut proyek. Tujuan utama penjadwalan ini adalah memberikan keseimbangan proses-prosees campuran, biasa disebut sebagai analisis jaringan kerja.

Menjamin setiap file crontab milik user dieksekusi sesuai jadwal adalah fungsi dari perintah?

________

Jawaban: Cron daemon atau cron service.

• • • • • • • • • • • • • • • • • • • • • • • • • • • • • •

PEMBAHASAN

Cron daemon atau cron service adalah program yang berjalan pada sistem operasi Unix atau Linux yang bertanggung jawab untuk menjalankan pekerjaan yang dijadwalkan dalam file crontab. Setiap pengguna dapat membuat file crontab untuk menjadwalkan tugas-tugas yang harus dilakukan secara otomatis pada waktu tertentu. Cron daemon memeriksa file crontab untuk setiap pengguna secara teratur dan menjalankan tugas-tugas yang dijadwalkan pada waktu yang ditentukan dalam file crontab. Dengan menggunakan cron daemon, pengguna dapat memastikan bahwa tugas-tugas yang dijadwalkan akan dijalankan pada waktu yang diinginkan dengan akurasi dan keandalan yang tinggi.

Jadi, fungsi dari perintah crontab yang menjamin setiap file crontab milik user dieksekusi sesuai jadwal adalah crontab -e. Perintah ini digunakan untuk mengedit file crontab milik user saat ini.
