17. contoh gerakan lokomotor antara lain adalah ..soal kelas 5 sd ​


Contoh gerakan lokomotor antara lain adalah :

BerjalanBerlariMelompatMeloncatMemanjat

Pembahasan

         Gerakan lokomotor adalah gerakan yang dilakukan oleh tubuh sehingga membuat tubuh berpindah tempat. Seperti yang terlah disebutkan diatas, gerakan-gerakan diatas adalah gerakan yang membuat tubuh bergerak berpindah tempat. Gerak lokomotor termasuk ke dalam salah satu gerakan tubuh manusia. Gerakan dasar tubuh manusia dibagi menjadi 3 macam, yaitu gerakan lokomotor, gerakan nonlokomotor, dan gerakan manipulatif.

         Berikut pembahasannya untuk masing-masing gerakan :

Gerakan Lokomotor

Gerakan nonlokomotor ialah, aktivitas yang dilakukan tubuh sehingga tubuh bergerak dan berpindah tempat. Contoh gerakan lokomotor antara lain :

BerjalanBerlariMelompatMeloncatMemanjat

     2. Gerakan Nonlokomotor

Gerakan nonlokomotor adalah gerakan yang dilakukan tubuh sehingga tubuh bergerak namun tidak berpindah tempat/tetap di tempat. Contoh gerakan nonlokomotor antara di antaranya :

JongkokMembungkukMenolehMengayun

     3. Gerakan Manipulatif

Gerakan manipulatif yaitu, gerakan yang dilakukan oleh manusia dengan melibatkan/menggunakan alat tertentu. Contohnya :

MelemparMenendangMenerima/Menangkap

jawab :

1⅜ + 2⅝ = 1 + 2 + (3 + 5)/8

= 3 + 8/8

jawab :

¾ ÷ 6/12

= ¾ x 12/6

= 36/24

= 3/2

3. sebuah balok panjangnya 12 cm lebarnya 5 cm dan tingginya 8 cm.

volume balok = .... cm³

jawab :

p = 12 cm

l = 5 cm

t = 8 cm

V = p x l x t

V = 12 x 5 x 8

V = 12 x 40

V = 480 cm³

31. contoh soal sifat asosiatif dan distributif kelas 5 sd mapel matematika


Kelas : V (5 SD)
Materi : bilangan bulat
Kata Kunci : bilangan, bulat, sifat-sifat, asosiatif, distributif

Pembahasan :
1. Penjumlahan
Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

(a + b) + c = a + (b + c)

contoh :

(2 + 6) + 8 = 2 + (6 + 8)

⇔ 8 + 8 = 2 + 14

⇔ 16 = 16


2. Pengurangan

tidak bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c tidak berlaku:

(a – b) – c ≠ a – (b – c).

contoh :

(3 - 5) - 7 ≠ 3 - (5 - 7)

⇔ -2 - 7 ≠ 3 - (-2)

⇔ -9 ≠ 5


3. Perkalian

Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c, berlaku :

(a x b) x c = a x (b x c)

contoh :

(2 x 5) x 6 = 2 x (5 x 6)

⇔ 10 x 6 = 2 x 30

⇔ 60 = 60.


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

a x (b + c) = (a x b) + (a x c)

contoh : 

–5 x [9 + (–1)] = [–5 x 9] + [–5 x (–1)] 

⇔ –5 x 8 = –45 + 5

⇔ –40 = –40


ii. distributif terhadap pengurangan

a x (b – c) = (a x b) – (a x c)

contoh :

7 x [–10 – 12] = [7 x (–10)] – [7 x 12]

⇔ 7 x -22 = -70 - 84

⇔ -154 = -154.


4. Pembagian

Tidak bersifat asosiatif, sehingga Untuk sembarang bilangan bulat a, b, dan c berlaku:

(a : b) : c ≠ a : (b : c)

contoh :

(8 : 4) : 2 ≠ 8 : (4 : 2)

⇔ 2 : 2 ≠ 8 : 2

⇔ 1 ≠ 4


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

(a + b) : c = (a : c) + (b : c)

contoh : 

(12 + 6) : 2 = (12 : 2) + (6 : 2)

⇔ 18 : 2 = 6 + 3

⇔ 9 = 9


ii. distributif terhadap pengurangan

(a – b) : c = (a : c) – (b : c)

contoh :

(21 – 9) : 3 = (21 : 3) – (9 : 3)

⇔ 12 : 3 = 7 - 3

⇔ 4 = 4

 

Semangat!
