- Ini menghasilkan persamaan garis: \(k(x) = -2x + 16\).

- Ketika harga (\(x\)) adalah \(10\), jumlah yang dibeli (\(y\)) adalah \(2\).

- Ketika harga (\(x\)) adalah \(5\), jumlah yang dibeli (\(y\)) adalah \(6\).

- Ini menghasilkan persamaan garis: \(r(x) = -4x + 42\).

4. Fungsi \(s\) (kita sebut sebagai \(s(x)\)):

5. Fungsi \(t\) (kita sebut sebagai \(t(x)\)):

- Ketika harga (\(x\)) adalah \(6\), jumlah yang dibeli (\(y\)) adalah \(2\).

34. Kecepatan Maglev. Dalam suatu uji kecepatan sebuah kereta maglev sepanjang suatu lintasan lurus, data yang diperoleh dari pembacaan speedometer menunjukkan bahwa kecepatan maglev pada saat t dapat dirincikan oleh fungsi kecepatan berikut. v(t) = 8t (0 ≤ 1 ≤30) Tentukan fungsi posisi maglev. Anggap bahwa maglev mula-mula bergerak dari garis koordinat asal.


Untuk menentukan fungsi posisi maglev, kita perlu mengintegrasikan fungsi kecepatan terhadap waktu. Karena v(t) = 8t, maka:

s(t) = ∫v(t)dt = ∫8tdt = 4t^2 + C

di mana C adalah konstanta integrasi. Untuk menentukan nilai C, kita perlu menggunakan informasi bahwa maglev mula-mula bergerak dari garis koordinat asal. Oleh karena itu, ketika t = 0, s = 0, sehingga:

s(0) = 4(0)^2 + C = 0

Dari sini, kita dapatkan C = 0, sehingga:

s(t) = 4t^2

Jadi, fungsi posisi maglev adalah s(t) = 4t^2, di mana s diukur dalam satuan jarak (misalnya meter) dan t diukur dalam satuan waktu (misalnya detik).

Kecepatan

v(t) = 8t

Posisi

[tex]\sf s(t)=\int{v(t)} \, dt\\\\=\int{8t} \, dt\\\\=\frac{8}{2} t^{2}+C \\\\=4t^{2} +C[/tex]

Maglev mula-mula bergerak dari garis koordinat asal, ketika waktu (t) masih 0 dan posisi (s) juga masih 0.

s(0) = 4(0)² + C

0 =  4(0)² + C

C = 0

Maka

s(t) = 4(t)² + 0

s(t) = 4(t)²

Jadi, fungsi posisi maglev adalah s(t) = 4(t)².
