10. Fungsi yang di gunakan untuk menebalkan tulisan Adalah.


Jawaban:

bold atau font-weight

Penjelasan:

Fungsi yang biasa digunakan untuk menebalkan tulisan adalah fungsi bold atau fungsi font-weight. Fungsi bold dapat digunakan untuk menebalkan tulisan dengan menambahkan tag <b> sebelum tulisan yang ingin ditebalkan dan tag </b> setelah tulisan tersebut.

Sebagai contoh, jika kita ingin menebalkan tulisan "tebalkan tulisan ini", kita dapat menuliskannya sebagai berikut:

<b>tebalkan tulisan ini</b>

Sedangkan fungsi font-weight dapat digunakan untuk menebalkan tulisan dengan menambahkan atribut font-weight pada tag yang digunakan untuk menuliskan tulisan tersebut. Nilai dari atribut font-weight dapat diatur sesuai keinginan, misalnya 700 untuk menebalkan tulisan secara maksimal.

Sebagai contoh, jika kita ingin menebalkan tulisan "tebalkan tulisan ini" dengan menggunakan fungsi font-weight, kita dapat menuliskannya sebagai berikut:

<p style="font-weight:700">tebalkan tulisan ini</p>

21. Tag yang digunakan untuk menebalkan tulisan adalah…


Tag yang digunakan untuk menebalkan tulisan adalah tag <b> (bold) dan tag <strong>. HTML merupakan bahasa markah yang digunakan untuk mendefinisikan struktur atau bagian-bagian dari halaman web dan kontennya sehingga menjadi tampilan visual atau dengan kata lain bahasa markah digunakan untuk menampilkan.

Pembahasan

HTML merupakan bahasa markah yang digunakan untuk mendefinisikan struktur atau bagian-bagian dari halaman web dan kontennya sehingga menjadi tampilan visual atau dengan kata lain bahasa markah digunakan untuk menampilkan. Fungsi HTML adalah sebagai berikut :

sebagai struktur halaman web yang memberikan aturan tentang bagaimana halaman dan konten web tersebut ditampilkanmengatur format teksmembuat listmenampilkan multimediamembuat tabel dan form.

Pada bagian format teks untuk memunculkan atau menampilkan teks bertulisan tebal pada HTML menggunakan tag  <b> (bold) dan tag <strong>. Perbedaan antara kedua tag tersebut adalah tag <b> dari kata bold digunakan untuk membuat huruf tebal saja sedangkan tag <strong> digunakan untuk menegaskan bahwa teks atau huruf tersebut lebih penting dari yang lainnya. Teks tebal ini biasanya digunakan untuk judul, subjudul, huruf penting, dll.

38. Ikon Center digunakan untuk Bagaimana cara mengubah jenis huruf Bagaimana cara mengubah ukuran huruf Menjadi 36. Bagaimana cara menebalkan huruf​


Untuk mengubah jenis huruf, format font dapat diubah melalui menu Format atau dengan menekan tombol Ctrl + Shift + F.

Untuk mengubah ukuran huruf menjadi 36, ukuran font dapat diubah melalui menu Format atau dengan menekan tombol Ctrl + Shift + P dan memilih ukuran 36.

Untuk menebalkan huruf, format tebal dapat diaktifkan melalui menu Format atau dengan menekan tombol Ctrl + B.
