13. Diketahui f(x)= x-5, domain dari fungsi f tersebut adalah…


Domain dari fungsi f(x) = x - 5 adalah himpunan semua nilai x yang dapat diterima oleh fungsi tersebut. Dalam hal ini, tidak ada pembatasan pada variabel x, yang berarti domain fungsi f(x) adalah seluruh himpunan bilangan real, yaitu (-∞, ∞). Dengan kata lain, fungsi f(x) dapat menerima input berapapun bilangan real sebagai nilai x.

Misalkan f(x) = x - 5. Domain fungsi f adalah seluruh bilangan real kecuali nilai tertentu yang menyebabkan fungsi f tak terdefinisi, entah karena pembagian dengan nol atau akar pangkat genap dari bilangan negatif. Karena tidak ada nilai x yang menyebabkan hal ini, domain fungsi f adalah seluruh bilangan real pada R = (-∞, ∞).

19. diketahui f(x)=√1-x dan g(x)=x². tentukan (a) domain dan range fungsi f(x). (b) domain dan range fungsi g. (c) domain FoG​


Jika diketahui fungsi  [tex]f(x) = \sqrt{1- x}[/tex] dan g(x) = x² , maka

a. domain dan range dari fungsi [tex]f(x) = \sqrt{1 - x}[/tex] adalah Df = {x| x ≤ 1, x ∈ R} dan Rf = {y| y ≥ 0, y ∈ R}.

b. domain dan range dari fungsi g(x) = x² adalah Dg = {x| x ∈ R} dan Rg = {y| y ≥ 0, y ∈ R }.

c. domain fungsi [tex]fog(x) = \sqrt{1 - x^2}[/tex] adalah [tex]D_{fog(x)}[/tex] = {x| -1 ≤ x ≤ 1, x ∈ R}.

Pembahasan

Suatu fungsi dapat dinyatakan dalam bentuk: f : x ---> f(x).

Nilai fungsi untuk setiap nilai x yang diberikan dapat dihitung dengan cara mensubstitusikan nilai x ke dalam rumus fungsi f(x).

Domain (daerah asal) fungsi adalah himpunan nilai-nilai agar suatu fungsi terdefinisi.

Daerah hasil fungsi (range) adalah himpunan nilai-nilai fungsi dari setiap anggota daerah asal (domain).

Fungsi komposisi adalah suatu penggabungan fungsi dari operasi dua jenis fungsi yaitu f(x) dan g(x) sehingga menghasilkan fungsi baru.

Operasi fungsi komposisi dinotasikan dengan "o" dibaca sebagai bundaran. Fungsi baru yang terbentuk dari fungsi f(x) dan fungsi g(x) sebagai berikut.

(fog(x)) = f(g(x)) ---> maksudnya adalah nilai x pada fungsi f(x) diganti dengan fungsi g(x) (gof(x)) = g(f(x)) ---> maksudnya adalah nilai x pada fungsi g(x) diganti dengan fungsi f(x)

Penyelesaian

diket:

[tex]f(x) = \sqrt{1 - x}[/tex]

g(x) = x²

ditanya:

a. domain dan range fungsi f(x)....?

b. domain dan range fungsi g(x)....?

c. domain fungsi fog(x)...?

jawab:

a. - mencari domain fungsi f(x)

Karena fungsi [tex]f(x) = \sqrt{1 - x}[/tex] berbentuk [tex]\sqrt{a}[/tex] , maka agar fungsi tersebut dapat terdefinisi adalah dengan syarat a ≥ 0, sehingga

1 - x ≥ 0

1 ≥ x

x ≤ 1

- mencari range fungsi f(x)

 Karena domain fungsi f(x) adalah x ≤ 1, maka range dapat ditentukan dengan memasukkan nilai x ≤ 1, yaitu 1, 0, -1, -2, -3 .... dan seterusnya.

misal

 x = 1, maka  [tex]f(1) = \sqrt{1 - 1} = 0[/tex]

 x = 0, maka  [tex]f(0) = \sqrt{1-0} = 1[/tex]

 dan seterusnya

 Karena nilai range paling kecil adalah 0, maka range dapat dituliskan sebagai berikut.

Rf = {x| x ≥ 0, x ∈ R}

b. - mencari domain fungsi g(x)

     Karena fungsi g(x) berbentuk fungsi kuadrat yaitu g(x) = x², maka domain untuk fungsi g(x) adalah semua anggota real.

  - mencari range fungsi g(x)

    Karena domain fungsi g(x) semua anggota real, maka daerah hasil (range) fungsi g(x) juga semua anggota real.

c. - mencari fungsi fog(x) lebih dulu

     fog(x) = f(g(x))

               = [tex]\sqrt{1 - x^2}[/tex]

   

    - mencari domain fungsi fog(x)

      Karena fungsi [tex]fog(x) = \sqrt{1 - x^2}[/tex] berbentuk [tex]\sqrt{a}[/tex] , maka agar fungsi tersebut dapat terdefinisi adalah dengan syarat a ≥ 0, sehingga

      1 - x² ≥ 0

      x² - 1 ≤ 0

      (x - 1)(x + 1) ≤ 0

       x - 1 = 0     atau x + 1 = 0

           x = 1                     x = -1

   Tuliskan nilai x pada garis bilangan

   ++++++++--------------+++++++++

                 -1    ≤ x ≤    1

   sehingga domain fungsi fog(x) adalah [tex]D_{fog(x)}[/tex] = {x| -1 ≤ x ≤ 1, x ∈ R}.

Kesimpulan

Jadi,

a. domain dan range dari fungsi [tex]f(x) = \sqrt{1 - x}[/tex] adalah Df = {x| x ≤ 1, x ∈ R} dan Rf = {y| y ≥ 0, y ∈ R}.

b. domain dan range dari fungsi g(x) = x² adalah Dg = {x| x ∈ R} dan Rg = {y| y ≥ 0, y ∈ R }.

c. domain fungsi [tex]fog(x) = \sqrt{1 - x^2}[/tex] adalah [tex]D_{fog(x)}[/tex] = {x| -1 ≤ x ≤ 1, x ∈ R}.

34. F(x) = √25 - x² maka domain fungsi f adalah


Jawaban:

x lebih dari -5 kurang dari 5

Penjelasan dengan langkah-langkah:

Domain:

[tex]f(x) = \sqrt{25 - {x}^{2} } \\ 25 - {x}^{2} \geqslant 0 \\ {x}^{2} \leqslant 25 \\ x \leqslant 5 \: atau \: x \geqslant - 5 \\ - 5 \leqslant x \leqslant 5[/tex]

Domain dari fungsi [tex]\sf{f(x) = \sqrt{25 - {x}^{2}}}[/tex] adalah [tex]\boxed{\sf{D_{f} : \{ \: x \: | \: - 5 \leqslant x \leqslant 5, x \in \Re \: \}}}.[/tex]

ㅤPEMBAHASAN

Domain fungsi disebut juga daerah asal fungsi. Domain fungsi menyatakan nilai-nilai agar fungsi tersebut terdefinisi. Domain dinotasikan dengan huruf D dan diberi label dengan notasi fungsinya. Ambil contoh, domain dari fungsi f dituliskan dengan [tex]\sf{D_{f}}[/tex] dan domain dari fungsi g dinotasikan dengan [tex]\sf{D_{g}}.[/tex]

ㅤ

Berikut beberapa cara menentukan domain fungsi.

1. Domain Fungsi Linier

[tex]\boxed{\sf{f(x) = ax + b}} \iff \boxed{\sf{D_{f} :\{ \: x \: | \: x \in \Re\}}}[/tex]

ㅤ

2. Domain Fungsi Kuadrat

[tex]\boxed{\sf{f(x) = {ax}^{2} + bx + c}} \iff \boxed{\sf{D_{f} :\{ \: x \: | \: x \in \Re\}}}[/tex]

ㅤ

3. Domain Fungsi Rasional

[tex]\boxed{\sf{f(x) = \dfrac{u}{v}}} \iff \boxed{\sf{D_{f} :\{ \: x \: | \: v \ne 0, \: x \in \Re\}}}[/tex]

ㅤ

4. Domain Fungsi Irasional

[tex]\boxed{\sf{f(x) = \sqrt{u}}} \iff \boxed{\sf{D_{f} :\{ \: x \: | \: u \geqslant 0, \: x \in \Re\}}}[/tex]

ㅤ

Diketahui : [tex]\sf{f(x) = \sqrt{25 - {x}^{2}}}[/tex]

ㅤ

Ditanyakan : [tex]\sf{D_{f}}[/tex]

ㅤ

Jawab :

[tex]\sf{f(x) = \sqrt{25 - {x}^{2}}}[/tex]

[tex]\sf{ \: \: \: \: \: \: \: 25 - {x}^{2} \: \: \: \: \: \geqslant 0} \\ \sf{(5 - x)(5 + x) \geqslant 0} \\ \sf{(5 - x)(5 + x) = 0} \\ \sf{ \: \: x = 5 \: \: \vee \: \: x = - 5}[/tex]

ㅤ

Uji titik:

Buat garis bilangan seperti dibawah.

[tex]\\ \sf{ - - - - - - \bullet - - - - - - \bullet - - - - - -} \\ \sf{ \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: - 5 \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: 5}[/tex]

ㅤ

Ambil sembarang titik, misalnya x = 0 dan substitusikan ke f(x).

[tex]\sf{f(x) = \sqrt{25 - {x}^{2}}} \\ \sf{f(0) = \sqrt{25 - {0}^{2}}} \\ \sf{f(0)= \sqrt{25}} \\ \sf{f(0) = 5 \: (positif)}[/tex]

ㅤ

Sehingga:

[tex]\sf{ \: \: \: \: \: \: \: \: \: \: (-) \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: (+) \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: (-) }\\ \sf{ - - - - - - \bullet - - - - - - \bullet - - - - - -} \\ \sf{ \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: - 5 \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: 5}[/tex]

ㅤ

Diperoleh [tex]\sf{D_{f} : \{ \: x \: | \: - 5 \leqslant x \leqslant 5, x \in \Re \: \}}.[/tex]

ㅤ

Jadi domain dari fungsi [tex]\sf{f(x) = \sqrt{25 - {x}^{2}}}[/tex] adalah [tex]\boxed{\sf{D_{f} : \{ \: x \: | \: - 5 \leqslant x \leqslant 5, x \in \Re \: \}}}.[/tex]
