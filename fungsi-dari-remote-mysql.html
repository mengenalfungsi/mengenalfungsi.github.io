5. jelaskan fungsi dari software remote desktop connections


Remote Desktop Connection adalah salah satu dari sekian banyak software yang dapat mengendalikan komputer lain melalui jaringan komputer (LAN). Fasilitas ini satu paket dengan Win XP ketika pertama sekali menginstall Win XP di komputer. Sayangnya masih banyak yang belum mengerti bagaimana cara menggunakannya, padahal manfaatnya sangat banyak sekali, salah satunya adalah kita dapat memperbaiki komputer lain tanpa harus bersentuhan dengan komputer tersebut melalui jaringan (LAN). Apa yang anda lihat di monitor anda adalah tampak persis seperti yang ditampilkan dikomputer yang sedang kita remote tersebut, namun jika untuk iseng atau untuk mengerjai komputer teman rasanya fasilitas ini bukanlah pilihan yang tepat, karena pada saat sesi remote dalam keadaan 'on', maka komputer yang sedang diremote akan masuk ke tampilan welcome screen.

Untuk penggunaannya, disarankan komputer 1 dan komputer 2 sama-sama Win XP. Terlebih dahulu kita melakukan pengaturan pada komputer yang akan kita remote (jika komputer teman anda yang hendak di remote, minta izinlah terlebih dahulu).

Catatan:- komputer 1 adalah komputer anda- komputer 2 adalah komputer target (yang akan kita remote) Kita dapan memperbaiki komputer lain tanpa menyentuhnya melalui jaringan LAN

6. Tuliskan 7 jenis aplikasi remote server beserta dengan fungsinya masing-masing.​


Jawaban:

1. Web Server

fungsi:Web Server adalah software yang menyediakan layanan permintaan dengan protocol https atau http yang dipasang pada komputer server. Software untuk server ini bisa diakses melalui web browser yang ada di komputer. Cara kerja sederhana web server bisa dipahami saat web server mengirim permintaan yang ada dan ditampilkan pada browser dalam bentuk tulisan, gambar, video atau yang lainnya.

singkat aja gpp

2. Fax Server

fungsi:Beralih ke jenis contoh aplikasi server yang kedua, yakni Fax Server. Dari namanya saja kita sudah mengetahui bila Fax Server adalah komputer server yang ditujukan untuk melayani kebutuhan Fax untuk komputer client. Artinya, kebutuhan penerimaan dan pengiriman facsimile bisa dilakukan melalui server.

3. Mail Server

fungsi:Mail Server adalah server yang mengurusi berbagai macam atau hal yang berhubungan dengan surat menyurat. Mail Server juga memiliki layanan pendukung tugas berkirim surat tersebut berupa web interface yakni sebuah fitur di mana client bisa mengorganisir atau menulis surat. Surat-surat yang akan dikirimkan akan disimpan pada Mail Server tersebut.

4. FTP Server

fungsi:FTP Server adalah kepanjangan dari File Transfer Protocol server adalah sebuah server yang mendukung adanya pertukaran data dalam server jaringan luas seperti internet. Beberapa software atau aplikasi yang digunakan untuk FTP server antara lain: WU-FTPD, VSFTPD, FTPD, dan PROFTPD.  Contoh fungsi FTP Server adalah mendownload atau mengupload file dari maupun ke server.

5. Database Server

fungsi:Database server adalah server yang mengatur penyimpanan dan pengolahan data pada sebuah jaringan komputer. Database server melayani permintaan client mengenai aktivitas seperti insert, delete, update data dan lain-lain. Beberapa jenis aplikasi database server Oracle Database, MySQL Database, ProgressSQL dan Microsoft SQL Server.

6. Proxy Server

funsi:Adalah jenis aplikasi server yang menjalankan peran sebagai penengah untuk melayani request dari komputer client yang mencari sumber dari server lain. Beberapa fungsi proxy adalah mempercepat akses internet ke situs yang pernah dibuka oleh Client Server, mengatur bandwidth atau sinyal, memblokir situs-situs berbahaya, dan memfilter malware atau virus yang bisa masuk ke jaringan.

7. VPN

fungsi:Jenis aplikasi server selanjutnya adalah VPN yang merupakan singkatan dari Virtual Private Network, adalah koneksi aman antar dua bagian jaringan private (jaringan pribadi)  pada jaringan internet untuk mengurangi biaya operasional. Cara kerja VPN dapat dijelaskan sebagai berikut: Anda hanya perlu mendapatkan Internet Service Provider untuk terhubung ke koneksi internet pribadi.

11. Apakah pengertian dan fungsi dari RDP(Remote Dekstop Protocol)


Remote Desktop Protocol (sering disingkat menjadi RDP) adalah sebuah protokol jaringan yang digunakan oleh Microsoft Windows Terminal Services dan Remote Desktop. RDP dibuat berdasarkan protokol T.120 yang spesifikasinya diumumkan oleh International Telecommunication Union (ITU), yang juga merupakan protokol yang digunakan di dalam perangkat lunak konferensi jarak jauh Microsoft NetMeeting.Klien-klien yang mendukungnya bervariasi, mulai dari sebagian besar sistem operasi Windows 32-bit (termasuk Windows CE dan PocketPC), hingga sistem operasi lainnya, seperti Linux, FreeBSD, UNIXSolaris, dan Apple Mac OS X. Secara default, server yang membuka protokol ini, akan membuka port TCP 3389.

Dengan Remote desktop kita bisa melihat tampilan layar komputer lain (yang diremote) sama seperti di komputer aslinya, dan kita dapat melakukan  perintah sesuai dengan yang kita inginkan.
Misal komputer A dan komputer B. Kita ingin agar komputer A bisa dilakukan remote desktop oleh user dari komputer B. ( komputer A dikendalikan oleh user di  komputer B).

24. kak tolong dong tuliskan fungsi nilai pada mysql


COUNT merupakan perintah SQL yang berfungsi untuk menghitung record (baris) pada suatu tabel atau lebih.

SUM merupakan Syntak SQL yang berfungsi untuk menjumlahkan field yang bernilai angka (INTEGER). SUM meiliki pola hampir sama dengan COUNT.

MIN merupakan Syntak SQL yang berfungsi untuk menyeleksi nilai terendah (minimum) dari field yang bernilai angka (INTEGER).

38. apakah fungsi oscilator dalam remote control​


Jawaban:

Fungsi oscilator dalam remote control ADALAH sebagai pembangkit sinyal dengan bentuk gelombang persegi (square wave), dari sebuah pencatu daya berupa baterai DC, yang nantinya akan dimodulasi oleh gelombang carrier sebelum dikodekan dan ditransmisikan melalui LED pada remote control.

PEMBAHASAN :

=== Pengertian Osilator ===

--> Osilator merupakan suatu rangkaian elektronika yang berfungsi untuk membangkitkan sinyal AC dengan bentuk gelombang seperti :

sinusoidal (gelombang sinus)gelombang persegi (square wave)gelombang segitiga (triangle wave), maupun gelombang gigi gergaji (saw tooth wave)

dari sebuah pencatu daya DC (frekuensi nol dan amplitudo konstan).  

--> Pada umumnya dalam rangkaian osilator, digunakan Rangkaian Penguat Operasional (RPO) berupa amplifier, kapasitor, dan dioda (bergantung dari jenis sinyal output yang diinginkan).

--> Seringkali rangkaian osilator disandingkan dengan sebuah rangkaian komparator sehingga dapat menggantikan peran saklar dan dihasilkan sinyal dengan gelombang output tertentu secara kontinu. Penambahan kapasitor biasa dilakukan pada osilator dengan output gelombang tanjakan (ramp), dimana proses charging dan discharging kapasitor turut berperan dalam mempengaruhi bentuk sinyal ramp yang dihasilkan.

=== Pengaplikasian Osilator ===

--> Osilator biasa digunakan dalam perangkat bel rumah, remote control, pemancar radio, serta penerima (receiver) televisi dalam mengatur berkas elektron pada tabung gambar.

--> Pada remote control (terlebih yang menggunakan led sebagai pemancar sinar inframerah), osilator berperan dalam membangkitkan sinyal listrik berbentuk gelombang kotak dengan frekuensi sekitar 3GHz - 400THz (rentang frekuensi inframerah), dari sebuah pencatu daya berupa baterai DC.

--> Gelombang kotak (square wave) yang dibangkitkan nantinya akan dikodekan dalam bentuk pulsa dan dikirim sesuai dengan frekuensi gelombang yang diinginkan dan disepakati oleh transmitter (remote) dan receiver (televisi) melalui sinyal inframerah, dengan terlebih dahulu dimodulasi dengan sinyal carrier tertentu.
