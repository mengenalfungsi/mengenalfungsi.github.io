8. Sebuah bola dilambungkan ke atas dari * tanah membentuk parabola. Seorang siswa kelas 10 sedang mempelajari fungsi kuadrat, diketahui f(x) = -3x²+12x-5 adalah persamaan fungsi kuadrat dari lambungan bola tersebut. Koordinat titik puncak grafik fungsi kuadrat tersebut adalah ... O (-2,7) O (-1,4) O (1,4) O (2-7) O (2,7)


Persamaan fungsi kuadrat yang diberikan adalah f(x) = -3x² + 12x - 5.

Untuk menentukan koordinat titik puncak grafik fungsi kuadrat, kita perlu menggunakan rumus umum f(x) = a(x - h)² + k, di mana (h, k) adalah koordinat titik puncak.

Dalam persamaan fungsi kuadrat yang diberikan, a = -3, sehingga kita perlu mencari h dan k.

Rumus umum tersebut dapat dituliskan ulang menjadi -3x² + 12x - 5 = -3(x - h)² + k.

Dalam bentuk tersebut, h = -b/2a dan k = f(h), di mana b = 12.

h = -12/(2*(-3)) = -12/(-6) = 2.

Kemudian, kita substitusikan nilai h ke dalam persamaan awal untuk mencari k:

-3(2)² + 12(2) - 5 = -3(4) + 24 - 5 = -12 + 24 - 5 = 7.

Jadi, koordinat titik puncak grafik fungsi kuadrat tersebut adalah (2, 7).

(2,7) adalah jawaban yang tepat.

12. Suatu grafik fungsi kuadrat memiliki titik puncak (4,−6) dengan salah satu titik potong sumbu x adalah (8,0). Dari yang diketahui ini, manakah dari tiga bentuk di atas yang paling tepat digunakan untuk menentukan model fungsi kuadratnya? Tentukan fungsi tersebut.


Pembahasan
Rumus umum fungsi kuadrat
adalah f(x)= ax² + bx + c

diketahui titik puncak (4,-6)
artinya gantikan f(x) dengan -6 dan x dengan 4, maka persamaan di atas menjadi
-6 = a(4)² + b(4)+ c
-6 = 16a + 4b + c .....(1)

Karena titik potong di sumbu x adalah (8,0), maka
artinya f(x) = 0 dan x = 8, sehingga
f(x)= ax² + bx + c
0 = a(8)² + b(8) + c
0 = 64a + 8b + c ....(2)

Dari titik puncak (4,−6)
artinya sumbu simetri x = 4, sehingga
titik potong lain dengan sumbu x selain titik (8,0) adalah titik yang mempunyai absis x = 2(4)-8= 0 dan ordinat y=0 , yaitu
(0,0)
Titik ini disubtitusikan ke f(x)= ax² + bx + c,
kita dapatkan
0 = 0 + 0 + c atau c = 0 .....(3)

Lalu,
pers.(3) disubstitusikan ke (1)
-6 = 16a + 4b + 0
16a + 4b = -6 ......(4)

dan
pers.(3) disubstitusikan ke (2)
0 = 64a + 8b + c
0 = 64a + 8b + 0
64a + 8b = 0 .......(5)

Kemudian,
dari (4) dan (5)
16a + 4b = -6 ]×2
64a + 8b = 0
_________ _
(32-64)a = -12
-32a = -12
a = 12/32 = 3/8

dan
16a + 4b = -6 ]×4
64a + 8b = 0
_________ _
(16 -8)b = -24
8b = -24
b = -24/8 = -33

Sehingga fungsi kuadrat tersebut adalah
f(x)= ax² + bx + c
f(x)= (3/8)x² +(-3 )x + 0
f(x) =(3/8)x² -3 x


Jawab
Fungsi tersebut adalah f(x) =(3/8)x² - 3x

Bentuk umum fungsi kuadrat adalah f(x) = ax² + bx + c dengan a ≠ 0.

Jika a > 0 maka kurva terbuka ke atas dan jika a < 0 maka kurva terbuka ke bawah

Ada beberapa cara untuk menentukan persamaan fungsi kuadrat, yaitu

Jika diketahui titik puncak (xp, yp) dan melalui titik (x, y)

y = a(x – xp)² + yp

Jika diketahui titik potong terhadap sumbu x di titik (x₁, 0) dan (x₂, 0) serta melalui titik (x, y)

a < 0 (a = bilangan negatif)

Kurva memotong sumbu x di titik (–1, 0) dan (3, 0)

a. sumbu simetri

xp = ½ (–1 + 3)

xp = ½ (2)

xp = 1

b. rumus fungsi kuadrat

y = a(x – x₁)(x – x₂)

y = a(x – (–1))(x – 3)

y = a(x + 1)(x – 3)

y = a(x² – 3x + x – 3)

y = a(x² – 2x – 3)

c. nilai optimum

diperoleh dengan mensubstitusikan x = xp = 1

y = a(x² – 2x – 3)

yp = a(xp² – 2xp – 3)

yp = a(1² – 2(1) – 3)

yp = a(1 – 2 – 3)

yp = a(–4)

yp = –4a

d. titik puncak

(xp, yp)

= (1, –4a)

e. titik potong grafik dengan sumbu y

Diperoleh jika x = 0

y = a(x² – 2x – 3)

y = a(0² – 2(0) – 3)

y = a(0 – 0 – 3)

y = a(–3)

y = –3a

(0, –3a)

Jadi agar diperoleh nilai a, dalam soal harus diketahui titik lainnya.

Kita misalkan nilainya adalah a = –1

maka
