4. software yang diisntall pada workstation atau server yang berfungsi untuk menyaring IP mana saja yang bisa terkoneksi ke server adalah


Software yang kita install untuk menyaring IP aman saja yang bisa terhubung dengan server adalah Firewall. Firewall merupakan software yang memiliki fungsi untuk memilih IP mana saja yang bisa terhubung dengan Server. Dengan demikian, kita bisa menanggulangi adanya serangan ke server kita.

Pada sistem operasi windows, kita bisa mengaktifkan firewall melalui menu control panel dan memilih windows firewall. Selain menggunakan control panel, kita juga bisa mengakses windows firewall melalui menu services, kemudian cari servis dari windows firewall.

Pembahasan

Beberapa fungsi dari firewall serndiri adalah

Sebagai gerbang utama ketika server terhubung dengan clientSebagai pencatat semua log atau aktivitas yang terhubung dengan serverMelindungi data-data yang berada di server

Dengan demikian apabila komputer kita terhubung dengan komputer lain, bahkan komputer kita terhubung dengan internet, maka sudah semsetinya kita mengaktifkan firewall pada komputer kita agar komputer kita terlindungi.

7. Setiap komputer yang terhubung ke jaringan dapat bertindak baik sebagai workstation maupun server


Jawaban:

Benar, setiap komputer yang terhubung ke jaringan dapat bertindak sebagai workstation atau server. Workstation adalah komputer yang digunakan oleh pengguna untuk menjalankan aplikasi dan melakukan tugas-tugas komputasi. Server adalah komputer yang digunakan untuk menyediakan layanan kepada beberapa atau semua workstation di jaringan.

Workstation dan server dapat dibedakan berdasarkan fungsinya. Workstation biasanya digunakan oleh pengguna untuk menjalankan aplikasi dan melakukan tugas-tugas komputasi, sementara server digunakan untuk menyediakan layanan kepada beberapa atau semua workstation di jaringan. Namun, pada kenyataannya, setiap komputer yang terhubung ke jaringan dapat bertindak sebagai workstation atau server, tergantung pada kebutuhan dan konfigurasi jaringan yang telah ditetapkan.

15. Tuliskan hubungan antara jaringan LAN dengan server dan workstation?​


Hubungan antara jaringan LAN dengan server dan workstation adalah server adalah kunci kualitas jaringan LAN (local area network) yang dibutuhkan setiap workstation untuk berkomunikasi dengan komputer klien. Jika server mengalami kegagalan jaringan, hal ini akan diberitahukan langsung oleh workstation.

Pembahasan

Server adalah sistem komputer yang menyediakan jenis layanan tertentu pada jaringan komputer. Ini juga mendukung prosesor yang dapat diskalakan, RAM dalam jumlah besar, dan memiliki sistem operasi khusus yang disebut sistem operasi jaringan.

Komputer server biasanya berisi daftar pengguna yang memiliki hak akses khusus. Pengguna ini biasa disebut sebagai administrator. Hal ini karena pengguna tidak boleh memasukkan informasi secara acak karena berkaitan dengan pengoperasian dan pemeliharaan komputer server. Karena server adalah pusat jaringan untuk komputer klien, masalah dengan jaringan server menyebabkan masalah dengan komputer klien.

Workstation adalah jenis komputer yang dirancang untuk mendukung komputasi ilmiah, teknik, atau tujuan lain. Tidak seperti komputer biasa, workstation ini menggunakan perangkat keras yang sedikit lebih canggih, sehingga tidak mengherankan jika workstation komuter lebih mahal. Sedangkan untuk sistem operasi komputer workstation sendiri hampir semuanya menggunakan sistem operasi Unix atau yang mirip dengan sistem operasi UNIX.

31. 5. Perhatikan gambar berikut! Workstation Workstation Nilai File Server Workstation Workstation Apa kelebihan topologi tersebut? HOTS atatan​


Pada gambar yang terlampir merupakan topologi jaringan ring atau cincin. Berikut ini adalah kelebihan dari topologi jaringan ring:

a. Laju data (transfer rate) tinggi.

b. Dapat melayani lalu lintas data yang padat.

c. Tidak diperlukan Host, relatif lebih murah.

d. Dapat melayani berbagai media pengirim.

e. Komunikasi antar terminal mudah.

f. Waktu yang diperlukan untuk mengakses data optimal.

Pembahasan

Topologi cincin (ring) mirip dengan topologi bus. Data atau informasi yang ditransfer dari sebuah komputer akan diteruskan ke media transmisi melalui komputer ke komputer selanjutnya. Adapun kekurangan dari topologi ini yaitu jika ada satu komputer (simpul) yang mengalami kegagalan, akan menyebabkan terputusnya semua hubungan komputer tersebut.

Menurut Sofana (2011:12) “Topologi Ring menghubungkan host dengan host lainnya membentuk lingkaran tertutup atau loop”. Jaringan topologi ring ini mirip topologi bus, hanya saja pada ujungujungnya saling berhubungan membentuk suatu lingkaran dengan menggunakan segmen kabel. Pada lingkaran tertutup ini, sejumlah komputer dihubungkan ke lingkaran tersebut. Kinerja topologi ring ini diperkenalkan oleh perusahaan IBM untuk mendukung protokol, Token Ring yang juga diciptakan oleh IBM.
