1. fungsi windows server 2012 dan keunggulan dibandingkan dengan windows lainnya


A. Didasari komputasi melalui jaringan dan fitur cloud.

Windows server 2012 memiliki kelebihan dalam fitur yang telah disebutkan tadi. Karena jarang sistem operasi lainnya yang mensupport teknologi ini. Seperti fitur cloud dari Microsoft Azure yang dapat dihubungkan ke Windows Server 2012.

A. Banyak Versi yang dapat disesuaikan kebutuhan

Dengan bertumbuhnya perusahaan dan bisinis, diperlukan juga akan komputasi server yang sesuai dengan budget dan kebutuhan suatu bisnis. Dengan banyaknya versi membuat bisnis dapat memilih versi Windows Server 2012 yang tepat bagi mereka, tanpa perlu mengkonfigurasi suatu sistem operasi untuk sesuai dengan kebutuhan mereka ( bisnis ).

F. Kekurangan Windows Server 2012

1. Harga yang relatif mahal.
2. Fitur baru yang masih jarang digunakan, sehingga meningkatkan kemungkinan pemakaian yang lebih sulit.

4. Apa Itu Windows Server?


Windows Server adalah sistem operasi untuk server yang sistem operasinya 32 bit dari microsoft yang menjadi leluhur sistem operasi operasi windows 2000, windows XP, server 2003 dan windows Visata. sistem operasi tersebut pada awalnya mendukung beberapa platform mikroprosesor. windows NT server ini tidak di bngun dari MS-DOS mampu mengamati 4 GB.

Windows Server adalah merek dagang untuk sistem operasi server yang dirilis oleh Microsoft Corporation.

9. Jelaskan perbedaan dari sistem operasi windows server 2012 dan linux debian


Jawaban:

Ada perbedaan antara Linux dan Windows baik di tingkat permukaan dan di tingkat inti. Perbedaan utama akan menjadi saat Windows adalah sistem operasi komersial dengan antarmuka grafis.

Linux adalah sistem operasi open source. Linux dianggap lebih stabil daripada Windows sekalipun. Namun, navigasi lebih mudah dengan Windows daripada dengan Linux. Selain itu, sejak Windows memiliki antarmuka pengguna grafis yang umumnya lebih mudah untuk dipelajari dan digunakan, Windows tidak diragukan lagi dipilih lebih Linux sebagai lebih user-friendly.

Perbedaan lain adalah bahwa sementara Linux secara teknis kernel; komponen utama dari banyak sistem operasi, sedangkan Windows tidak. Selain itu, perbedaan lain dapat diamati dalam kaitannya dengan kode sumber. Kode sumber dari distribusi Linux tersedia secara gratis. Ini berarti bahwa setiap orang dapat men-download kode sumber Linux dan menyesuaikannya sesuai kebutuhan.

Hal ini tidak bisa dilakukan dalam kaitannya dengan Windows. Kesimpulannya, menggunakan sistem operasi Windows, Anda perlu membeli mereka, tetapi sistem operasi berbasis Linux yang tersedia secara gratis dan yang lebih menguntungkan.

16. langkah² burning ISO windows server 2012 di flashdisk!!!bantuu yuu!!​


Jawaban:

Berikut adalah langkah-langkah untuk membakar ISO Windows Server 2012 ke flashdisk:

1. Siapkan flashdisk yang memiliki kapasitas cukup untuk menyimpan file ISO Windows Server 2012. Pastikan flashdisk sudah diformat dengan benar sebelum memulai proses.

2. Unduh file ISO Windows Server 2012 dari sumber yang sah dan pastikan file tersebut telah selesai diunduh dengan sempurna.

3. Pastikan komputer Anda terhubung ke jaringan listrik atau menggunakan daya baterai yang cukup, agar tidak terjadi masalah selama proses pembakaran.

4. Unduh dan instal aplikasi pembakaran ISO, seperti Rufus, UNetbootin, atau Windows USB/DVD Download Tool. Anda dapat menemukan aplikasi ini dengan mudah di internet.

5. Sambungkan flashdisk ke komputer Anda dan pastikan flashdisk terdeteksi dengan benar oleh sistem operasi.

6. Jalankan aplikasi pembakaran ISO yang telah Anda instal sebelumnya.

7. Pilih file ISO Windows Server 2012 yang ingin Anda bakar ke flashdisk.

8. Pastikan flashdisk terpilih sebagai tujuan pembakaran. Biasanya, aplikasi akan mendeteksi flashdisk secara otomatis jika sudah terhubung.

9. Pastikan semua pengaturan sudah sesuai dengan kebutuhan Anda, seperti partisi dan sistem file.

10. Klik tombol "Start" atau "Burn" untuk memulai proses pembakaran. Aplikasi akan melakukan proses pembakaran ISO ke flashdisk.

11. Tunggu hingga proses pembakaran selesai. Proses ini akan memakan waktu tergantung pada kecepatan flashdisk dan ukuran file ISO.

12. Setelah proses selesai, Anda akan menerima notifikasi bahwa pembakaran telah berhasil.

Sekarang, flashdisk Anda sudah siap digunakan untuk menginstal Windows Server 2012. Pastikan untuk memeriksa kembali flashdisk tersebut dengan menjalankannya di komputer target sebelum menginstal sistem operasi. Juga, pastikan bahwa BIOS atau UEFI di komputer target sudah diatur untuk boot dari flashdisk. Jika semua telah siap, Anda dapat menginstal Windows Server 2012 dari flashdisk dengan mengikuti petunjuk pada layar.

24. Tuliskan langkah langkah instal windows server (versi 2012 atau lainnya)btw yang 2012 yg ke atas yhh tolong jawab skrg nanti aku follow dehh janji​


Jawaban:

Windows Server 2012 merupakan sistem operasi buatan Microsoft yang diperuntukan pengunaannya untuk server. untuk menginstall sistem operasi ini tidak jauh berbeda langkahnya dengan sistem operasi Windows lainnya seperti Windows 7, 8, 8.1, dan 10. Sistem operasi ini memiliki beberapa versi yang masing-masing edisi memiliki perbedaan yaitu :

Windows Server Standard

edisi ini merupakan edisi yang cukup lengkap fiturnya, edisi ini dapat membaca RAM hingga 4 TerraBytes, chip processor dapat mencapai 64 buah, jumlah users tidak terbatas, hanya saja edisi ini hanya dapat membuat dua Virtual Machine saja sehingga kurang cocok untuk lingkungan Virtual Machine. Edisi ini juga mempunyai dua versi yaitu versi GUI dan Core, GUI untuk mode dengan tampilan grafis sedangkan untuk Core hanya berupa teks saja. Edisi ini juga dapat bergainti-ganti mode dari Core ke GUI maupun GUI ke Core.

Windows Server Data Center

Edisi ini tidak jauh berbeda dengan edisi Standard hanya saja edisi ini dapat membuat Virtual Machine tanpa batas, maka dari itu edisi ini cocok pada lingkungan Virtual Machine. sama dengan edisi standard edisi ini juga memiliki versi Core dan GUI dan dapat berganti-ganti versi tanpa Install ulang.

Sebelum menginstall anda harus memperhatikan speksifikasi komputer yang ingin anda Install Windows Server 2012, speksifikasi minimum untuk sistem operasi ini adalah :

CPU : harus berarsitektur x64 (64bit) dengan clock rate 1,4GHz

RAM : minimal 512 MegaBytes

HDD : 32 GB

"Nama jaringan yang dimaksud sudah tidak lagi tersedia"

server itu sendiri.  Anda mungkin menerima galat berikut ini:

 

"Nama jaringan yang dimaksud sudah tidak lagi tersedia"

"gagal dengan 64: nama jaringan yang dimaksud tidak tersedia"

"Nama jaringan yang dimaksud sudah tidak lagi tersedia"
