33. jelaskan fungsi dari menu edit pada program Corel draw dan Adobe Photoshop CS​


**CorelDRAW:**

- **Undo/Redo (Ctrl+Z, Ctrl+Y):** Memungkinkan Anda untuk membatalkan tindakan terakhir atau mengembalikannya kembali.

- **Cut (Ctrl+X), Copy (Ctrl+C), Paste (Ctrl+V):** Digunakan untuk memotong, menyalin, dan menempelkan elemen grafis.

- **Clone (Ctrl+K):** Membuat salinan objek yang sedang dipilih.

- **Group (Ctrl+G) dan Ungroup (Ctrl+U):** Mengelompokkan dan mengurai objek untuk mengedit bersama atau secara terpisah.

- **Transformations:** Berisi perintah seperti "Rotate," "Scale," "Mirror," dan lainnya untuk mengubah bentuk atau orientasi objek.

- **Find and Replace (Ctrl+F):** Memungkinkan Anda mencari elemen dan menggantinya dengan elemen lain dalam desain.

- **Object Properties:** Menyediakan akses ke pengaturan dan properti objek yang sedang dipilih.

- **Special Effects:** Mengandung perintah untuk memberikan efek khusus seperti "Drop Shadow," "Blur," dan lainnya.

**Adobe Photoshop CS:**

- **Undo/Redo (Ctrl+Z, Ctrl+Y):** Untuk membatalkan tindakan terakhir atau mengembalikannya.

- **Cut (Ctrl+X), Copy (Ctrl+C), Paste (Ctrl+V):** Digunakan untuk memotong, menyalin, dan menempelkan bagian gambar.

- **Transform (Ctrl+T):** Untuk mengubah ukuran, rotasi, atau transformasi lainnya pada layer atau seleksi.

- **Image Adjustments:** Mengandung perintah untuk menyesuaikan warna, kecerahan, kontras, dan banyak aspek lain dari gambar.

- **Layers:** Memungkinkan Anda mengelola lapisan (layers) yang berisi elemen-elemen gambar yang bisa diatur dan dimanipulasi secara terpisah.

- **Crop (C):** Untuk memotong gambar sesuai dengan area yang diinginkan.

- **Select (Ctrl+A, Ctrl+D, Ctrl+Shift+I):** Untuk memilih bagian tertentu dari gambar.

- **Fill (Shift+F5) dan Stroke (Ctrl+L):** Digunakan untuk mengisi area dengan warna atau pola tertentu, serta mengatur garis tepi.

38. Soal1. Sebutkan beberapa macam format untuk menyimpan data Photoshop ?2. Apakah yang dimaksud dengan PDD dan PSD?3. Sebutkan salah satu fungsi utama dari Photoshop ?4. Corel PHOTO-PAINT adalah perangkat lunak buatan Corel yang dikhususkan untuk ?5. Software aplikasi yang digunakan dalam bidang desain grafis dan berbasis Bitmap adalah ?​


Jawaban:

Bukan hanya menggunakan save ataupun save as saja, tapi juga ada cara-cara lain loh untuk menyimpan hasil editan sobat. Tapi, bisa juga menggunakan menu Export (yang memiliki banyak pengaturan tambahan) serta Automate PDF (untuk menyimpan dalam format PDF). Oke, daripada berlama-lama lagi berikut ini adalah tutorial untuk menyimpan file pada Adobe Photoshop.

save

Kalau yang satu ini sih, saya yakin semuanya sudah pada tau ya. Yap, cara yang pertama ini kita menggunakan menu Save yang berada di menu File->Save atau bisa juga menggunakan shortcut Ctrl+S. Sayangnya, jika sobat menggunakan menu ini sobat hanya bisa melakukan proses penyimpanan dengan format yang sama (contoh : jika sobat membuka PSD, maka akan disimpan menjadi PSD juga).

Cara Kedua : Save As

save as

Lanjut ke cara yang kedua yaitu melalui menu Save As yang berada di File->Save As atau bisa menggunakan shortcut Shift+Ctrl+S. Perintah save as ini digunakan untuk menyimpan file dengan format, nama, dan lainnya berbeda dengan format aslinya. (contoh : jika sobat membuka PSD, maka sobat bisa mengubah formatnya menjadi JPEG dengan cara ini)

Selengkapnya : https://mahirphotoshop.com/save-as/

Cara Ketiga : Export

menyimpan file (export)

Nah, mungkin sobat hanya tahu dengan kedua cara diatas ya. Menu export ini bisa sobat gunakan juga loh untuk melakukan perintah menyimpan. Caranya yaitu melalui menu File->Export dan pilih ingin export yang apa. Jika untuk situs web atau blog pilih yang Save For Web. Keunggulan dalam menggunakan menu Export ini, sobat bisa dengan bebas mengatur warna, kompresi, dan lainnya lengkap.

Export As : https://mahirphotoshop.com/export-gambar-photoshop/

Save For Web : https://mahirphotoshop.com/gambar-untuk-web/

Cara Keempat : Automate PDF

menyimpan file PDF

Kalau sobat ingin menyimpan file nya ke dalam format PDF, sobat bisa menggunakan cara yang keempat ini yaitu melalui menu File->Automate->PDF Presentation. Format desain PSD ini biasanya dapat kita gunakan untuk beberapa percetakan, sayembara logo, dan lainnya.

Selengkapnya : https://mahirphotoshop.com/save-as-pdf/

Oke, itu tadi macam-macam cara untuk menyimpan file pada adobe photoshop. Untuk detail lebih jelasnya saya akan mempostingnya di lain kesempatan dan pasti saya sisipkan tautan (link) nya kok di artikel ini hehe.
