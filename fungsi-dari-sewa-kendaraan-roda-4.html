Jika tarif sewa mobil adalah 5000 per mobil dan tarif sewa motor adalah 2000 per motor, maka besar uang sewa seluruhnya yang diperoleh dari penyewaan tersebut adalah

motor = y


x + y = 40 (x4)
4x + 2y = 104 (x1)

-2y = -56
y = -56 : -2
y = 28 ( jumlah motor)


x + y = 40
x + 28 = 40
x = 40 - 28
x = 12 (jumlah mobil)


uang sewa keseluruhan
= x (5000) + y (2000)
= 12 (5000) + 28 (2000)
=60000 + 48000
=108000

8. Seorang pedagang harus menyewa paling sedikit 25 kendaraan. Ia menyewa truk dan colt untuk mengakut paling banyak 224 karung barang. Truk dapat mengangkut 14 karung dan colt 8 karung. Ongkos sewa truk 100.000 tiap kendaraan dan colt 75000 tiap kendaraan. Ongkos minimal yang harus dikeluarkan adalah


Matematika Kelas 10
Sistem Persamaan Linear
Dua Variabel (SPLDV)

Materi :
Bentuk umum SPLDV :
ax + by = c

Metode penyelesaian SPLDV :
a. Eliminasi
b. Substitusi
c. Grafik

Pembahasan :
Pada soal diketahui :
Mis :
Truk = x dan Colt = y

Maka, dapat dibuat persamaan linearnya sbb :
Pers. 1
x + y = 25

Pers. 2
14x + 8y = 224

Langkah 1 :
Eliminasi Ke-2 Persamaan untuk mendapatkan nilai dari salah satu variabel (x atau y).

x + y = 25 | x 8
14x + 8y = 224 | x 1

Jadi :
8x + 8y = 200
14x + 8y = 224
___________-
-6x = -24
x = 4

Langkah ke-2 :
Substitusi nilai x = 4 ke salah satu persamaan (mis. pers. 1), maka :
x + y = 25
4 + y = 25
y = 21

Jadi banyak truk adalah 4. Dan colt adalh 21.

Langkah ke - 3 :
Untuk menghitung ongkos minimal sewa kendraan adalah :

= 4 (100.000) + 21 (75.000)
= 400.000 + 1.575.000
= 1.975.000.

Jadi, ongkos minimal sewa kendaraan sebesar Rp. 1.975.000,-
