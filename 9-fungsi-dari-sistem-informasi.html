1. Komponen yang menyediakan cara untuk input (pengguna memasukkan perintah dan memanipulasi sistem dalam bentuk tulisan atau teks) dan output (sistem menunjukkan efek dari perintah dan manipulasi pengguna) adalah User Interface (Antarmuka Pengguna).

2. Saat ini interface yang mayoritas digunakan dalam perangkat lunak (software) adalah Graphical User Interface (GUI).

3. Komponen GUI yang memiliki bentuk persegi panjang dan dapat tumpang tindih dengan area jendela lainnya adalah Window.

4. Sistem dapat berfungsi dengan baik agar bisa mengembangkan dan meningkatkan ketergunaan yang dikenal dengan istilah Usability (Ketergunaan).

5. Di masa sekarang, penamaan file cenderung menggunakan nama yang panjang karena kapasitas penyimpanan dan ketersediaan ruang telah meningkat.

6. Perangkat lunak yang berfungsi menampilkan dan melakukan interaksi dengan dokumen-dokumen yang disediakan oleh server web adalah Web Browser (Penjelajah Web).

7. Informasi yang diberikan atas perintah yang digunakan secara keseluruhan menggunakan fasilitas khusus yang disediakan oleh browser web. Fasilitas tersebut biasanya lebih dikenal dengan sebutan Browser Developer Tools (Alat Pengembang Browser).

8. Penjelajah web bisa dibedakan lewat berbagai fitur dan kemampuan seperti kecepatan, keamanan, dan dukungan untuk teknologi web tertentu.
