22. Bagaimana cara untuk memotong dokumen slide dengan menggunakan keyboard maka menekan … ctrl s ctrl a ctrl v ctrl l ctrl p


Jawaban:

Untuk memotong dokumen slide dalam program presentasi seperti Microsoft PowerPoint atau Google Slides, Anda dapat menggunakan fitur pemotongan (cut) yang tersedia di dalam program. Caranya adalah dengan memilih teks atau elemen yang ingin dipotong, lalu tekan tombol keyboard "Ctrl + X" untuk memotong dan "Ctrl + V" untuk menyisipkan (paste) ke tempat lain.

Namun, jika Anda ingin memotong dokumen slide di luar program presentasi, misalnya dengan menggunakan software pengolah gambar atau editor teks, caranya mungkin berbeda-beda tergantung pada program yang Anda gunakan.

Penjelasan:

Biasanya, Anda dapat menggunakan kombinasi tombol "Ctrl + X" untuk memotong, "Ctrl + V" untuk menyisipkan, dan "Ctrl + A" untuk memilih seluruh dokumen slide. Namun, pastikan untuk memeriksa dokumentasi atau menu bantuan program yang Anda gunakan untuk mengetahui cara yang tepat.

26. Dalam suatu aplikasi komputer terdapat software microsoft office,salah satunya microsoft word berfungsi untuk membuat file documen, yang terdiri banyak tombol kombinasi, tombol kombinasi yang digunakan untuk menebalkan, membuat garis bawah dan tulisan miring pada aplikasi pengolah kata adalah…..a. Ctrl +F, Ctrl +A, Ctrl+ Ib. Ctrl +B, Ctrl +U, Ctrl+Ic. Ctrl +S, Ctrl +C, Ctrl +Fd. Ctrl +W, Ctrl +A, Ctrl +Se. Ctrl +U, Ctrl+ F, Ctrl +Bnt:tlg cpatt, aku dlm masalh besar. gwsh pke cara gpp. thx​


Jawaban:

B. Ctrl + B, Ctrl + U, Ctrl I

Pendahuluan :

Taukah kamu apa itu Microsoft Office word ? Ya, itu merupakan sebuah aplikasi yang paling populer dalam mengetik, memprint sesuatu baik berupa surat (dokumen) maupun berupa foto (gambar). Tentunya dalam mengetik sesuatu pada aplikasi ini membutuhkan yang namanya tombol untuk mengetik. Tombol komputer tersebut biasa kita kenal dengan sebutan keyboard. Keyboard adalah alat bantu kita untuk mengetik maupun memberi perintah sesuai yang kita inginkan. Di dalam tombol keyboard terdiri atas huruf, angka, maupun simbol - simbol lainnya.

Keyboard (Qwerty) diciptakan oleh Christopher Latham Sholes pada tahun 1868. Ia lahir di negara Amerika Serikat pada tanggal 14 Februari 1819. Setelah anda menyimak penjelasan di atas, kini saatnya membahas pertanyaan fungsi tombol dari keyboard.

Pertanyaan :

Tombol kombinasi yang digunakan untuk menebalkan, membuat garis bawah dan tulisan miring pada aplikasi pengolah kata adalah . . . ?

Pembahasan :

Tombol perintah (ctrl) yang berfungsi untuk menebalkan tulisan di aplikasi software microsoft office adalah tombol Ctrl + B.

Sesuai nama tombolnya, ctrl artinya perintah dan b (bold) artinya menebalkan huruf.

Tombol perintah (ctrl) yang berfungsi untuk membuat garis bawah tulisan di aplikasi software microsoft office adalah tombol Ctrl + U.

Sesuai nama tombolnya, huruf u merupakan awalan kata Underline yang artinya garis bawah.

Tombol kombinasi untuk membuat huruf menjadi miring adalah tombol Ctrl + I.

Sesuai nama tombolnya, huruf I merupakan awalan kata Italic yang artinya tulisan miring.

Kesimpulan :

Ctrl + B = Berfungsi untuk mempertebal huruf

Ctrl + U = Berfungsi untuk mengaris bawahi huruf

Ctrl + I = Berfungsi untuk membuat garis miring
