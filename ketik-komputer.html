Persamaan Linear 2 Variabel adalah suatu bentuk persamaan matematika yang melibatkan 2 variabel dan masing masing variabel berpangkat 1.

Ada beberapa penyelesaian Persamaan Linear 2 Variabel :

Metode GrafikMetode EliminasiMetode SubtitusiMetode Gabungan (Eliminasi - Subtitusi)

PENYELESAIAN

Misal :

komputer = x

mesin ketik = y

.

8x + 5y = 11.300.000 ....(1)

x + 2y = 1.800.000 → x = 1.800.000 - 2y ....(2)

• Subtitusikan nilai x ke persamaan 1

8(1.800.000 - 2y) + 5y = 11.300.000

14.400.000 - 16y + 5y = 11.300.000

-11y = -3.100.000

y = 281.818, ... → Harga 1 mesin ketik

• Subtitusikan nilai y ke persamaan 1

x = 1.800.000 - 2(281.818)

x = 1.800.000 - 563.636

x = 1.236.364 → Harga 1 komputer
