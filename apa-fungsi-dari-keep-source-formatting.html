22. terdapat beragam jenis tombol paste salah satunya bentuk keep scource formatting . Apa yang anda ketahui tentang keep scource formatting pliss bantu ya soalnya hari ini wajib di kumpulin ​


Dalam Microsoft Word, terdapat banyak sekali jenis tombol paste. Salah satunya adalah keep source formating. Dimana Keep Source Formatting berfungsi untuk menyalin seluruh data salinan beserta formatnya.

Pembahasan

Cut, copy, dan paste adalah perintah  paling dasar dan umum digunakan di Word. Fungsinya  untuk menyalin, memindahkan dan menempelkan teks dan gambar ke dalam dokumen, dan ada beberapa cara untuk menerapkannya. Cut, copy, dan paste adalah perintah paling dasar dan umum digunakan di Microsoft Word. Fungsinya untuk menyalin, memindahkan dan menempelkan teks dan gambar dalam dokumen, dan ada beberapa cara untuk menerapkannya.  Jika Anda sering menggunakan sesuatu seperti Microsoft Word, Anda akan sering menggunakan perintah salin dan potong. Ketika Anda mengklik perintah Salin atau Potong pada menu Mulai, Anda dapat menggunakan perintah ini dalam berbagai bentuk. Anda dapat menyalin gambar dari luar dokumen atau memindahkan gambar dari dokumen lain.

Misalnya, jika Anda menempelkan teks menggunakan  kombinasi tombol Ctrl+V pada keyboard, Anda juga dapat menempelkan teks gambar dan memformat teks. Microsoft Word  menyalin karakter pemformatan ke dalam teks. Jadi, jika Anda menemukan bahwa teks yang Anda salin dari internet menjadi lebih besar atau lebih kecil di Microsoft Word. Ini karena Microsoft Word  membaca pemformatan situs web. Menu tarik-turun Tempel juga memiliki perintah Tempel Spesial, yang digunakan untuk menempelkan apa  yang telah Anda salin sebagai jenis dokumen khusus. Misalnya, Anda dapat menyisipkan dokumen Word, gambar, atau HTML. Ini akan ditampilkan sesuai dengan konten yang disalin. Set Default Paste di menu ini memungkinkan Anda untuk mengubah format teks ke format default saat menempelkan  teks.

36. Apa fungsi toolbar standard dan toolbar formatting?​


Jawaban:

Toolbar standard dan toolbar formatting umumnya dapat dilihat pada program perkantoran (program office), seperti pada Ms. Office Word.

Toolbar standard berfungsi untuk mengolah dokumen secara umum.

Toolbar formatting berfungsi untuk mengolah isi dokumen secara khusus.

Penjelasan:

Beberapa contoh aktivitas yg dapat dilakukan dg toolbar standard: membuat dokumen baru, membuka dokumen, menyimpan dokumen.

Beberapa contoh aktivitas yg dapat dilakukan dg toolbar formatting: mengubah jenis teks dalam dokumen, memberi efek teks pada dokumen, mengatur paragraf teks dokumen.

Untuk lebih jelasnya, perhatikan gambar berikut ini.
