6. Jelaskan apa yang dimaksud GPU! Dan Guna GPU dalam bermain game apa?!!


Jawaban:

Dalam definisinya, Graphic Processing Unit merupakan chip yang dapat di program dan dispesialisasikan untuk keperluan display atau tampilan. GPU akan bertugas untuk melakukan render pada gambar, animasi serta video yang akan ditampilkan pada layar laptop atau desktop Anda.

GPU adalah kartu grafis.

Guna GPU dalam bermain game adalah untuk meningkatkan performa bermain game, seperti misalkan bermain PUBG, jika semain bagus kartu grafis (GPU) maka game pun akan semakin lancar, biasanya jika GPU kelas atas bisa menyentuh angka 120 fps bahkan lebih.

17. Apa itu MOBO,GPU,SSD,HDD,GPU dalam sebuah pc


MOBO (Motherboard) = papan sirkuit yang berguna untuk meletakkan unit kecil dan chipset pc yang saling berhubungan
GPU (Graphics Processing Unit) = Sebuah unit yang berfungsi untuk mengolah dan mengarahkan grafis pada pc
SSD (Solid State Drive) = Berguna untuk menyimpan data seperti halnya harddisk
HDD (Hard Disk Drive) = Salah satu unit yang berfungsi untuk menyimpan data

22. Apa fungsi CPU dan GPU


Fungsi CPU adalah menangani hampir semua tugas dan perhitungan yang dilakukan dari semua perangkat keras yang disematkan di dalam sebuah komputer.

Fungsi GPU adalah memproses instruksi yang dikirimkan oleh sebuah software untuk ditampilkan ke layar.

CPU berfungsi untuk mengolah data berdasarkan instruksi dan data dari software.

GPU berfungsi untuk mengolah tampilan grafik
