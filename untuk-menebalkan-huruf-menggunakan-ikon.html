Untuk mengubah jenis huruf, format font dapat diubah melalui menu Format atau dengan menekan tombol Ctrl + Shift + F.

Untuk mengubah ukuran huruf menjadi 36, ukuran font dapat diubah melalui menu Format atau dengan menekan tombol Ctrl + Shift + P dan memilih ukuran 36.

Untuk menebalkan huruf, format tebal dapat diaktifkan melalui menu Format atau dengan menekan tombol Ctrl + B.
