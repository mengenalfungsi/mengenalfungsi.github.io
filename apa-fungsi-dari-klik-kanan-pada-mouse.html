21. apa fungsi klik satu kali pada penggunaan mouse?


fungsinya membukan aplikasi yg diklik1. Klik kanan pada Mouse : Cara penggunaannya adalah cukup menekan tombol kanan pada mouse, kemudian akan muncul menu menu baru berdasarkan aplikasi yang sedang berjalan, misalnya pada Microsoft Word jika ditekan klik kanan akan muncul menu copy, paste, cut, font, paragraph, bullets, numbering, dll.

2. Klik kiri pada Mouse : Cara  menggunakan adalah dengan menekan tombol kiri pada mouse, penggunaan ini biasa disebut dengan "klik" saja. Tombol kiri pada mouse digunakan untuk mengkses menu, menandai / memilih pada dialog box, memindahkan kursor saat menulis, memilih objek tertentu dan sebagainya.

3. Scroll pada Mouse : Mouse jenis terbaru selalu dilengkapi dengan tombol scroll, tombol scroll ini berbentuk seperti roda yang berfungsi menggulung (keatas / kebawah) halaman pada layar monitor. Apabila mouse tidak dilengkapi dengan tombol scroll seperti pada touchpad laptop , kita tetap dapat menggulung halaman /layar dengan cara klik scroll bar yang biasa ada pada sebelah kanan suatu program.

4. Drag pada Mouse : Drag ialah menyeret atau memindahkan kursor / pointer ke arah tertentu dan tetap mempertahankan klik kiri tidak terlepas. Ini biasa digunakan untuk memindahkan objek, melakukan blok teks, atau  memilih beberapa objek sekaligus.

5. Double click pada Mouse : Klik ganda atau biasa disebut dengan  istilah double click / dobel klik biasa digunakan untuk membuka / menjalankan suatu program tertentu, misalnya jika klik ganda pada shortcut winamp maka program tersebut akan segera terbuka. Caranya cukup 2 kali klik kiri secara cepat.

Itulah pengertian dan fungsi mouse pada komputer serta 5 operasi dasar mouse. Berdasarkan portnya, mouse juga dibedakan menjadi beberapa macam, yaitu Mouse Wireless (nirkabel) Mouse USB, Mouse PS/2, dan Mouse Serial. Untuk mouse jenis Serial dan Mouse PS/2 mungkin saat ini sudah jarang digunakan. dan Mouse USB lah yang paling banyak digunakan saat ini karena menggunakan port USB dan pastinya lebih efisien.

28. jelaskan arti perintah klik, dobel klik, dan klik kanan pada mouse


Klik merupakan perintah untuk memilih objek atau menjalankan suatu perintah pada komputer. Sedangkan dobel klik merupakan perintah untuk membuka suatu objek atau menjalankan perintah yang lebih kompleks. Dan klik kanan merupakan perintah untuk menampilkan suatu menu konteks.

Pembahasan

Klik pada komputer merupakan suatu perintah dasar untuk berinteraksi dengan objek yang dilakukan dengan menekan tombol kiri mouse kemudian melepaskannya. Perintah ini dapat digunakan untuk memilih suatu objek, menjalankan perintah, atau membuka jendela.

Klik juga bisa digunakan untuk memilih atau mengaktifkan objek seperti ikon, menu, atau teks.

Pada dobel klik akan membuka objek atau menjalankan perintah yang lebih kompleks dengan menekan tombol kiri mouse dua kali secara cepat. Perintah ini sering digunakan untuk membuka file atau folder.

Dobel klik biasanya digunakan untuk membuka atau menjalankan objek untuk membuka file, folder, atau aplikasi.

Sedangkan klik kanan termasuk perintah untuk menampilkan menu konteks dengan menekan tombol kanan mouse. Menu ini akan menampilkan daftar perintah yang digunakan untuk objek yang dipilih.

Klik kanan bisa juga digunakan untuk membuka menu konteks yang digunakan untuk menampilkan opsi tambahan untuk objek yang diklik.
