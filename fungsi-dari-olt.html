Vp = 220 V
Vs = 22 Volt
Ip = 0,1 A
Is = 0,6 A
Ditanyaakn : Efisiensi ?

Dijawab ;
Efisiensi ;
η = (VsIs) / (VpIp) x 100%
   = (22x0,6) / (220x0,1) x 100%
   = 13,2/22 x 100%
   = 0,6 x 100%
   = 60 %
