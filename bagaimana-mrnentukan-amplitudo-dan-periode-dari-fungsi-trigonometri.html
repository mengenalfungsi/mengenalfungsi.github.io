8. Tentukan Amplitudo dan Periode grafik fungsi trigonometri y = 3 cos 2 Dan juga membuat grafik fungsi


PEMBAHASAN

Diketahui fungsi seperti berikut ini :

f(x) = 3 cos 2x°

Marilah kita mencoba mencari beberapa nilai - nilai yang penting dari grafik fungsi ini :

Untuk mencari Amplitudo tinggal di lihat koefisien di depan fungsi kosinus nya yakni angka 3 , jadi Amplitudo untuk fungsi ini adalah 3.

f(x) = 3 cos 2x

Untuk mencari Periode tinggal di bagi 360° dengan koefisien variabel x yakni 2 di fungsi ini , jadi Periodenya adalah 360° / 2 = 180°

f(x) = 3 cos 2x

Untuk mencari Nilai maksimum dan minimum tinggal di masukkin nilai maksimum fungsi kosinus yakni 1 dan nilai minimum fungsi kosinus yakni -1 , sehingga persamaan menjadi :

f(x) maksimum → 3 cos 2x = 3 (1) = 3

f(x) minimum → 3 cos 2x = 3 (-1) = -3

Grafik fungsi nya bisa di gambarkan seperti terlihat pada gambar di lampiran.

13. Diketahui fungsi trigonometri f(x) = 2 sin ( x - 30°) +3 tentukan ; a. Amplitudo b. Periode C. Nilai maks dan min D. Grafik fungsi


Diketahui fungsi trigonometri f(x) = 2 sin ( x - 30° ) + 3 , maka :

A. Amplitudo = 2

B. Periode = 360°

C. Nilai Maks dan Min = 5 dan 1

D. Grafik fungsi = ada pada gambar di lampiran

PEMBAHASAN

Sebelumnya kita mesti mengetahui prinsip dasar dari trigonometri.

Jikalau di ketahui segitiga siku-siku maka bisa di buat :

sinus sebuah sudut = sisi di depan sudut / sisi miring

kosinus sebuah sudut = sisi di samping sudut / sisi miring

tangen sebuah sudut = sisi di depan sudut / sisi di samping sudut

Beberapa identitas trigonometri yang sering kita jumpai adalah sebagai berikut ini :

cosec A = 1 / sin A

sec A = 1 / cos A

cot A = 1 / tan A

tan A = sin A /  cos A

cos²A + sin² A = 1

Untuk mencari nilai sudut dari fungsi trigonometri dapat di gunakan rumus - rumus berikut ini :

Untuk Fungsi Sinus :

f(x) = sin x = sin α

x = α + k. 360° atau

x = 180° - α + k. 360°  

Untuk Fungsi Kosinus :

f(x) = cos x = cos α

x = α + k. 360° atau

x = - α + k. 360°

Untuk Fungsi Tangen :

f(x) = tan x = tan α

x = α + k. 180°

Untuk mencari nilai fungsi trigonometri sudut - sudut yang lebih besar dari satu putaran bisa menggunakan rumus berikut ini :

sin ( n. 360° + α ) = sin α

cos ( n. 360° + α ) = cos α

tan ( n. 360° + α ) = tan α

Marilah kita gunakan prinsip dasar ini untuk menyelesaikan soalnya.

Diketahui fungsi trigonometri sebagai berikut ini :

f(x) = 2 sin ( x - 30° ) + 3

Untuk mencari Amplitudo tinggal di lihat koefisien di depan fungsi sinus nya yakni angka 2 , jadi Amplitudo untuk fungsi ini adalah 2.

f(x) = 2 sin ( x - 30° ) + 3

Untuk mencari Periode tinggal di bagi 360° dengan koefisien variabel x yakni 1 di fungsi ini , jadi Periodenya adalah 360° / 1 = 360°

f(x) = 2 sin ( 1.x - 30° ) + 3

Untuk mencari nilai maksimum dan nilai minimum , kita bisa masukkin nilai maksimum fungsi sinus yakni 1 dan nilai minimum fungsi sinus yakni - 1 , sehingga persamaan menjadi :

f(x) maksimum →  2 sin ( x - 30° ) + 3 =  2(1) + 3 = 5

f(x) minimum →  2 sin ( x - 30° ) + 3 =  2(-1) + 3 = 1

Grafik fungsi nya bisa di gambarkan seperti grafik dasar sinus tetapi memiliki amplitudo 2 , lalu digeser kekanan sebesar 30° dan terakhir di geser ke atas sebesar 3 satuan. Untuk lebih jelasnya bisa di lihat pada gambar di lampiran.

Nilai maksimum dan minimum dapat memanfaatkan amplitudo, dengan nilai stabil y = -1 (konstanta pada f(x)):
Maksimum terjadi saat cos 2x = 1, diperoleh:
Maks = 2(1) - 1 = 1
Minimum terjadi saat cos 2x = -1. diperoleh:
Min = 2(-1) - 1 = -3
